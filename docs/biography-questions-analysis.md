# 人物誌題目分析與建議

> 版本: 1.0
> 建立日期: 2026-01-11
> 目標: 透過故事題目設計，達到「鼓勵與分享」的精神

---

## 目前人物誌題目

### 現有 7 道題目

| 編號 | 題目 | 類型 | 目的 |
|-----|------|-----|------|
| 1 | 哪一年開始攀岩 | 基本資訊 | 了解攀岩資歷 |
| 2 | 平常出沒岩場 | 基本資訊 | 建立地理連結 |
| 3 | 喜歡的路線型態 | 偏好分享 | 找到同好 |
| 4 | 踏上攀岩不歸路的原因 | 起源故事 | 引發共鳴 |
| 5 | 攀岩對你來說，是什麼樣的存在 | 意義探索 | 深度反思 |
| 6 | 在攀岩世界裡，想做的人生清單有什麼 | 目標分享 | 激勵他人 |
| 7 | 對於初踏入攀岩的岩友，留言給他們的一句話 | 建議傳承 | 鼓勵新手 |

### 現有問題分析

**優點：**
- 題目簡潔易懂
- 涵蓋基本資訊到深度反思
- 有傳承精神（給新手的話）

**可改進之處：**
- 缺乏「成長故事」的深度挖掘
- 沒有「挫折與突破」的敘事空間
- 缺少社群互動元素
- 題目偏靜態，缺乏動態更新的機制

## 建議新增題目

### 第一類：成長故事 (Growth Stories)

這類題目旨在挖掘攀岩旅程中的轉折點，讓讀者看到真實的成長過程。

#### 1. 攀岩路上最難忘的一天
```
題目：描述一個攀岩中最難忘的時刻或一天
目的：創造具體的故事場景，引發情感共鳴
填寫提示：可以是第一次完攀、受傷、突破瓶頸、或與岩友的特別經歷
```

#### 2. 曾經想放棄的時刻
```
題目：有沒有曾經想放棄攀岩的時刻？是什麼讓你繼續？
目的：展現真實的掙扎與堅持，鼓勵正在低潮的人
填寫提示：可以是受傷、瓶頸期、生活壓力等
```

#### 3. 最大的突破經歷
```
題目：攀岩生涯中最大的一次突破是什麼？
目的：分享突破的喜悅與方法
填寫提示：可以是難度突破、恐懼克服、心態轉變等
```

### 第二類：心理與哲學 (Mind & Philosophy)

探索攀岩帶來的內在轉變與人生體悟。

#### 4. 面對恐懼的方式
```
題目：你如何面對攀岩中的恐懼？（墜落、高度、失敗等）
目的：分享克服恐懼的實用經驗
填寫提示：具體的方法或心理建設過程
```

#### 5. 攀岩教會我的事
```
題目：攀岩教會你最重要的一件事是什麼？
目的：將攀岩經驗連結到更廣的人生智慧
填寫提示：可以是關於失敗、堅持、專注、信任等
```

#### 6. 在岩壁上的當下
```
題目：攀爬時，你的腦袋裡在想什麼？
目的：分享攀岩時的心理狀態，展現專注的美
填寫提示：可以描述心流狀態、內心對話、專注點等
```

### 第三類：社群與連結 (Community & Connection)

強化攀岩社群的歸屬感與互助精神。

#### 7. 攀岩路上的貴人
```
題目：在攀岩路上，誰給你最大的影響或幫助？
目的：感謝與傳承，建立社群連結
填寫提示：可以是教練、岩友、家人、甚至是影片中的攀岩者
```

#### 8. 最想一起攀岩的人
```
題目：如果可以跟任何人一起攀岩，你會選擇誰？為什麼？
目的：展現攀岩英雄與夢想
填寫提示：可以是專業選手、朋友、已故的攀岩者等
```

#### 9. 給過去自己的話
```
題目：如果能回到剛開始攀岩的時候，你會對自己說什麼？
目的：傳承智慧，幫助新手少走彎路
填寫提示：比現有「給新手的話」更具個人特色
```

### 第四類：目標與夢想 (Goals & Dreams)

將人生清單細化，增加追蹤與更新機制。

#### 10. 下一個挑戰
```
題目：目前正在努力的目標是什麼？
目的：展現進行中的挑戰，增加動態感
填寫提示：具體的路線、難度、或挑戰
可更新：允許定期更新進度
```

#### 11. 夢想中的攀登
```
題目：夢想中最想完成的攀登是什麼？
目的：分享終極夢想，激發想像
填寫提示：國內外皆可，描述為什麼這個地方/路線吸引你
```

### 第五類：生活整合 (Life Integration)

展現攀岩者的多元面向。

#### 12. 攀岩之外的我
```
題目：除了攀岩，你還有什麼興趣或專長？
目的：展現完整的人，建立更多連結點
填寫提示：工作、其他運動、藝術、音樂等
```

#### 13. 攀岩如何改變生活
```
題目：攀岩如何影響或改變了你的日常生活？
目的：展現攀岩的正面影響
填寫提示：作息、飲食、社交、工作態度等
```

## 建議題目架構

### 核心題目（必填，7題）

保留原有精華，微調表達方式：

| 原題目 | 建議調整 | 調整理由 |
|-------|---------|---------|
| 哪一年開始攀岩 | 保持不變 | 基本資訊，簡潔 |
| 平常出沒岩場 | 保持不變 | 地理連結 |
| 喜歡的路線型態 | 保持不變 | 找同好 |
| 踏上攀岩不歸路的原因 | 「你與攀岩的相遇」| 更溫暖，減少「不歸路」負面感 |
| 攀岩對你來說，是什麼樣的存在 | 保持不變 | 深度反思 |
| 人生清單 | 改為結構化清單 | 見前一份規劃文件 |
| 給新手的一句話 | 「給剛開始攀岩的自己/岩友」| 更有個人特色 |

## 鼓勵與分享機制設計

### 互動功能

| 功能 | 說明 | 目的 |
|-----|------|------|
| 故事按讚 | 對特定回答按讚 | 鼓勵分享者 |
| 留言回應 | 在故事下方留言 | 建立對話 |
| 「我也是」 | 標記有類似經歷 | 建立共鳴 |
| 「受到鼓勵」 | 表達受到激勵 | 正向回饋 |
| 分享故事 | 將故事分享到社群 | 擴大影響 |
| 加入清單 | 將他人目標加入自己清單 | 目標傳遞 |

### 內容推廣機制

1. **每週精選故事** - 編輯挑選感人/實用的故事推送
2. **主題週** - 如「恐懼週」集中展示克服恐懼的故事
3. **新人故事** - 優先曝光新加入者的故事
4. **達成里程碑** - 有人完成清單目標時推播通知

### 徽章與成就

```
分享者徽章：
🌱 故事新芽 - 完成第一篇故事
📝 分享達人 - 完成 5 個進階題目
💫 啟發者 - 故事被 10 人標記「受到鼓勵」
🔥 熱門故事 - 單篇故事獲得 50 讚

互動者徽章：
👀 好奇寶寶 - 閱讀 20 篇人物誌
❤️ 鼓勵大使 - 給出 50 個「受到鼓勵」
💬 對話者 - 留言 20 則回應
```
