## 分類說明

### 世界經典路線
介紹全球最具代表性的攀岩路線，包含：
- **大岩壁路線**：El Capitan 系列路線（The Nose、Dawn Wall）
- **極限難度路線**：如 Silence (9c)、Change (9b+)
- **歷史意義路線**：首攀紀錄、攀岩史里程碑

### 台灣經典路線
收錄台灣本土戶外岩場的經典路線：
- **龍洞岩場**：台灣最大的天然岩場
- **關子嶺**：南台灣熱門抱石區
- **其他岩場**：大砲岩、熱海岩場等

### 抱石經典問題
世界知名抱石區的代表性問題：
- **Fontainebleau**：法國楓丹白露森林
- **Bishop**：美國加州東部
- **Rocklands**：南非開普敦

## 寫作風格指南

1. **路線介紹**：包含難度等級、路線長度、首攀資訊
2. **歷史背景**：路線的開發歷史與重要首攀紀錄
3. **攀登建議**：季節、裝備、注意事項
4. **實用資訊**：交通、住宿、當地資源
