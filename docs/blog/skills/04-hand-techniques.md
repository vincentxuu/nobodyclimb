## 內容大綱

### 1. 引言：手法的重要性與安全
- 正確手法提升攀爬效率
- 錯誤抓握是手指受傷的主因
- 學習手法的正確心態（效率優於蠻力）

### 2. 手部解剖學基礎
- 手指肌腱與滑車系統（Pulley System）
- 常見攀岩手部傷害
- 為什麼特定抓握方式較危險
- 暖身與預防傷害的重要性

### 3. 基礎抓握技巧

#### 3.1 Open Hand（開放式抓握）
- **定義**：手指維持自然彎曲，關節角度較大
- **使用時機**：較大的岩點、長路線需要省力
- **技術要點**：
  - 手指自然彎曲，不過度彎折
  - 讓岩點「落在」手指上
  - 拇指的輔助位置
- **優點**：對手指壓力最小、適合耐力路線
- **適合點型**：圓潤邊緣、中大型岩點

#### 3.2 Half Crimp（半封閉式抓握）
- **定義**：第一指節彎曲約 90 度，拇指不壓在食指上
- **使用時機**：中小型邊緣岩點
- **技術要點**：
  - 第一指節（DIP）與第二指節（PIP）的角度
  - 手腕位置的調整
  - 力量傳遞的方向
- **優點**：比 Full Crimp 安全，仍有良好握力
- **練習重點**：建立正確的手指姿勢習慣

#### 3.3 Full Crimp（全封閉式抓握）
- **定義**：第一指節過度彎曲，拇指壓在食指上
- **使用時機**：非常小的邊緣、需要最大握力
- **技術要點**：
  - 拇指的壓力位置
  - 手指的完全閉合
- **⚠️ 風險警告**：
  - 對 A2 滑車壓力最大
  - 是最常見的手指受傷原因
  - 建議減少使用頻率
- **建議**：能用 Half Crimp 就不用 Full Crimp

### 4. 特殊抓握技巧

#### 4.1 Sloper（斜面抓握/開掌抓）
- **定義**：用手掌與手指的摩擦力抓住圓滑、無明確邊緣的岩點
- **使用時機**：圓潤岩點、光滑表面
- **技術要點**：
  - 最大化接觸面積
  - 手腕的角度（通常向下彎曲）
  - 身體位置的配合（重心低於手點）
  - 肩膀與手臂的發力方向
- **常見錯誤**：
  - 只用指尖抓
  - 身體位置太高
  - 手臂完全伸直
- **練習方法**：
  - 專門的 Sloper 路線練習
  - 手指力量訓練
  - 學會「包覆」岩點的感覺

#### 4.2 Pinch（捏握）
- **定義**：用拇指與其他手指對向捏住岩點
- **使用時機**：突出的岩塊、柱狀岩點
- **類型**：
  - 窄 Pinch（拇指與食指/中指）
  - 寬 Pinch（拇指與所有手指）
- **技術要點**：
  - 拇指力量的重要性
  - 捏握角度的調整
  - 手腕與前臂的參與
- **常見錯誤**：
  - 只用手指前端捏
  - 拇指位置不佳
- **訓練建議**：
  - Pinch Block 專項訓練
  - 逐漸增加負重

#### 4.3 Undercling（反握/下拉握）
- **定義**：手掌朝上，從下方抓住岩點向上拉
- **使用時機**：岩點下緣可抓、仰角路線
- **技術要點**：
  - 手臂彎曲的角度
  - 身體相對於岩點的位置
  - 核心力量的參與
  - 與腳點配合推拉
- **動作要領**：
  1. 找到岩點下緣可抓處
  2. 翻轉手掌向上抓握
  3. 配合腳點形成推拉力
  4. 利用反向力量移動
- **常見錯誤**：身體太近或太遠

#### 4.4 Sidepull（側拉）
- **定義**：從側面拉住垂直方向的岩點
- **使用時機**：垂直邊緣、側向岩點
- **技術要點**：
  - 身體相對位置的調整
  - 形成對向力
  - 與腳點的配合
- **常見錯誤**：沒有建立對向力

#### 4.5 Gaston（外撐/推握）
- **定義**：手肘朝外，從內向外推壓岩點
- **使用時機**：寬闊路線、雙手向外推
- **技術要點**：
  - 肩膀與肘部的位置
  - 胸部與核心的參與
  - 穩定性控制
- **注意事項**：對肩膀壓力較大

#### 4.6 Pocket（指洞）
- **定義**：將一至三根手指插入岩洞中
- **類型**：
  - Mono（單指洞）
  - Two-finger（雙指洞）
  - Three-finger（三指洞）
- **使用時機**：有洞穴的岩壁，常見於石灰岩
- **技術要點**：
  - 選擇正確的手指組合
  - 避免過度彎曲
  - 注意深度與角度
- **⚠️ 風險警告**：
  - 對手指壓力極大
  - Mono 對初學者危險
  - 建議漸進式訓練

### 5. 手法轉換技巧
- **Matching（併手）**：兩手抓同一個點
- **Bumping（彈手）**：同一手在點上換位置
- **Crossing（交叉）**：手交叉抓取
- 選擇正確時機的判斷
