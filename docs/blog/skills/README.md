## 分類說明

### 心理訓練
專注於攀岩中的心理層面，包括克服恐懼、建立自信、處理壓力等心理技巧。

### 營養飲食
攀岩運動所需的營養知識，包括賽前飲食、訓練期營養補充、體重管理等。

### 腳法技巧
攀岩中各種腳點運用技巧，如 Smear、Edging、Backstep、Drop Knee 等。

### 手法技巧
攀岩中各種抓握方式，如 Crimp、Sloper、Pinch、Undercling 等手法詳解。

### 路線規劃
讀線、Beta 規劃、動作序列安排等策略性技巧。
