- 4.1 什麼是 GriGri？
  - Petzl GriGri 的發展歷史
  - GriGri / GriGri+ / GriGri 2 差異
- 4.2 GriGri 的運作原理
  - 凸輪輔助制動機制
  - 為什麼能「自動鎖定」？
- 4.3 GriGri 的優點
  - 輔助制動增加安全性
  - 長時間確保較輕鬆
  - 適合 Hang Dog（工作路線時懸吊休息）
  - 對新手更友善
- 4.4 GriGri 的缺點
  - 價格較高
  - 重量較重
  - 繩徑限制較嚴格
  - 可能產生錯誤的安全感
  - 不適合直接垂降
- 4.5 GriGri 使用技巧與注意事項
  - 正確的給繩與收繩
  - 「Panic Grab」問題
  - 繩索方向的重要性

### 5. ATC vs GriGri 直接比較

| 比較項目 | ATC | GriGri |
|---------|-----|--------|
| 價格 | ~$20-30 USD | ~$100-120 USD |
| 重量 | ~60g | ~175-200g |
| 制動方式 | 純摩擦力 | 凸輪輔助 |
| 繩徑適用 | 廣泛 | 較受限 |
| 學習曲線 | 較陡峭 | 較平緩 |
| 多功能性 | 可垂降 | 主要確保用 |
| 長時間確保 | 較累 | 較輕鬆 |
| 給繩順暢度 | 順暢 | 需練習 |

- 永遠不要放開制動手
- 定期檢查裝備磨損
