## 內容大綱

### 一、前言（約 200 字）
- 攀岩運動的多元面貌
- 為什麼了解不同類型很重要
- 本文涵蓋範圍預告

### 二、抱石（Bouldering）（約 400 字）
1. **定義與特色**
   - 無繩索、低高度（通常 3-5 公尺）
   - 以軟墊（Crash Pad）作為保護
   - 路線稱為「問題」（Problem）

2. **優點**
   - 入門門檻低、裝備需求少
   - 可獨自練習
   - 專注於單一動作的解題樂趣

3. **適合對象**
   - 喜歡解謎、追求爆發力的攀岩者
   - 時間有限、想快速練習者

4. **知名場地**
   - 室內：各大抱石館
   - 戶外：龍洞短岩壁區、日本御岳

### 三、頂繩攀登（Top Rope Climbing）（約 350 字）
1. **定義與特色**
   - 繩索預先架設於路線頂端
   - 攀登者始終受繩索保護
   - 墜落距離極短

2. **優點**
   - 最安全的繩索攀登方式
   - 適合初學者建立信心
   - 可專注於技術練習

3. **設備需求**
   - 攀岩繩、確保器、安全吊帶
   - 需要確保者配合

4. **適合對象**
   - 初次嘗試高度攀登者
   - 練習新動作、新路線時

### 四、先鋒攀登（Lead Climbing）（約 400 字）
1. **定義與特色**
   - 攀登者邊爬邊掛快扣保護
   - 墜落距離較長，心理壓力大
   - 為運動攀登與傳統攀登的基礎形式

2. **技術要點**
   - 正確的掛繩技巧
   - 墜落姿勢與心理準備
   - 繩索管理

3. **風險與注意事項**
   - 墜落係數概念
   - 地面墜落（Ground Fall）風險
   - 確保者責任加重

4. **適合對象**
   - 已具備頂繩經驗者
   - 追求更大挑戰與自主性

### 五、運動攀登（Sport Climbing）（約 350 字）
1. **定義與特色**
   - 路線已預先打好固定錨點（Bolt）
   - 使用快扣（Quickdraw）連接繩索
   - 專注於攀爬技術與體能

2. **與傳統攀登的差異**
   - 不需自行放置保護點
   - 路線通常經過開發與清理
   - 安全性較高

3. **比賽形式**
   - 奧運攀岩項目之一
   - 難度賽（Difficulty）規則說明

4. **熱門場地**
   - 室內：攀岩館先鋒牆
   - 戶外：龍洞、西班牙 Siurana、泰國 Railay

### 六、傳統攀登（Traditional Climbing / Trad）（約 400 字）
1. **定義與特色**
   - 攀登者自行放置可移除的保護裝備
   - 對岩壁傷害最小的攀登方式
   - 需要更多技術與經驗

2. **保護裝備介紹**
   - 岩楔（Nuts / Stoppers）
   - 凸輪裝置（Cams）
   - 六角塞（Hexes）

3. **技術門檻**
   - 裝備放置判斷能力
   - 路線閱讀與保護點選擇
   - 心理素質要求高

4. **適合對象**
   - 有豐富攀登經驗者
   - 追求冒險精神與自然互動

5. **經典路線**
   - 美國優勝美地、英國威爾斯

### 七、深水獨攀（Deep Water Soloing / DWS）（約 300 字）
1. **定義與特色**
   - 在水面上方的岩壁無保護攀登
   - 墜落後落入水中
   - 結合攀岩與水上活動

2. **必備條件**
   - 足夠深度的水域
   - 水下無障礙物
   - 良好的游泳能力

3. **風險評估**
   - 水深判斷錯誤的危險
   - 高度與入水姿勢
   - 建議有經驗者帶領

4. **知名場地**
   - 泰國 Railay Beach
   - 西班牙 Mallorca
   - 越南下龍灣

### 八、其他攀登類型補充（約 200 字）
1. **大牆攀登（Big Wall）**
   - 多繩距、需過夜的長路線

2. **冰攀（Ice Climbing）**
   - 攀爬冰瀑或冰凍岩壁

3. **高山攀登（Alpine Climbing）**
   - 結合登山與技術攀登

### 九、類型比較總表（約 100 字）

| 類型 | 保護方式 | 入門難度 | 裝備需求 | 適合場地 |
|------|----------|----------|----------|----------|
| 抱石 | 軟墊 | 低 | 少 | 室內/戶外 |
| 頂繩 | 固定繩 | 低 | 中 | 室內/戶外 |
| 先鋒 | 快扣 | 中 | 中 | 室內/戶外 |
| 運動 | 固定錨點 | 中 | 中 | 戶外為主 |
| 傳統 | 可移除裝備 | 高 | 高 | 戶外 |
| DWS | 水面 | 中 | 少 | 特定戶外 |

### 十、結語與建議（約 150 字）
- 建議新手從抱石或頂繩開始
- 循序漸進嘗試不同類型
- 尊重每種攀登形式的文化與風險
