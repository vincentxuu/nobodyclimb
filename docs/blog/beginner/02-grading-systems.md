### 二、路線攀登分級系統（約 600 字）

#### 2.1 YDS 系統（Yosemite Decimal System）
1. **起源與使用地區**
   - 發源於美國優勝美地
   - 主要在北美地區使用

2. **分級結構**
   - Class 1-4：步行到簡易攀爬
   - Class 5：技術攀登（5.0 - 5.15d）
   - 細分等級說明（5.10a, 5.10b, 5.10c, 5.10d）

3. **難度對照表**
   | YDS 等級 | 難度描述 | 適合對象 |
   |----------|----------|----------|
   | 5.5-5.7 | 入門級 | 初學者 |
   | 5.8-5.9 | 初級 | 有基礎者 |
   | 5.10a-5.10d | 中級 | 定期攀爬者 |
   | 5.11a-5.11d | 中高級 | 認真訓練者 |
   | 5.12a-5.12d | 高級 | 資深攀岩者 |
   | 5.13a+ | 專業級 | 職業選手 |

#### 2.2 法式分級系統（French Scale）
1. **起源與使用地區**
   - 法國發展
   - 歐洲、亞洲廣泛使用
   - 台灣岩館常見

2. **分級結構**
   - 數字 + 字母（1 - 9c）
   - 加號（+）表示該等級較難端

3. **難度說明**
   | 法式等級 | 難度描述 | 對應 YDS |
   |----------|----------|----------|
   | 3-4 | 入門級 | 5.4-5.6 |
   | 5a-5c | 初級 | 5.7-5.9 |
   | 6a-6c+ | 中級 | 5.10a-5.11c |
   | 7a-7c+ | 中高級 | 5.11d-5.13a |
   | 8a-8c+ | 高級 | 5.13b-5.14d |
   | 9a+ | 世界級 | 5.15a+ |

#### 2.3 UIAA 分級系統
1. **簡介**
   - 國際山岳聯盟制定
   - 使用羅馬數字（I - XII）
   - 歐洲部分地區使用

### 三、抱石分級系統（約 500 字）

#### 3.1 V 級系統（Hueco Scale）
1. **起源與使用地區**
   - 美國 Hueco Tanks 發展
   - 北美、台灣常用

2. **分級結構**
   - VB（最簡單）到 V17（目前最難）
   - VB = V-Beginner

3. **難度說明**
   | V 等級 | 難度描述 | 適合對象 |
   |--------|----------|----------|
   | VB-V0 | 入門級 | 初學者 |
   | V1-V3 | 初級 | 有基礎者 |
   | V4-V6 | 中級 | 定期訓練者 |
   | V7-V9 | 高級 | 資深抱石者 |
   | V10+ | 專業級 | 職業選手 |

#### 3.2 Fontainebleau 系統（Font Scale）
1. **起源與使用地區**
   - 法國楓丹白露森林
   - 歐洲、國際比賽常用

2. **分級結構**
   - 數字 + 字母（1 - 9A）
   - 與法式路線分級相似但獨立

3. **與 V 級對照**
   | Font 等級 | V 等級 |
   |-----------|--------|
   | 4 | VB-V0 |
   | 5 | V0-V1 |
   | 6A-6C+ | V2-V5 |
   | 7A-7C+ | V6-V10 |
   | 8A-8C+ | V11-V16 |

### 四、台灣常見的分級使用情況（約 300 字）

1. **室內岩館**
   - 多數使用顏色區分難度
   - 搭配 V 級或法式標示
   - 各館定線風格差異

2. **戶外岩場**
   - 龍洞：YDS 為主
   - 部分場地使用法式分級

3. **台灣特有現象**
   - 「軟」與「硬」的評價
   - 不同岩館的難度校準差異

### 五、分級的主觀性與注意事項（約 250 字）

1. **影響難度感受的因素**
   - 身高臂展
   - 攀爬風格
   - 路線類型（crimpy、slopey、overhang）

2. **常見誤區**
   - 過度執著於等級數字
   - 不同場地難度不可直接比較
   - 戶外與室內的難度差異

3. **正確心態**
   - 分級僅供參考
   - 專注於進步而非數字
   - 享受攀爬過程

### 七、結語（約 100 字）
- 理解分級是攀岩溝通的基礎
- 建議從自己常去的場地熟悉起
- 持續攀爬自然會建立難度感
