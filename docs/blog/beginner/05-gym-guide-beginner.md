### 二、岩館類型介紹（約 300 字）

#### 2.1 抱石館
- **特色**: 只有抱石牆，無繩索設施
- **高度**: 約 3-5 公尺
- **優點**:
  - 入門門檻最低
  - 不需確保者
  - 可獨自前往
- **適合**: 新手、喜歡解題者

#### 2.2 綜合攀岩館
- **特色**: 同時有抱石區與繩索區（先鋒/頂繩）
- **高度**: 繩索區可達 10-15 公尺以上
- **優點**:
  - 可體驗多種類型
  - 設施較完整
- **注意**: 繩索區需確保技術

#### 2.3 健身房附設
- **特色**: 健身房內的小型攀岩區
- **規模**: 通常較小
- **優點**: 可結合其他運動

### 三、如何選擇適合的岩館（約 350 字）

#### 3.1 考量因素

| 因素 | 說明 |
|------|------|
| 距離 | 通勤時間影響練習頻率 |
| 費用 | 單次、月票、年票比較 |
| 路線數量 | 換線頻率決定新鮮感 |
| 換線頻率 | 多久更新一次路線 |
| 營業時間 | 配合個人作息 |
| 氛圍 | 新手友善程度 |

#### 3.2 新手友善指標
- 有無教練課程
- 工作人員態度
- 入門路線比例
- 是否有教學區

#### 3.3 建議做法
- 先查看 Google 評價
- 可先購買單次體驗
- 多嘗試幾家再決定

### 四、入場流程完整說明（約 400 字）

#### 4.1 第一次入場步驟

1. **抵達與登記**
   - 填寫個人資料
   - 簽署安全同意書
   - 聽取安全須知

2. **費用支付**
   - 入場費（依岩館定價）
   - 裝備租借費（岩鞋為主）
   - 部分岩館提供首次體驗價

3. **更衣與準備**
   - 更換運動服裝
   - 租借岩鞋
   - 存放個人物品

4. **暖身區**
   - 使用岩館提供的暖身設施
   - 進行關節活動與伸展

5. **開始攀爬**
   - 從最簡單的路線開始
   - 可詢問工作人員建議

#### 4.2 常見費用結構

| 項目 | 價格範圍 |
|------|----------|
| 單次入場 | $300-500 |
| 岩鞋租借 | $50-100 |
| 粉袋租借 | $30-50 |
| 月票 | $1,500-2,500 |
| 新手體驗課 | $500-1,000 |

*實際價格依各岩館為準*

---

### 五、裝備租借指南（約 300 字）

#### 5.1 岩鞋租借
- **選擇尺寸**: 略緊但不痛
- **試穿要點**:
  - 腳趾碰到鞋頭
  - 沒有明顯空隙
  - 可忍受的緊度
- **衛生考量**: 可穿薄襪或赤腳

#### 5.2 粉袋與碳酸鎂粉
- 多數岩館提供公用粉缸
- 新手初期可不使用
- 手汗嚴重者可考慮租借

#### 5.3 何時該買自己的裝備
- 確定持續練習後（約 5-10 次）
- 岩鞋是第一優先購買
- 詳見「裝備入門」文章

### 六、岩館禮儀與規範（約 400 字）

#### 6.1 安全相關

- **不站在他人下方**
  - 隨時注意上方攀爬者
  - 等待前一個人結束再靠近

- **確認落地區淨空**
  - 墜落前喊「落」
  - 觀察周圍有無物品或人

- **一條路線一個人**
  - 避免同時攀爬相近路線
  - 互相禮讓

#### 6.2 社交禮儀

- **不主動給 Beta**
  - 除非對方詢問
  - 避免「Spray Beta」行為

- **分享岩壁空間**
  - 不要長時間佔用熱門路線
  - 嘗試幾次後讓給他人

- **控制音量**
  - 加油鼓勵可以
  - 避免過度喧嘩

#### 6.3 環境維護

- **清理碳酸鎂粉**
  - 使用刷子清潔岩點
  - 不過度使用

- **愛護設施**
  - 不踩踏軟墊外區域
  - 使用後歸位

- **保持整潔**
  - 垃圾分類
  - 更衣室整理

### 七、課程與訓練資源（約 250 字）

#### 7.1 新手體驗課
- **內容**: 基礎安全、攀爬技巧
- **時長**: 通常 1-2 小時
- **對象**: 完全新手
- **費用**: $500-1,000

#### 7.2 基礎課程
- **內容**: 進階技巧、動作分析
- **時長**: 多堂連續課程
- **對象**: 已有數次經驗者

#### 7.3 確保課程（繩索區）
- **內容**: 確保技術、安全檢查
- **必要性**: 使用繩索區必學
- **考核**: 通過後取得確保資格

#### 7.4 私人教練
- **優點**: 客製化指導
- **費用**: 較高但進步快
- **適合**: 想快速進步者

### 八、台灣岩館分佈概況（約 200 字）

#### 8.1 北部地區
- 台北市、新北市密度最高
- 多種類型岩館可選
- 競爭激烈、設施新穎

#### 8.2 中部地區
- 台中市逐漸發展
- 選擇相對較少
- 多為綜合型岩館

#### 8.3 南部地區
- 高雄市發展中
- 新岩館陸續開幕
- 社群逐漸壯大

#### 8.4 查詢資源
- NobodyClimb 岩館搜尋功能
- Google Maps 搜尋「攀岩」
- 各岩館社群媒體

### 九、常見問題 FAQ（約 200 字）

**Q1: 一個人去岩館可以嗎？**
> 可以！抱石不需要確保者，很多人都是獨自練習。

**Q2: 第一次去會很難融入嗎？**
> 多數岩友都很友善，不用擔心。可以從工作人員開始交流。

**Q3: 多久去一次比較好？**
> 建議一週 2-3 次，讓身體有恢復時間。

**Q4: 岩館會很吵雜嗎？**
> 有時會有音樂，但通常氣氛輕鬆。每間岩館風格不同。

**Q5: 可以拍照錄影嗎？**
> 多數岩館允許，但請注意不要拍到不想入鏡的人。
