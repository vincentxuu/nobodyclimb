# 攀岩訓練系列文章

本系列涵蓋攀岩訓練的核心主題，從指力訓練到週期化規劃，幫助攀岩者系統性提升能力。

| 04 | [攀岩耐力訓練方法](./04-endurance-training.md) | ARC、4x4、間歇訓練 | 全階段攀岩者 | 2,500-3,000 |

## 系列特色

- **科學基礎**：引用運動科學研究與專業訓練理論
- **實用導向**：提供可直接執行的訓練計畫
- **漸進式學習**：從基礎概念到進階技巧的完整路徑
- **安全第一**：強調正確姿勢與傷害預防

## 建議閱讀順序

### 初學者路徑
1. 攀岩肌力訓練計畫（建立基礎肌力）
2. 攀岩耐力訓練方法（提升基礎耐力）
3. 攀岩指力訓練完整指南（至少攀岩6個月後）

### 進階者路徑
1. 週期化訓練完整解析（規劃年度訓練）
2. 攀岩指力訓練完整指南（系統化指力訓練）
3. 依據週期選擇肌力或耐力訓練
