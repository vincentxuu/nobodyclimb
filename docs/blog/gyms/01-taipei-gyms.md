## 內容大綱

### 一、前言（約 300 字）

1. 台北攀岩運動蓬勃發展現況
2. 為什麼選擇室內攀岩館入門
3. 本文涵蓋範圍與選館建議

### 二、台北岩館總覽表（約 200 字）

快速比較表格：
- 岩館名稱
- 所在區域
- 攀岩類型（抱石/上攀/速度）
- 票價範圍
- 捷運可達性
- 適合族群標籤

### 三、各岩館詳細介紹（約 2,500 字）

#### 3.1 原岩攀岩館（約 400 字）

**基本資訊**
- 內湖店、南港店、中和店、萬華店位置
- 營業時間與票價
- 交通方式

**設施特色**
- 各分店牆面規模
- 路線難度分布
- 定期換線頻率

**適合族群**
- 連鎖品牌，各區都能找到
- 初學者課程完善
- 社群活動豐富

#### 3.2 CORNER Bouldering Gym（約 400 字）

**基本資訊**
- 地址：台北市信義區
- 營業時間與票價
- 交通方式

**設施特色**
- 寬敞明亮的空間設計
- 牆面坡度多樣
- 咖啡廳複合空間

**適合族群**
- 注重空間氛圍者
- 喜歡社交型攀岩
- 拍照打卡需求

#### 3.3 市民抱石攀岩館（約 400 字）

**基本資訊**
- 地址：台北市內湖區運動中心
- 營業時間與票價（運動中心價格）
- 交通方式

**設施特色**
- 公營運動中心附設
- 票價親民
- 場地規模

**適合族群**
- 預算有限的入門者
- 附近居民
- 搭配游泳、健身一起運動

#### 3.4 MegaSTONE Climbing Gym（約 400 字）

**基本資訊**
- 地址與分店資訊
- 營業時間與票價
- 交通方式

**設施特色**
- 牆面高度與規模
- 上攀與抱石兼具
- 訓練設施

**適合族群**
- 想體驗上攀者
- 進階訓練需求
- 想參加比賽者

#### 3.5 Camp4 攀岩館（約 400 字）

**基本資訊**
- 地址：台北市
- 營業時間與票價
- 交通方式

**設施特色**
- 場館特色
- 路線風格
- 社群氛圍

**適合族群**
- 特定風格偏好者
- 社群導向攀岩者

#### 3.6 Redrock 紅石攀岩館（約 400 字）

**基本資訊**
- 地址與位置
- 營業時間與票價
- 交通方式

**設施特色**
- 牆面類型
- 路線特色
- 附加設施

**適合族群**
- 適合的攀岩者類型
- 鄰近區域居民

### 四、選館建議（約 300 字）

依需求推薦：
1. **新手首攀推薦**：課程完善、氛圍友善的岩館
2. **進階訓練推薦**：設施完整、路線挑戰高的岩館
3. **交通便利推薦**：捷運站周邊可達的岩館
4. **預算考量推薦**：票價親民的選擇
5. **親子同樂推薦**：適合帶小朋友的岩館

### 五、實用資訊（約 200 字）

1. 首次攀岩準備事項
2. 岩館禮儀注意事項
3. 建議攜帶物品清單
4. 票券優惠資訊

### 六、結語（約 100 字）

鼓勵讀者實際體驗，歡迎留言分享心得

## 寫作備註

### 資料蒐集待辦
- [ ] 確認各岩館最新票價
- [ ] 拍攝或取得岩館照片授權
- [ ] 訪談各岩館工作人員
- [ ] 實地體驗並記錄心得
- [ ] 確認營業時間（疫情後可能調整）

### 注意事項
- 票價資訊需標註查詢日期
- 避免過度主觀評價
- 提供平衡的各館介紹
- 鼓勵讀者多方體驗
