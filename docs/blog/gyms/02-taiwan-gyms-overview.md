## 寫作備註

### 資料蒐集待辦
- [ ] 各岩館最新票價確認（建議致電或官網查詢）
- [ ] 東部岩館資訊較少，需額外調查
- [ ] 各岩館營業時間確認
- [ ] 收集各岩館實拍照片或取得授權
- [ ] 確認各岩館設施類型（抱石/上攀/速度）
- [ ] 訪問各地攀岩者取得第一手心得

### 更新維護
- 建議每季更新票價資訊
- 新開幕岩館及時補充
- 結束營業岩館標註說明
- 標註資料更新日期

### 注意事項
- 票價資訊需標註查詢日期，提醒讀者以官方公告為準
- 避免排名式比較，減少爭議
- 各區平衡報導，不偏重北部
- 提供客觀資訊，避免過度主觀評價
- 鼓勵讀者實地體驗，多方比較
