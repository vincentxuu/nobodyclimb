## 內容規範

### 文章結構

每篇岩館介紹文章應包含：

1. **基本資訊**
   - 岩館名稱、地址、營業時間
   - 票價（首攀、單次、月票）
   - 交通方式

2. **設施介紹**
   - 攀岩牆類型（抱石、上攀、速度）
   - 牆面高度與坡度
   - 路線數量與難度分布
   - 其他設施（訓練區、休息區）

3. **適合族群**
   - 初學者友善度
   - 進階訓練適合度
   - 親子活動適合度

4. **特色與評價**
   - 該岩館獨特優勢
   - 社群氛圍
   - 課程服務
