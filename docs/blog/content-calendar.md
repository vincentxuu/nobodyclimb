# 時事行銷搭配規劃表

> 運用時事作為「行銷鉤子」，帶動常青文章流量

## 使用方式

1. 關注攀岩時事動態
2. 找到對應的時事類型
3. 搭配推薦文章進行推廣
4. 在社群媒體發布時事貼文，連結相關文章

---

## 社群發文模板

### 時事貼文格式

```
🧗 [時事標題]

[簡短時事說明，1-2 句]

想了解更多？推薦閱讀：
📖 [文章標題1] - [簡短說明]
📖 [文章標題2] - [簡短說明]

#攀岩 #Climbing #NobodyClimb #[相關標籤]
```

### 範例

```
🧗 Alex Honnold 來台灣了！

傳奇攀岩家 Alex Honnold 將於 1/24 來台進行活動，
還不認識他？先來補課！

📖 Free Solo 徒手攀岩 - 奧斯卡最佳紀錄片
📖 當代最強攀岩選手 - 認識 Honnold 的傳奇故事
📖 攀岩心理訓練 - 他如何克服恐懼？

#攀岩 #AlexHonnold #FreeSolo #NobodyClimb
```
