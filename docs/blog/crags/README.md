## 分類標籤

### 依岩質分類
- **砂岩**: 龍洞
- **石灰岩**: 關子嶺
- **珊瑚礁岩**: 壽山、墾丁
- **溪谷岩場**: 德芙蘭

### 依地區分類
- **北部**: 龍洞
- **中部**: 德芙蘭
- **南部**: 關子嶺、壽山、墾丁

### 依難度分類
- **入門友善** (5.6-5.10): 龍洞（校門口區）、關子嶺
- **進階挑戰** (5.11-5.12): 龍洞、德芙蘭
- **高手路線** (5.13+): 龍洞、關子嶺

### 依季節分類
- **夏季適合**: 龍洞（注意防曬）
- **冬季首選**: 關子嶺、壽山、墾丁
- **四季皆宜**: 德芙蘭（溪谷涼爽）
