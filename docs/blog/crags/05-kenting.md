## 岩場概述

- **地點**: 屏東縣恆春鎮墾丁地區
- **岩質**: 珊瑚礁石灰岩
- **特色**: 海邊攀岩、度假氛圍、結合水上活動
- **難度範圍**: 5.6 - 5.11
- **攀登類型**: 運動攀登、Top Rope、抱石
- **最佳季節**: 10月 - 4月（避開落山風最強時期）

### 5. 季節與天氣（約 300 字）
- **最佳季節**: 10月 - 4月
- **落山風影響**
  - 10月-3月落山風季節
  - 風大時不適合攀岩
  - 選擇避風岩點
- **夏季問題**
  - 酷熱難耐
  - 颱風季節
  - 岩壁可能過熱
- **冬季優勢**
  - 南部冬季溫暖

- **最佳條件**
  - 無風或微風
  - 陰天或傍晚
  - 乾燥天氣
