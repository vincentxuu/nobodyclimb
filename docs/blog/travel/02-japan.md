### 1. 日本攀岩概述
- 日本攀岩文化與歷史

- 適合台灣攀岩者的理由

### 2. 小川山（Ogawayama）- 日本優勝美地

#### 2.1 目的地介紹
- 位置：長野縣川上村，海拔約 1,500 公尺
- 被譽為「日本的優勝美地」（Yosemite of Japan）
- 日本最具代表性的花崗岩抱石聖地
- 攀岩季節：4月下旬至11月（最佳：5-6月、9-10月）

#### 2.2 岩質與特色
- **岩質**：花崗岩（Granite）
- **特色**：巨大圓潤的花崗岩巨石、獨特的摩擦點攀登
- **風格**：技巧型、平衡感、摩擦攀登

#### 2.3 路線數量與難度
| 項目 | 數據 |
|------|------|
| 路線總數 | 700+ 條抱石路線 |
| 難度範圍 | 3級 至 初段以上（日本級數） |
| 經典路線 | エイハブ船長（初段）、穴社員（一級）、忍者返し（三級） |

#### 2.4 交通方式
- 從東京出發：JR 中央本線至小淵沢站，轉乘小海線至信濃川上站
- 車程約 3 小時
- 建議租車自駕更為便利

#### 2.5 住宿選擇
- **廻り目平營地**（Mawarime-daira）：攀岩者聚集的露營區
- 營地費用：約 700-1,000 日圓/人/晚
- 周邊民宿與旅館
- 小木屋租借

#### 2.6 費用預算（7天行程參考）
| 項目 | 費用（日圓） | 費用（台幣） |
|------|-------------|-------------|
| 機票 | - | 8,000-15,000 |
| 交通 | 15,000-25,000 | 3,500-6,000 |
| 住宿 | 5,000-15,000 | 1,200-3,500 |
| 餐食 | 10,000-20,000 | 2,400-4,800 |
| 其他 | 5,000 | 1,200 |
| **總計** | - | **16,000-30,000** |

### 3. 瑞牆山（Mizugaki）

#### 3.1 目的地介紹
- 位置：山梨縣北杜市，海拔 2,230 公尺
- 日本百名山之一
- 壯觀的花崗岩岩壁與抱石區
- 攀岩季節：5月至10月

#### 3.2 岩質與特色
- **岩質**：花崗岩
- **特色**：巨大岩壁的多繩距攀登、高品質抱石
- **風格**：傳統攀登、抱石

#### 3.3 路線數量與難度
| 項目 | 數據 |
|------|------|
| 抱石路線 | 300+ 條 |
| 多繩距路線 | 數十條經典路線 |
| 難度範圍 | 入門至高階 |

#### 3.4 交通與住宿
- 從東京出發約 2.5 小時車程
- 瑞牆山莊（山屋住宿）
- 周邊露營區

### 4. 城ヶ崎（Jogasaki）- 海蝕岩攀登

#### 4.1 目的地介紹
- 位置：靜岡縣伊東市，伊豆半島東岸
- 壯觀的海蝕岩柱與懸崖
- 日本著名的海岸攀登勝地
- 全年可攀，冬季溫暖是最佳季節

#### 4.2 岩質與特色
- **岩質**：火山岩 / 安山岩
- **特色**：海蝕形成的獨特地形、潮汐影響部分路線
- **風格**：運動攀登、Deep Water Solo（DWS）

#### 4.3 路線數量與難度
| 項目 | 數據 |
|------|------|
| 路線總數 | 200+ 條 |
| 難度範圍 | 5.6 至 5.14（YDS） |
| 經典區域 | 海金剛、シーサイド、門脇崎 |

#### 4.4 交通與住宿
- 從東京出發：JR 特急踊り子號至伊豆高原站
- 車程約 2 小時
- 伊豆高原周邊民宿、溫泉旅館
- 可結合溫泉旅行

### 5. 其他推薦岩場

#### 5.1 御岳（Mitake）
- 位置：東京都青梅市
- 特色：距離東京最近的抱石區
- 路線：200+ 條河邊抱石
- 交通：從新宿約 1.5 小時

#### 5.2 笠間（Kasama）
- 位置：茨城縣笠間市
- 特色：花崗岩抱石、適合新手
- 交通：從東京約 1.5 小時

#### 5.3 二子山（Futago-yama）
- 位置：埼玉縣小鹿野町
- 特色：石灰岩運動攀登
- 路線：經典的日本石灰岩

### 6. 日本攀岩實用資訊

#### 6.1 裝備租借
- 日本攀岩用品店（B-Pump、Gravity Research 等）
- Crash Pad 租借服務
- 當地攀岩嚮導

#### 6.2 攀岩地圖與指南書
- 「Rock & Snow」雜誌
- 各岩場專用 Topo（地形圖）
- 日本山岳協會資源

#### 6.3 注意事項
- 部分岩場有入山規定
- 停車場使用禮儀
- 熊出沒區域注意（小川山等）
- 天氣與季節考量

#### 6.4 結合觀光
- 溫泉旅行
- 日本料理體驗
- 其他戶外活動（登山、露營）
