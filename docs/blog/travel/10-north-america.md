## 內容大綱

### 1. 北美攀岩概述
- 北美攀岩的歷史地位

- 美國國家公園系統與攀岩

### 2. 優勝美地（Yosemite）- 大岩壁聖地

#### 2.1 目的地介紹
- 位置：加州中部，舊金山東方約 300 公里
- 世界最著名的大岩壁攀登聖地
- 現代攀岩運動的精神故鄉
- 攀岩季節：4-6月、9-11月（最佳）

#### 2.2 岩質與特色
- **岩質**：花崗岩（Granite）
- **特色**：
  - 世界級的大岩壁（El Capitan、Half Dome）
  - 完美的花崗岩裂隙系統
  - 豐富的攀岩歷史與文化
  - Camp 4 傳奇營地
- **風格**：傳統攀登、大岩壁、抱石

#### 2.3 路線數量與難度
| 項目 | 數據 |
|------|------|
| 路線總數 | 3,000+ 條 |
| 難度範圍 | 5.4 至 5.15（YDS） |
| 經典路線 | The Nose（5.14a, 31 繩距）、Snake Dike（5.7）、Astroman（5.11c） |
| 主要岩壁 | El Capitan, Half Dome, Washington Column |

#### 2.4 主要攀登區域

**El Capitan（酋長岩）**
- 高度：約 900 公尺
- 世界最著名的大岩壁
- The Nose、Salathé Wall、Dawn Wall
- 需要大岩壁經驗

**Half Dome（半圓頂）**
- 經典的 Snake Dike 路線
- Regular Northwest Face
- 需要許可證（部分路線）

**Camp 4 周邊抱石**
- 傳奇的攀岩者營地
- 經典抱石區
- Midnight Lightning（V8）

#### 2.5 交通方式
- 從台灣出發：飛舊金山或洛杉磯
- 舊金山至優勝美地：租車約 4 小時
- 公園內有免費接駁巴士

#### 2.6 住宿選擇
- **Camp 4**：傳奇攀岩者營地（先到先得）
- 公園內旅館（需提前預訂）
- El Portal 等周邊小鎮
- 費用：$15-300/晚（差距大）

#### 2.7 費用預算（10天行程參考）
| 項目 | 費用（美金） | 費用（台幣） |
|------|-------------|-------------|
| 機票 | - | 30,000-60,000 |
| 租車 | 400-800 | 12,800-25,600 |
| 住宿 | 150-500 | 4,800-16,000 |
| 餐食 | 200-400 | 6,400-12,800 |
| 公園門票 | 35 | 1,100 |
| **總計** | - | **55,000-115,000** |

### 3. Squamish - 加拿大花崗岩天堂

#### 3.1 目的地介紹
- 位置：加拿大卑詩省，溫哥華北方約 65 公里
- 北美第二大花崗岩巨石（The Chief）
- 世界級的多樣攀登選擇
- 攀岩季節：5-10月（最佳：7-9月）

#### 3.2 岩質與特色
- **岩質**：花崗岩（Granite）
- **特色**：
  - 壯觀的 Stawamus Chief 岩壁
  - 完美的裂隙攀登
  - 多樣難度選擇
  - 優美的自然環境
- **風格**：傳統攀登、運動攀登、抱石

#### 3.3 路線數量與難度
| 項目 | 數據 |
|------|------|
| 路線總數 | 3,000+ 條 |
| 難度範圍 | 5.4 至 5.14（YDS） |
| 經典路線 | University Wall（5.10b）、Angel's Crest（5.10a）|
| 主要區域 | The Chief, Murrin Park, Shannon Falls |

#### 3.4 The Chief 三峰

**First Peak**
- 最多路線
- 各級距選擇
- 經典 Squamish Buttress

**Second Peak**
- 技術性路線
- University Wall 所在地

**Third Peak**
- 較少人跡
- 優質路線

#### 3.5 交通與住宿
- 溫哥華機場出發租車約 1 小時
- Squamish 小鎮住宿
- 露營區選擇多
- 費用：$30-200 CAD/晚

### 4. Bishop - 抱石聖地

#### 4.1 目的地介紹
- 位置：加州東部，優勝美地東南方
- 北美最著名的抱石區之一
- High Sierra 山腳下的沙漠環境
- 攀岩季節：10-5月（最佳：10-11月、3-4月）

#### 4.2 岩質與特色
- **岩質**：火山凝灰岩（Volcanic Tuff）
- **特色**：
  - 獨特的火山岩抱石
  - 大量的高品質問題
  - 壯觀的山脈背景
  - Crash Pad 文化發源地
- **風格**：純抱石

#### 4.3 路線數量與難度
| 項目 | 數據 |
|------|------|
| 問題總數 | 2,500+ 條 |
| 難度範圍 | V0 至 V15 |
| 主要區域 | Buttermilks, Happy Boulders, Sad Boulders, Volcanic Tablelands |
| 經典路線 | Buttermilker（V4）、The Mandala（V12）|

#### 4.4 主要區域

**Buttermilks**
- 最具代表性的區域
- 大型 Highball 抱石
- 經典的 Peabody 巨石

**Happy/Sad Boulders**
- 較小的石頭
- 各級距選擇
- 適合暖身

**Volcanic Tablelands**
- 獨特的火山岩
- 不同的攀登風格

#### 4.5 交通與住宿
- 從洛杉磯開車約 5 小時
- 從優勝美地開車約 4 小時
- Bishop 小鎮住宿
- BLM 免費露營（部分區域）
- 費用：$0-150/晚

### 5. 其他北美攀岩目的地

#### 5.1 Red River Gorge（肯塔基州）
- 美國東部最佳運動攀登
- 砂岩懸岩
- 2,000+ 路線
- 季節：3-5月、9-11月

#### 5.2 Joshua Tree（加州）
- 獨特的約書亞樹沙漠
- 花崗岩傳統攀登與抱石
- 8,000+ 路線
- 季節：10-4月

#### 5.3 Indian Creek（猶他州）
- 世界最佳裂隙攀登
- 砂岩平行裂隙
- 需要完整傳攀裝備
- 季節：3-5月、9-11月

#### 5.4 Smith Rock（奧勒岡州）
- 美國運動攀登發源地
- 火山凝灰岩
- 1,800+ 路線
- 季節：3-5月、9-11月

#### 5.5 Hueco Tanks（德州）
- 世界級抱石
- 獨特的 Hueco（凹洞）
- 需要許可證
- 季節：10-3月

### 6. 北美攀岩實用資訊

#### 6.1 交通規劃
- 租車：北美攀岩基本需要自駕
- 國際駕照：台灣駕照 + 國際駕照
- 長途駕駛準備

#### 6.2 住宿選擇
- 國家公園營地（需預訂）
- BLM/國家森林免費露營
- 小鎮旅館
- Airbnb

#### 6.3 攀岩資源
- **Mountain Project**：北美最完整路線資料庫
- **Climbing.com**：攀岩新聞與報導
- 各區域指南書

#### 6.4 簽證資訊
- ESTA 電子旅行授權（美國）
- eTA 電子旅行授權（加拿大）
- 需在出發前申請

#### 6.5 注意事項
- 國家公園規定（許可證、露營限制）
- 野生動物（熊、山獅）
- 高海拔適應（部分地區）
- 沙漠環境準備（水、防曬）

### 7. 最佳季節建議
| 地點 | 最佳季節 | 氣溫 | 備註 |
|------|----------|------|------|
| 優勝美地 | 4-6月、9-11月 | 10-25°C | 避開夏季高溫人潮 |
| Squamish | 7-9月 | 15-25°C | 夏季最佳 |
| Bishop | 10-11月、3-4月 | 5-20°C | 避開夏季高溫 |
| Joshua Tree | 10-4月 | 10-25°C | 冬季為主 |
