### 2. 仁壽峰（Insubong）- 首爾近郊經典

#### 2.1 目的地介紹
- 位置：首爾北部北漢山國家公園內
- 高度：810 公尺，岩壁高達 200 公尺
- 韓國最具代表性的花崗岩大岩壁
- 交通便利，從首爾市區 1 小時可達
- 攀岩季節：3月至11月（最佳：4-5月、9-10月）

#### 2.2 岩質與特色
- **岩質**：花崗岩（Granite）
- **特色**：壯觀的花崗岩岩壁、裂隙系統發達
- **風格**：傳統攀登為主、部分運動攀登路線

#### 2.3 路線數量與難度
| 項目 | 數據 |
|------|------|
| 路線總數 | 100+ 條 |
| 難度範圍 | 5.4 至 5.13（YDS） |
| 經典路線 | Choi Route（5.8）、B Route（5.10a）、仁壽峰正面（5.9） |
| 繩距 | 多為 3-8 繩距的多繩距路線 |

#### 2.4 交通方式
- 從首爾出發：地鐵 3 號線至舊把撥站，轉乘巴士
- 總車程約 1-1.5 小時
- 步行至岩壁基地約 30-40 分鐘

#### 2.5 住宿選擇
- 首爾市區住宿，當日來回
- 北漢山周邊民宿
- 登山客旅館

#### 2.6 費用預算（5天行程參考）
| 項目 | 費用（韓元） | 費用（台幣） |
|------|-------------|-------------|
| 機票 | - | 6,000-12,000 |
| 交通 | 50,000-100,000 | 1,200-2,400 |
| 住宿 | 200,000-400,000 | 4,800-9,600 |
| 餐食 | 150,000-250,000 | 3,600-6,000 |
| 其他 | 50,000 | 1,200 |
| **總計** | - | **17,000-31,000** |

### 3. 雪嶽山（Seoraksan）

#### 3.1 目的地介紹
- 位置：江原道束草市，韓國東北部
- 韓國最美的國家公園之一
- 壯觀的花崗岩峰群
- 攀岩季節：5月至10月

#### 3.2 岩質與特色
- **岩質**：花崗岩
- **特色**：高山環境、壯麗景色
- **風格**：傳統攀登、高山攀登

#### 3.3 路線數量與難度
| 項目 | 數據 |
|------|------|
| 主要岩壁 | 蔚山岩（Ulsanbawi）、鬼面岩 |
| 路線數 | 50+ 條 |
| 難度範圍 | 5.6 至 5.12 |

#### 3.4 交通與住宿
- 從首爾出發：高速巴士至束草，約 2.5 小時
- 束草市區或雪嶽洞住宿
- 國家公園內山屋（需預約）

### 4. 釜山地區

#### 4.1 目的地介紹
- 位置：韓國第二大城市
- 海岸與山區攀岩兼具
- 氣候溫和，冬季也可攀登
- 結合都市旅遊

#### 4.2 主要岩場

**金井山（Geumjeongsan）**
- 釜山市區附近
- 花崗岩抱石與短路線
- 交通便利

**海雲台周邊**
- 海岸岩石
- 適合輕鬆攀登

#### 4.3 路線數量與難度
| 項目 | 數據 |
|------|------|
| 路線總數 | 80+ 條 |
| 難度範圍 | 入門至中階 |
| 類型 | 抱石、短程運動攀登 |

#### 4.4 交通與住宿
- 從台灣直飛釜山約 2.5 小時
- 釜山市區住宿選擇多
- 地鐵系統便利

### 5. 其他推薦岩場

#### 5.1 冠岳山（Gwanaksan）
- 位置：首爾南部
- 特色：距離首爾最近的攀岩場
- 適合：半日攀登行程

#### 5.2 月岳山（Woraksan）
- 位置：忠清北道
- 特色：石灰岩攀登
- 風格：運動攀登

#### 5.3 設岳山（Seoraksan）冬季冰攀
- 季節：12月至2月
- 特色：韓國最佳冰攀地點
- 需要：冰攀裝備與經驗

### 6. 韓國攀岩實用資訊

#### 6.1 裝備與嚮導
- 首爾攀岩用品店
- 當地攀岩社團
- 嚮導服務資訊

#### 6.2 攀岩資源
- 韓國山岳會資訊
- 當地攀岩 App（如 Climbmate）
- Facebook 攀岩社群

#### 6.3 注意事項
- 國家公園入園規定
- 部分路線需要許可
- 季節性封閉區域
- 天氣變化快速（尤其山區）

#### 6.4 結合觀光
- 韓國料理體驗
- 傳統文化景點
- 溫泉（汗蒸幕）
- 購物行程

### 7. 最佳季節建議
| 季節 | 氣溫 | 建議 |
|------|------|------|
| 春季（4-5月）| 10-20°C | 最佳攀岩季節 |
| 夏季（6-8月）| 25-35°C | 炎熱潮濕，避開 |
| 秋季（9-10月）| 10-20°C | 最佳攀岩季節 |
| 冬季（12-2月）| -10-5°C | 冰攀季節 |
