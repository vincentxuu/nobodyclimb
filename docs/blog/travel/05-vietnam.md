## 內容大綱

### 1. 越南攀岩概述
- 越南攀岩發展現況
- 新興的亞洲攀岩目的地
- 主要攀登類型：運動攀登、深水攀岩（DWS）、抱石
- 越南攀岩社群：VietClimb

### 2. 胡瓏（Huu Lung）- 越南攀岩首都

#### 2.1 目的地介紹
- 位置：諒山省（Lang Son），距河內約 120 公里
- 越南最大、最成熟的攀岩區域
- 壯觀的喀斯特石灰岩地形
- 攀岩季節：10月至4月（最佳：11月至2月）

#### 2.2 岩質與特色
- **岩質**：石灰岩（Limestone）
- **特色**：
  - 喀斯特塔狀山峰
  - 多樣的路線風格：面攀、懸岩、洞穴
  - 相對較新的開發區，路線品質優良
- **風格**：運動攀登為主

#### 2.3 路線數量與難度
| 項目 | 數據 |
|------|------|
| 路線總數 | 250+ 條 |
| 難度範圍 | 5a 至 8b+（法國級數）/ 5.4 至 5.14a（YDS） |
| 主要岩區 | Hon Ma, Heaven's Gate, Son Tien, Nui Hoanh |
| 經典路線 | 各難度級距都有經典 |

#### 2.4 主要岩場介紹

**Hon Ma（馬山）**
- 路線密集
- 適合各級距
- 交通便利

**Heaven's Gate（天門）**
- 壯觀的岩壁
- 中高難度路線
- 美麗的景觀

**Son Tien**
- 新開發區域
- 高品質路線
- 相對人少

#### 2.5 交通方式
- 從台灣出發：飛河內內排機場
- 河內至胡瓏：包車/巴士約 2.5-3 小時
- 當地交通：摩托車租借

#### 2.6 住宿選擇
- **VietClimb Hostel**：攀岩者專屬
- **當地民宿**：價格實惠
- **Homestay**：體驗在地生活
- 費用：100,000-400,000 越南盾/晚（約 130-520 台幣）

#### 2.7 費用預算（7天行程參考）
| 項目 | 費用（越南盾） | 費用（台幣） |
|------|---------------|-------------|
| 機票 | - | 6,000-12,000 |
| 交通 | 1,000,000-2,000,000 | 1,300-2,600 |
| 住宿 | 700,000-2,500,000 | 900-3,200 |
| 餐食 | 700,000-1,400,000 | 900-1,800 |
| 嚮導/租裝 | 500,000-1,500,000 | 650-1,950 |
| **總計** | - | **10,000-22,000** |

### 3. Cat Ba 與蘭哈灣（Lan Ha Bay）- 深水攀岩天堂

#### 3.1 目的地介紹
- 位置：海防市吉婆島（Cat Ba Island）
- 蘭哈灣：下龍灣旁的姊妹灣
- 世界級的深水攀岩（Deep Water Soloing）目的地
- 攀岩季節：4月至10月（水溫較暖）

#### 3.2 岩質與特色
- **岩質**：石灰岩（海上喀斯特地形）
- **特色**：
  - 壯觀的海上石灰岩島嶼
  - 從水面直接攀登
  - 獨特的深水攀岩體驗
- **風格**：深水攀岩（DWS）為主、抱石

#### 3.3 路線數量與難度
| 項目 | 數據 |
|------|------|
| DWS 路線 | 100+ 條 |
| 難度範圍 | 5a 至 8a（法國級數） |
| 特色 | 海上攀岩、無需繩索 |
| 落水高度 | 3-15 公尺不等 |

#### 3.4 深水攀岩須知
- **安全評估**：水深、潮汐、海況
- **必備裝備**：攀岩鞋、救生衣（建議）
- **團體活動**：建議有船隻接應
- **季節考量**：水溫與天氣

#### 3.5 交通方式
- 河內至 Cat Ba：巴士+渡輪約 4 小時
- 海防至 Cat Ba：快艇約 45 分鐘
- 島上交通：摩托車租借

#### 3.6 住宿選擇
- Cat Ba 鎮上旅館
- 海邊度假村
- 攀岩行程通常包含住宿

### 4. 其他越南攀岩地點

#### 4.1 河內周邊
- 位置：河內市郊
- 特色：一日遊攀岩
- 適合：短暫停留的攀岩者

#### 4.2 寧平（Ninh Binh）
- 位置：河內南方約 90 公里
- 特色：陸地下龍灣，喀斯特地形
- 開發中的攀岩潛力區

#### 4.3 峴港/會安周邊
- 位置：越南中部
- 特色：海岸攀岩潛力
- 結合海灘度假

### 5. 越南攀岩實用資訊

#### 5.1 VietClimb 社群
- 官網：https://vietclimb.vn/
- Facebook 社群
- 路線資訊與更新
- 當地嚮導聯繫

#### 5.2 裝備與服務
- VietClimb 裝備租借
- 當地攀岩嚮導
- DWS 行程預約（船隻安排）

#### 5.3 攀岩路線資訊
- TheCrag Vietnam 區
- VietClimb 路線指南
- 當地 Topo 購買

#### 5.4 注意事項
- 簽證：落地簽或電子簽
- 貨幣：越南盾（VND）
- 語言：英語在觀光區通用
- 交通：摩托車為主，注意安全

#### 5.5 健康與安全
- 飲水：瓶裝水
- 蚊蟲：防蚊措施
- 醫療：大城市醫療設施較完善
- 保險：建議購買旅遊險

#### 5.6 結合觀光
- 下龍灣遊船
- 河內古城
- 胡志明市
- 美食體驗：河粉、春捲、咖啡

### 6. 最佳季節建議
| 地點 | 最佳季節 | 氣溫 | 備註 |
|------|----------|------|------|
| 胡瓏 | 11月-2月 | 15-25°C | 乾季，涼爽 |
| Cat Ba/蘭哈灣 | 5月-9月 | 25-35°C | 水溫適合 DWS |
| 全越南 | 避開 6-9月 | - | 雨季較多 |
