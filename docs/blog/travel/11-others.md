## 內容大綱

### 1. 南美與其他地區攀岩概述
- 世界盡頭的攀岩聖地
- 探險性質較強的目的地
- 主要攀登類型：大岩壁、抱石、運動攀登
- 適合有經驗的攀岩者

### 2. 阿根廷 Patagonia - El Chaltén

#### 2.1 目的地介紹
- 位置：阿根廷聖克魯斯省，巴塔哥尼亞地區
- 世界最壯觀的高山攀登目的地之一
- Fitz Roy 與 Cerro Torre 雙峰
- 攀岩季節：12-3月（南半球夏季）

#### 2.2 岩質與特色
- **岩質**：花崗岩（Granite）
- **特色**：
  - 壯觀的巴塔哥尼亞花崗岩尖峰
  - 極端的天氣條件
  - 世界級的高山大岩壁
  - 需要高度技術與經驗
- **風格**：高山攀登、大岩壁、傳統攀登

#### 2.3 路線數量與難度
| 項目 | 數據 |
|------|------|
| 經典路線 | Fitz Roy（3,405m）、Cerro Torre（3,102m） |
| 難度範圍 | 高山技術路線（需完整高山經驗） |
| 特色路線 | Supercanaleta、Afanassieff Route |
| 要求 | 自給自足、高山救援能力 |

#### 2.4 主要攀登目標

**Fitz Roy**
- 巴塔哥尼亞最具代表性的山峰
- 高度：3,405 公尺
- 經典路線：Afanassieff、Franco-Argentine

**Cerro Torre**
- 世界最難攀登的山峰之一
- 高度：3,102 公尺
- 極端的天氣與冰雪條件

**Aguja 群峰**
- 較小的花崗岩尖峰
- 技術性攀登
- 相對較短的路線

#### 2.5 交通方式
- 從台灣出發：飛布宜諾斯艾利斯轉國內班機
- 布宜諾斯艾利斯至 El Calafate：約 3 小時飛行
- El Calafate 至 El Chaltén：巴士約 3 小時

#### 2.6 住宿與費用
- El Chaltén 小鎮住宿
- 基地營露營
- 費用較高（偏遠地區）
- 預算：10天約 $2,000-4,000 美金（不含機票）

### 3. 智利 Cochamó - 南美優勝美地

#### 3.1 目的地介紹
- 位置：智利湖區（Los Lagos），Puerto Montt 東北方
- 被譽為「南美的優勝美地」
- 壯觀的花崗岩峽谷
- 攀岩季節：11-4月（最佳：1-3月）

#### 3.2 岩質與特色
- **岩質**：花崗岩（Granite）
- **特色**：
  - 巨大的花崗岩岩壁
  - 原始的自然環境
  - 需要徒步進入
  - 較少開發，探險性質
- **風格**：傳統攀登、大岩壁

#### 3.3 路線數量與難度
| 項目 | 數據 |
|------|------|
| 路線總數 | 200+ 條 |
| 難度範圍 | 5.6 至 5.13（YDS） |
| 主要岩壁 | Anfiteatro, Trinidad, Cerro Capilla |
| 經典路線 | The Waiting（5.11b, 11繩距）|

#### 3.4 進入方式
- 從 Puerto Montt 至 La Junta：陸路交通
- La Junta 至攀岩區：徒步約 5-6 小時
- 需要背負裝備進入
- 牛隊可幫忙運送重裝

#### 3.5 住宿選擇
- La Junta 露營或小屋
- Refugio Cochamó（山屋）
- 需自備露營裝備
- 簡單的設施

### 4. 南非 Rocklands - 抱石聖地

#### 4.1 目的地介紹
- 位置：南非西開普省，開普敦北方約 300 公里
- 世界頂級的砂岩抱石區
- 壯觀的橘色砂岩巨石
- 攀岩季節：4-9月（南半球秋冬）

#### 4.2 岩質與特色
- **岩質**：砂岩（Sandstone）
- **特色**：
  - 獨特的橘紅色砂岩
  - 完美的摩擦攀登
  - 壯觀的沙漠景觀
  - 世界級的高難度問題
- **風格**：純抱石

#### 4.3 路線數量與難度
| 項目 | 數據 |
|------|------|
| 問題總數 | 1,500+ 條 |
| 難度範圍 | V0 至 V15 |
| 主要區域 | De Pakhuys, Tra Tra, Agter Pakhuis |
| 經典路線 | Black Shadow（V13）、Amandla（V11）|

#### 4.4 交通與住宿
- 飛開普敦機場
- 開普敦至 Rocklands：租車約 3 小時
- 露營為主
- De Pakhuys 農場住宿
- 費用：$10-30/晚

### 5. 其他世界攀岩目的地

#### 5.1 摩洛哥 Todra Gorge
- 位置：摩洛哥東南部
- 特色：壯觀的石灰岩峽谷
- 路線：300+ 條運動攀登
- 季節：10-5月

#### 5.2 約旦 Wadi Rum
- 位置：約旦南部沙漠
- 特色：紅砂岩沙漠攀登
- 風格：傳統攀登、抱石
- 季節：10-4月

#### 5.3 馬達加斯加 Tsaranoro
- 位置：馬達加斯加南部
- 特色：巨大的花崗岩岩壁
- 路線：大岩壁攀登
- 季節：4-10月

#### 5.4 澳洲 Blue Mountains
- 位置：雪梨西方約 80 公里
- 特色：砂岩傳統攀登
- 路線：2,000+ 條
- 季節：3-5月、9-11月

#### 5.5 墨西哥 El Potrero Chico
- 位置：墨西哥新萊昂州
- 特色：石灰岩運動攀登與多繩距
- 路線：1,000+ 條
- 季節：10-4月

### 6. 探險攀岩準備須知

#### 6.1 技術要求
- 完整的傳統攀登經驗
- 高山攀登基礎（部分地區）
- 自給自足能力
- 急救與救援知識

#### 6.2 裝備準備
- 完整傳攀裝備
- 高山裝備（視地區）
- 野營裝備
- 通訊與導航設備

#### 6.3 安全考量
- 偏遠地區救援困難
- 天氣變化快速
- 醫療設施有限
- 充足的保險覆蓋

#### 6.4 環境保護
- Leave No Trace 原則
- 尊重當地文化
- 支持當地社區
- 減少環境衝擊

### 7. 規劃探險攀岩旅行

#### 7.1 事前研究
- 詳細的路線資訊
- 當地氣候與季節
- 緊急撤退計劃
- 語言與文化準備

#### 7.2 嚮導服務
- 考慮雇用當地嚮導
- 了解當地規定
- 安全的第二道防線

#### 7.3 保險
- 確保涵蓋高山活動
- 救援費用覆蓋
- 醫療後送保障

### 8. 最佳季節建議
| 地點 | 最佳季節 | 氣溫 | 備註 |
|------|----------|------|------|
| Patagonia | 12月-3月 | 5-15°C | 南半球夏季，天氣仍不穩 |
| Cochamó | 1月-3月 | 10-20°C | 乾季 |
| Rocklands | 5月-8月 | 5-20°C | 南半球秋冬 |
| Wadi Rum | 10月-4月 | 10-25°C | 避開酷暑 |
