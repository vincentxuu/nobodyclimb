## 內容大綱

### 1. 寮國攀岩概述
- 寮國作為東南亞攀岩新興目的地
- 極高的性價比與優質的攀岩環境
- 主要攀登類型：運動攀登、深水攀岩
- Green Climbers Home 的重要性

### 2. Thakhek（他曲）- 寮國攀岩首都

#### 2.1 目的地介紹
- 位置：甘蒙省（Khammouane），寮國中部
- 東南亞最佳的運動攀登目的地之一
- Green Climbers Home 攀岩營地
- 攀岩季節：10月至4月（最佳：11月至2月）

#### 2.2 岩質與特色
- **岩質**：石灰岩（Limestone）
- **特色**：
  - 壯觀的喀斯特地形
  - 高品質的岩壁開發
  - 完善的攀岩營地設施
  - 極佳的攀岩者社群氛圍
- **風格**：運動攀登為主

#### 2.3 路線數量與難度
| 項目 | 數據 |
|------|------|
| 路線總數 | 477+ 條（持續增加）|
| 難度範圍 | 4 至 8c（法國級數）/ 5.4 至 5.14b（YDS） |
| 主要岩區 | Sleeping Wall, Tiger, Tian, Silver Cliff |
| 經典路線 | 各級距豐富選擇 |

#### 2.4 Green Climbers Home

**關於 Green Climbers Home**
- 寮國最著名的攀岩營地
- 由攀岩者經營，為攀岩者服務
- 完善的設施與社群氛圍
- 提供住宿、餐飲、裝備租借、嚮導服務

**設施與服務**
- 多種住宿選擇（宿舍、小屋、帳篷）
- 餐廳與酒吧
- 攀岩裝備租借
- 瑜伽課程
- 攀岩教學
- WiFi 與工作空間

**費用參考**
| 項目 | 費用（美金） | 費用（台幣） |
|------|-------------|-------------|
| 宿舍床位 | $5-8/晚 | 160-250 |
| 私人小屋 | $15-30/晚 | 470-940 |
| 餐食 | $3-6/餐 | 95-190 |
| 裝備租借 | $5-10/天 | 160-310 |

#### 2.5 主要岩區介紹

**Sleeping Wall**
- 適合各級距
- 交通便利
- 經典入門區

**Tiger**
- 較高難度路線
- 壯觀的懸岩
- 技術性要求高

**Tian**
- 美麗的岩壁
- 中等難度為主
- 景觀優美

**Silver Cliff**
- 較新開發區
- 高品質路線
- 持續增加中

#### 2.6 交通方式
- 從台灣出發：飛曼谷或河內轉機
- 曼谷至 Thakhek：過境泰國 Nakhon Phanom，再渡河
- 永珍至 Thakhek：巴士約 6-7 小時
- 機場接送可透過 Green Climbers Home 安排

#### 2.7 費用預算（14天行程參考）
| 項目 | 費用（美金） | 費用（台幣） |
|------|-------------|-------------|
| 機票 | - | 10,000-18,000 |
| 交通 | 50-100 | 1,600-3,100 |
| 住宿（14晚）| 70-350 | 2,200-11,000 |
| 餐食（14天）| 80-160 | 2,500-5,000 |
| 裝備租借 | 50-100 | 1,600-3,100 |
| **總計** | - | **18,000-40,000** |

### 3. Vang Vieng（萬榮）

#### 3.1 目的地介紹
- 位置：永珍省，寮國首都永珍北方約 160 公里
- 以戶外活動聞名的旅遊小鎮
- 攀岩、漂流、洞穴探險
- 攀岩季節：10月至4月

#### 3.2 岩質與特色
- **岩質**：石灰岩
- **特色**：
  - 壯觀的喀斯特山景
  - 結合多種戶外活動
  - 觀光資源豐富
- **風格**：運動攀登、深水攀岩（Blue Lagoon）

#### 3.3 路線數量與難度
| 項目 | 數據 |
|------|------|
| 路線總數 | 100+ 條 |
| 難度範圍 | 5a 至 7c（法國級數） |
| 主要岩區 | Sleeping Cave, Pha Poak |
| 特色 | DWS 體驗（Blue Lagoon） |

#### 3.4 交通與住宿
- 永珍至 Vang Vieng：巴士約 3-4 小時
- 住宿選擇豐富（從背包客到度假村）
- 費用：$5-50/晚

### 4. 寮國攀岩實用資訊

#### 4.1 Green Climbers Home 資源
- 官網：https://www.greenclimbershome.com/en/climbing
- 線上預訂
- 路線資訊下載
- 社群活動資訊

#### 4.2 攀岩資源
- TheCrag Laos 區
- Green Climbers Home 指南書
- 當地攀岩社群

#### 4.3 簽證資訊
- 寮國落地簽（30天）
- 費用：約 $30-42 美金（依國籍）
- 需準備：護照照片、填寫申請表

#### 4.4 注意事項
- **季節**：雨季（5-9月）岩壁潮濕
- **天氣**：乾季涼爽舒適
- **醫療**：鄉村地區醫療資源有限
- **網路**：營地有 WiFi，其他地方較弱

#### 4.5 健康與安全
- 飲水：瓶裝水
- 防蚊：重要
- 醫療：建議購買旅遊險
- 緊急：最近醫院在 Thakhek 市區

#### 4.6 長期停留建議
- Green Climbers Home 適合長住
- 工作交換機會（義工）
- 攀岩訓練營
- 數位游牧友善環境

### 5. 為什麼選擇寮國攀岩？

#### 5.1 優勢
- **極高性價比**：東南亞最便宜的攀岩目的地之一
- **優質路線**：精心開發的高品質路線
- **攀岩社群**：友善的國際攀岩者社群
- **專注攀岩**：遠離喧囂，專心攀登
- **長期訓練**：適合待上數週密集訓練

#### 5.2 適合對象
- 預算有限的攀岩者
- 想長期攀岩訓練的人
- 喜歡簡單生活的攀岩者
- 想認識國際攀岩朋友的人

### 6. 最佳季節建議
| 地點 | 最佳季節 | 氣溫 | 備註 |
|------|----------|------|------|
| Thakhek | 11月-2月 | 18-28°C | 乾季涼爽 |
| Vang Vieng | 11月-3月 | 15-28°C | 乾季 |
| 全寮國 | 避開 5-9月 | - | 雨季 |
