## 內容大綱

### 1. 中國攀岩概述
- 中國攀岩發展現況
- 世界級的攀岩資源
- 主要攀登類型：運動攀登、傳統攀登、抱石
- 台灣攀岩者前往注意事項

### 2. 陽朔（Yangshuo）- 亞洲運動攀登聖地

#### 2.1 目的地介紹
- 位置：廣西壯族自治區桂林市陽朔縣
- 世界著名的喀斯特地形攀岩勝地
- 亞洲路線數量最多的運動攀登區域之一
- 攀岩季節：全年可攀（最佳：10月至4月）

#### 2.2 岩質與特色
- **岩質**：石灰岩（Limestone）- 喀斯特地形
- **特色**：
  - 獨特的塔狀石灰岩山峰
  - 大量的洞穴與懸岩（Tufa）路線
  - 壯觀的自然景觀
- **風格**：運動攀登為主、部分傳統攀登、深水攀岩

#### 2.3 路線數量與難度
| 項目 | 數據 |
|------|------|
| 路線總數 | 1,000+ 條 |
| 難度範圍 | 5.5 至 5.15（YDS）/ 4 至 9b（法國級數） |
| 主要岩區 | 月亮山、白山、雷公岩、瑞士乳酪 |
| 經典路線 | Spicy Noodle（7a+）、White Mountain（各難度） |

#### 2.4 主要岩場介紹

**月亮山（Moon Hill）**
- 陽朔最具代表性的岩場
- 獨特的石灰岩拱門
- 路線難度：5.9 至 5.14

**白山（White Mountain）**
- 適合各級距攀岩者
- 大量中等難度路線
- 交通便利

**雷公岩（Lei Gong Yan）**
- 懸岩與 Tufa 路線
- 高難度路線集中地
- 需要較高技術水平

**蛋糕山（Cake Rock）**
- 適合新手與中階
- 路線密集
- 全日陰涼

#### 2.5 交通方式
- 從台灣出發：飛桂林兩江機場
- 機場至陽朔：巴士約 1.5 小時
- 陽朔當地：電動車租借、計程車

#### 2.6 住宿選擇
- **西街周邊**：熱鬧便利，選擇多
- **攀岩者旅館**：Climbers Inn、Rustic Inn 等
- **月亮山附近**：靠近岩場
- 費用：100-300 人民幣/晚

#### 2.7 費用預算（10天行程參考）
| 項目 | 費用（人民幣） | 費用（台幣） |
|------|---------------|-------------|
| 機票 | - | 8,000-15,000 |
| 交通 | 300-500 | 1,400-2,300 |
| 住宿 | 800-2,500 | 3,600-11,500 |
| 餐食 | 500-1,000 | 2,300-4,600 |
| 嚮導/租裝 | 300-800 | 1,400-3,700 |
| **總計** | - | **17,000-37,000** |

### 3. 雲南黎明（Liming）- 砂岩裂隙天堂

#### 3.1 目的地介紹
- 位置：雲南省麗江市玉龍縣黎明鄉
- 世界級的紅砂岩裂隙攀登區域
- 三江並流世界遺產區域內
- 攀岩季節：10月至5月（最佳：11月至3月）

#### 3.2 岩質與特色
- **岩質**：丹霞地貌紅砂岩（Sandstone）
- **特色**：
  - 完美的平行裂隙系統
  - 世界頂級的裂隙攀登（Crack Climbing）
  - 獨特的紅色岩壁景觀
- **風格**：傳統攀登為主

#### 3.3 路線數量與難度
| 項目 | 數據 |
|------|------|
| 路線總數 | 400+ 條 |
| 難度範圍 | 5.6 至 5.13（YDS） |
| 特色 | 各種寬度的裂隙：手指、手掌、拳頭、Off-width |
| 經典路線 | Flying Buttress、Thousand Kingdoms Wall |

#### 3.4 交通方式
- 從台灣出發：飛麗江三義機場或昆明轉機
- 麗江至黎明：包車約 3 小時

#### 3.5 住宿選擇
- 黎明當地客棧
- 攀岩者經營的住宿
- 費用：80-200 人民幣/晚

### 4. 其他推薦岩場

#### 4.1 格凸河（Getu）
- 位置：貴州省紫雲縣
- 特色：壯觀的石灰岩洞穴攀登
- 世界最大天然岩洞之一
- 難度：高階路線為主

#### 4.2 貴陽周邊
- 位置：貴州省貴陽市
- 特色：城市近郊攀岩
- 適合：短程攀岩旅行

#### 4.3 北京周邊
- 位置：北京市郊
- 主要岩場：白河、十渡
- 特色：花崗岩傳統攀登
- 適合：結合北京旅遊

#### 4.4 昆明周邊
- 位置：雲南省昆明市
- 特色：氣候舒適、岩場多
- 可作為前往黎明的中繼站

### 5. 中國攀岩實用資訊

#### 5.1 嚮導與裝備
- 陽朔當地攀岩店（Insight Adventures 等）
- 裝備租借服務
- 當地嚮導推薦

#### 5.2 攀岩資源
- TheCrag 中國區資料
- 微信攀岩群組
- China Climbing 網站

#### 5.3 台灣旅客注意事項
- 入境證件：台胞證（需提前辦理）
- 網路：VPN 使用（部分網站受限）
- 支付：支付寶/微信支付（需綁定）
- 現金：建議攜帶備用

#### 5.4 安全與保險
- 醫療設施（鄉村地區較有限）
- 旅遊保險建議
- 緊急聯絡方式

#### 5.5 結合觀光
- 陽朔：漓江遊船、十里畫廊、印象劉三姐
- 黎明：老君山、虎跳峽、麗江古城

### 6. 最佳季節建議
| 地點 | 最佳季節 | 氣溫 | 備註 |
|------|----------|------|------|
| 陽朔 | 10月-4月 | 10-25°C | 避開雨季與酷暑 |
| 黎明 | 11月-3月 | 5-20°C | 乾季，天氣穩定 |
| 北京 | 4-5月、9-10月 | 10-25°C | 避開極端氣溫 |
