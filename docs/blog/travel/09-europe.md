### 2. 法國楓丹白露（Fontainebleau）- 抱石聖地

#### 2.1 目的地介紹
- 位置：法國巴黎南方約 60 公里
- 世界最著名的抱石聖地，歷史超過百年
- 抱石運動的發源地之一
- 攀岩季節：全年可攀（最佳：3-5月、9-11月）

#### 2.2 岩質與特色
- **岩質**：砂岩（Sandstone）
- **特色**：
  - 獨特的砂岩圓石群
  - 技巧型、平衡感攀登
  - 豐富的路線歷史
  - 美麗的森林環境
- **風格**：純抱石

#### 2.3 路線數量與難度
| 項目 | 數據 |
|------|------|
| 路線總數 | 25,000+ 條 |
| 難度範圍 | 1 至 8C（Font 級數） |
| 主要區域 | Bas Cuvier, Franchard, Apremont, Elephant |
| 特色 | Circuit（色彩標記路線系統） |

#### 2.4 楓丹白露特有的 Circuit 系統
- 顏色標記代表不同難度
- 白/黃：入門
- 橘：中階
- 藍：中高階
- 紅：高階
- 黑/白：專家

#### 2.5 交通與住宿
- 巴黎至 Fontainebleau：火車約 40 分鐘
- 當地需要租車或自行車
- 住宿：Fontainebleau 鎮上、周邊民宿
- 費用：50-150 歐元/晚

#### 2.6 費用預算（7天行程參考）
| 項目 | 費用（歐元） | 費用（台幣） |
|------|-------------|-------------|
| 機票 | - | 25,000-45,000 |
| 交通 | 100-250 | 3,500-8,700 |
| 住宿 | 350-900 | 12,200-31,400 |
| 餐食 | 150-350 | 5,200-12,200 |
| 其他 | 50-100 | 1,700-3,500 |
| **總計** | - | **48,000-100,000** |

### 3. 希臘 Kalymnos - 運動攀登天堂

#### 3.1 目的地介紹
- 位置：希臘多德卡尼斯群島，愛琴海
- 歐洲最知名的運動攀登目的地之一
- 曾是海綿採集島，現為攀岩聖地
- 攀岩季節：3-5月、9-11月（最佳）

#### 3.2 岩質與特色
- **岩質**：石灰岩（Limestone）
- **特色**：
  - 高品質的灰色石灰岩
  - 大量的 Tufa 與洞穴路線
  - 希臘島嶼風情
  - 完善的攀岩基礎設施
- **風格**：運動攀登

#### 3.3 路線數量與難度
| 項目 | 數據 |
|------|------|
| 路線總數 | 3,500+ 條 |
| 難度範圍 | 4 至 9a+（法國級數） |
| 主要岩區 | Grande Grotta, Odyssey, Poets, Arginonta |
| 特色 | 大量中等難度優質路線 |

#### 3.4 交通與住宿
- 雅典飛 Kos 島，再轉渡輪
- 或從土耳其 Bodrum 搭船
- 住宿：Massouri 村為攀岩者聚集地
- 費用：30-100 歐元/晚

### 4. 西班牙 - 攀岩大國

#### 4.1 西班牙攀岩概述
- 歐洲路線數量最多的國家之一
- 多樣的岩質與攀登風格
- 氣候宜人，全年可攀

#### 4.2 Albarracin - 砂岩抱石

**目的地介紹**
- 位置：阿拉貢自治區，馬德里東方約 280 公里
- 歐洲頂級砂岩抱石區
- 中世紀古城風情

**岩質與特色**
- **岩質**：紅砂岩
- **路線數**：1,000+ 條抱石
- **難度**：3 至 8B+
- **季節**：3-5月、9-11月

#### 4.3 El Chorro - 運動攀登

**目的地介紹**
- 位置：安達盧西亞，馬拉加北方
- 壯觀的峽谷攀登
- 著名的 Caminito del Rey 步道

**岩質與特色**
- **岩質**：石灰岩
- **路線數**：1,000+ 條
- **難度**：4 至 9a
- **季節**：10月-5月

#### 4.4 Siurana - 石灰岩經典

**目的地介紹**
- 位置：加泰隆尼亞，巴塞隆納西南
- 西班牙最知名的運動攀登區之一
- 美麗的山村環境

**岩質與特色**
- **岩質**：石灰岩
- **路線數**：1,500+ 條
- **難度**：5 至 9b
- **季節**：10月-5月

### 5. 土耳其 Geyikbayiri

#### 5.1 目的地介紹
- 位置：安塔利亞省，地中海沿岸
- 新興的歐洲攀岩目的地
- 高性價比的選擇
- 攀岩季節：10月至5月

#### 5.2 岩質與特色
- **岩質**：石灰岩（Limestone）
- **特色**：
  - 大量的懸岩與 Tufa
  - 壯觀的石灰岩峽谷
  - 費用較西歐低廉
  - 持續開發中
- **風格**：運動攀登

#### 5.3 路線數量與難度
| 項目 | 數據 |
|------|------|
| 路線總數 | 900+ 條 |
| 難度範圍 | 4 至 8c（法國級數） |
| 主要岩區 | Sarkit, Trebenna, Citdibi |
| 特色 | 大量中等難度懸岩路線 |

#### 5.4 交通與住宿
- 飛安塔利亞機場
- 機場至岩區約 30 分鐘
- 攀岩者營地與民宿
- 費用：20-80 歐元/晚

### 6. 其他歐洲攀岩目的地

#### 6.1 義大利
- **Arco**：加爾達湖畔，運動攀登經典
- **Finale Ligure**：地中海岸石灰岩
- **白雲石山脈（Dolomites）**：高山大岩壁

#### 6.2 瑞士
- **Magic Wood**：頂級花崗岩抱石
- **Ticino**：花崗岩抱石天堂

#### 6.3 英國
- **Peak District**：砂岩抱石與傳統攀登
- **Lake District**：傳統攀登
- **North Wales**：多樣攀登風格

#### 6.4 德國
- **Frankenjura**：運動攀登，5,000+ 路線
- **Saxon Switzerland**：獨特的砂岩塔攀登

#### 6.5 挪威
- **Flatanger**：世界最難洞穴路線
- **Lofoten**：壯觀的高山花崗岩

### 7. 歐洲攀岩實用資訊

#### 7.1 交通規劃
- 廉價航空：Ryanair、EasyJet、Vueling
- 租車：多數岩場需要自駕
- 火車：適合部分地區

#### 7.2 住宿選擇
- 攀岩者營地
- 青年旅館
- Airbnb
- 露營（部分地區）

#### 7.3 攀岩資源
- **27 Crags**：歐洲岩場資料庫
- **TheCrag**：全球路線資料
- **Rockfax**：英國攀岩指南
- 各區域專用 App 與指南書

#### 7.4 申根簽證
- 台灣護照免簽 90 天
- 注意停留天數計算
- 部分國家非申根區

### 8. 最佳季節建議
| 地點 | 最佳季節 | 氣溫 | 備註 |
|------|----------|------|------|
| 楓丹白露 | 3-5月、9-11月 | 10-20°C | 避開雨季與夏季 |
| Kalymnos | 3-5月、9-11月 | 15-25°C | 避開酷暑 |
| 西班牙 | 10月-5月 | 10-25°C | 依地區而異 |
| 土耳其 | 10月-5月 | 10-25°C | 冬季溫和 |
