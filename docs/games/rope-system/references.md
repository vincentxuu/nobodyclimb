## 使用原則

### 引用優先順序

1. **專業認證機構**（AMGA、UIAA）- 作為標準程序的依據
2. **裝備製造商技術文件**（Petzl、BD）- 作為器材操作的依據
3. **經典教科書**（Freedom of the Hills）- 作為理論知識的依據
4. **事故報告**（AAC Accidents）- 作為安全警示的依據

### 內容衝突處理

當不同來源有衝突時：
1. 優先採用專業認證機構的標準
2. 參考最新版本的資料
3. 選擇更保守（安全）的做法
4. 在題目解釋中說明不同觀點（如適用）

### 定期更新

- 每年檢查連結是否有效
- 關注各機構的更新公告
- 納入新的安全研究發現
