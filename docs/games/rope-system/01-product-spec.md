# 產品規格書

## 產品概述

**產品名稱**：攀岩系統練習遊戲（Rope System Training Game）

**目標用戶**：
- 攀岩初學者（學習正確操作）
- 進階攀岩者（複習與精進）
- 岩館教練（教學輔助工具）
- 岩館管理者（認證考試系統）

**核心價值**：
- 在安全的環境中學習危險操作的正確步驟
- 透過遊戲化機制提高學習動機
- 標準化岩館教學與認證流程

---

## 功能需求

### 1. 學習模式

#### 1.1 類別選擇
- 顯示所有題目類別（運動攀登 / 傳統攀登）
- 顯示每個類別的題數、難度、完成進度
- 可選擇單一類別或混合練習

#### 1.2 答題流程
- 顯示情境描述與問題
- 提供 3-4 個選項（選擇題）或可拖曳項目（排序題）
- 選擇後立即顯示對錯
- 答錯時播放掉落動畫與音效
- 顯示詳細解釋與正確答案
- 可查看相關圖示或動畫

#### 1.3 進度追蹤
- 顯示當前進度（第 N / M 題）
- 顯示剩餘生命值
- 顯示累計分數
- 答完後顯示統計（正確率、用時）

### 2. 考試模式

#### 2.1 考試設定
- 選擇考試類別或指定考卷
- 顯示考試規則（題數、時限、及格分數）
- 開始前確認準備就緒

#### 2.2 考試流程
- 不顯示答案對錯（答完才結算）
- 可選擇跳過稍後作答
- 顯示剩餘時間（如有時限）
- 禁止返回修改已作答題目（可設定）

#### 2.3 成績結算
- 顯示總分與及格狀態
- 顯示各類別正確率
- 顯示答錯題目與正確答案
- 儲存成績紀錄

### 3. 認證系統

#### 3.1 認證等級

| 等級 | 名稱 | 需通過項目 | 對應能力 |
|------|------|-----------|---------|
| Level 1 | 確保員 | 基礎確保 ≥80% | 可進行頂繩確保 |
| Level 2 | 先鋒確保員 | Level 1 + 先鋒攀登 ≥80% | 可進行先鋒確保 |
| Level 3 | 戶外攀岩者 | Level 2 + 頂繩架設 + 垂降 ≥80% | 可進行戶外單繩距 |
| Level 4 | 多繩距跟攀者 | Level 3 + 固定點 + 多繩距基礎 ≥75% | 可進行多繩距跟攀 |
| Level 5 | 多繩距先鋒 | Level 4 + 完整多繩距 + 救援 ≥75% | 可獨立進行多繩距 |

> **注意**：Level 4-5 及格分數較低（75%）是因為傳統攀登題目難度較高，且情境更為複雜。岩館可自行調整及格標準。

#### 3.2 認證發放
- 通過考試後自動判定等級
- 產生可下載的認證圖片
- 可分享至社群媒體
- 岩館可查詢會員認證狀態

### 4. 岩館後台

#### 4.1 題庫管理
- 查看所有題目列表
- 新增題目（選擇題 / 排序題 / 情境題）
- 編輯現有題目
- 上傳題目圖片
- 設定題目難度與類別
- 停用 / 啟用題目

#### 4.2 考卷管理
- 建立自訂考卷
- 從題庫選擇題目
- 設定題目順序（固定 / 隨機）
- 設定時間限制
- 設定及格分數
- 發布 / 下架考卷

#### 4.3 學員管理
- 查看學員考試紀錄
- 查看學員認證狀態
- 手動發放 / 撤銷認證
- 匯出成績報表

#### 4.4 數據分析
- 各題目答對率統計
- 各類別平均分數
- 學員學習曲線
- 常見錯誤分析

## 遊戲機制

### 生命值系統
- 初始生命值：3 顆心
- 答錯扣 1 顆心
- 生命值歸零 → 遊戲結束
- 學習模式可選擇無限生命

### 計分系統
| 情況 | 分數 |
|------|------|
| 答對（簡單題 ⭐） | +100 |
| 答對（中等題 ⭐⭐） | +125 |
| 答對（困難題 ⭐⭐⭐） | +150 |
| 連續答對加成 | ×1.5 |
| 快速作答加成 | +20 |
| 答錯 | -50 |

### 掉落動畫觸發
- 答錯時觸發
- 角色從當前位置往下掉落
- 階梯式下墜（類似小朋友下樓梯）
- 搭配音效與震動回饋

## 非功能需求

### 效能
- 頁面載入時間 < 2 秒
- 動畫流暢度 60fps
- 支援離線快取題目

### 相容性
- 支援桌面瀏覽器（Chrome、Firefox、Safari、Edge）
- 支援行動裝置瀏覽器
- 響應式設計適配不同螢幕

### 無障礙
- 支援鍵盤操作
- 色彩對比符合 WCAG AA
- 提供音效開關選項
