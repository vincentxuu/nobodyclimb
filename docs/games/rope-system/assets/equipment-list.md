# 裝備圖示清單

> 遊戲中需要使用的攀岩裝備圖示，用於題目視覺輔助與情境說明。

## 圖示規格

| 項目 | 規格 |
|------|------|
| 格式 | SVG（向量）或 PNG（2x 解析度） |
| 尺寸 | 64×64px（標準）、128×128px（大圖） |
| 風格 | 線條圖示，配合網站主色系 |
| 顏色 | 主色 #1B1A1A、強調色 #FFE70C |

## 圖示來源建議

### 免費資源
- [Heroicons](https://heroicons.com/) - 基礎 UI 圖示
- [Lucide](https://lucide.dev/) - 開源圖示庫
- [SVG Repo](https://www.svgrepo.com/) - 搜尋攀岩相關

### 客製化
- 複雜裝備圖示建議客製繪製
- 保持風格一致性
- 可考慮 AI 生成後人工修正

### 參考風格
- 簡約線條風格
- 2D 扁平化
- 配色與網站主題一致
