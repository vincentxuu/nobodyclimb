## Q1: 固定點的基本要求

**難度**：⭐⭐

**情境**

你準備在戶外岩場架設頂繩。

**問題**

一個安全的頂繩固定點應該符合哪個原則？

**選項**

- A. SERENE（Solid, Equalized, Redundant, Efficient, No Extension）
- B. 越多裝備越好
- C. 只要有一個 bolt 就夠了
- D. 用最新最貴的裝備

**正確答案**：A

**解釋**

**SERENE 原則**是評估固定點的標準：

| 字母 | 意義 | 說明 |
|------|------|------|
| **S** | Solid（堅固） | 每個固定點都能承受預期負載 |
| **E** | Equalized（均力） | 負載平均分配到各點 |
| **R** | Redundant（冗餘） | 至少兩個獨立固定點 |
| **E** | Efficient（高效） | 簡單、快速架設 |
| **NE** | No Extension（無延展） | 一點失效不會造成衝擊 |

**另一個常用口訣**：EARNEST
- **E**qualized, **A**ngles, **R**edundant, **N**o Extension, **E**fficient, **S**olid, **T**imely

**參考來源**：AMGA, Climbing Anchors (John Long), Freedom of the Hills

**標籤**：`SERENE`, `固定點`, `安全原則`

## Q2: Bolt 狀態檢查

**難度**：⭐⭐

**情境**

你到達岩壁頂端，看到兩個 bolt 作為固定點。

**問題**

以下哪項是 bolt 狀態不良的警訊？

**選項**

- A. bolt 頭有些許灰塵
- B. bolt 明顯鏽蝕、鬆動或旋轉
- C. bolt 看起來不是最新款式
- D. bolt 表面有攀岩粉痕跡

**正確答案**：B

**解釋**

**bolt 狀態檢查項目**：

✅ **正常情況**：
- 輕微灰塵或粉塵
- 舊款式但狀態良好
- 表面有使用痕跡

❌ **危險警訊**：
- **明顯鏽蝕**：尤其是海邊或潮濕環境
- **鬆動**：用手搖會動
- **旋轉**：bolt 可以轉動
- **膨脹裂縫**：岩石有裂痕從 bolt 延伸
- **掛片變形**：金屬彎曲或斷裂

**遇到可疑 bolt**：
- 不要使用該固定點
- 回報給當地攀岩社群或岩場管理者

**參考來源**：AMGA, AAC Accidents, Petzl

**標籤**：`bolt檢查`, `固定點`, `安全評估`

## Q3: 天然固定點評估

**難度**：⭐⭐

**情境**

岩壁頂端沒有 bolt，但有一棵樹可以使用。

**問題**

使用樹木作為固定點時，最低要求是？

**選項**

- A. 任何大小的樹都可以
- B. 樹幹直徑至少 15-20 公分，根系穩固的活樹
- C. 枯樹比較硬，更適合
- D. 樹越高越好

**正確答案**：B

**解釋**

**樹木作為固定點的要求**：

| 項目 | 要求 |
|------|------|
| 直徑 | 至少 15-20 公分（約手臂粗） |
| 狀態 | 必須是**活樹**，有綠葉 |
| 根系 | 穩固紮根，不會搖晃 |
| 位置 | 靠近岩壁邊緣，減少繩索摩擦 |

**不可使用**：
- ❌ 枯樹（內部可能腐爛）
- ❌ 太細的樹（可能彎曲或斷裂）
- ❌ 根系不穩的樹
- ❌ 離岩邊太遠的樹

**架設方式**：
- 使用繩環繞過樹幹底部
- 避免繩環滑到高處
- 保護樹皮，使用護樹墊

**參考來源**：AMGA, Climbing Anchors, Freedom of the Hills

**標籤**：`天然固定點`, `樹木`, `固定點評估`

## Q4: 雙 bolt 固定點的角度

**難度**：⭐⭐

**情境**

你要在兩個 bolt 之間架設均力固定點。

**問題**

兩個 bolt 之間的繩環角度應該控制在多少以內？

**選項**

- A. 不超過 60 度
- B. 不超過 90 度
- C. 不超過 120 度
- D. 角度不重要

**正確答案**：A

**解釋**

**角度與負載分配**：

| 角度 | 每點負載 | 風險 |
|------|----------|------|
| 0° | 50% | 最理想 |
| 60° | 58% | 可接受 |
| 90° | 71% | 臨界值 |
| 120° | 100% | **危險！** |

**為什麼 60 度以內最好**：
- 負載分配接近均等
- 即使一點失效，衝擊較小
- 留有安全餘裕

**超過 90 度的問題**：
- 負載反而增加
- 可能超過裝備強度

**實際操作**：
- 繩環盡量長一些
- 主點位置靠下
- 視覺上，角度應該看起來「窄」而非「寬」

**參考來源**：AMGA, Climbing Anchors, Petzl

**標籤**：`角度`, `均力`, `固定點架設`

## Q5: 快扣架設頂繩

**難度**：⭐⭐

**情境**

你在運動攀登岩場，想用快扣快速架設頂繩。

**問題**

使用快扣架設頂繩時，以下哪種方式最正確？

**選項**

- A. 一個快扣掛一個 bolt 即可
- B. 兩個快扣分別掛兩個 bolt，繩子穿過兩個快扣
- C. 把快扣串在一起掛在一個 bolt 上
- D. 快扣不適合架設頂繩

**正確答案**：B

**解釋**

**快扣架設頂繩的正確方式**：

```
Bolt 1        Bolt 2
  │             │
  ▼             ▼
┌───┐         ┌───┐
│快扣│         │快扣│
└─┬─┘         └─┬─┘
  │             │
  └──── 繩子 ────┘
```

**重點**：
- 兩個獨立的 bolt = 冗餘
- 繩子穿過兩個快扣 = 均力
- 快扣繩門朝外，避免繩索開啟

**注意事項**：
- 只適合短時間使用
- 繩子會磨損快扣的繩端鉤環
- 長時間使用應改用專用器材（繩環 + 鉤環）

**不建議的原因**：
- 快扣不是設計給固定點使用
- 沒有防止繩門打開的機制

**參考來源**：VDiff Climbing, REI Expert Advice

**標籤**：`快扣`, `頂繩架設`, `臨時固定點`

## Q6: 繩環架設方式

**難度**：⭐⭐

**情境**

你決定使用 120cm 繩環（Sling）架設頂繩固定點。

**問題**

繩環應該如何連接到兩個 bolt？

**選項**

- A. 一條繩環穿過兩個 bolt 的掛片
- B. 兩條獨立繩環分別掛兩個 bolt，在底部用主鎖連接
- C. 繩環打結後掛在一個 bolt 上
- D. 直接把繩環綁在岩石上

**正確答案**：B

**解釋**

**正確架設方式**：

```
Bolt 1          Bolt 2
  │               │
  ▼               ▼
繩環 1          繩環 2
  │               │
  └───┬───────┬───┘
      │       │
      ▼       ▼
    ┌───────────┐
    │   主鎖    │
    └───────────┘
         │
        繩子
```

**關鍵原則**：
1. **兩條獨立繩環**（Redundancy）
2. **各自掛入 bolt**
3. **底部用有鎖鉤環連接**
4. **繩子只穿過鉤環**

**為什麼不穿過繩環**：
- 繩子會嚴重磨損繩環
- Dyneema 繩環尤其不耐磨
- 可能在幾次降低後就損壞

**參考來源**：AMGA, Petzl, VDiff Climbing

**標籤**：`繩環`, `架設方式`, `固定點`

## Q7: 主點的設置

**難度**：⭐⭐

**情境**

你在架設頂繩固定點，需要設置主點（Master Point）。

**問題**

主點應該使用什麼裝備？

**選項**

- A. 一個普通（無鎖）鉤環
- B. 一個有鎖鉤環
- C. 兩個反向對扣的鉤環（有鎖或無鎖皆可）
- D. 快扣

**正確答案**：C

**解釋**

**主點設置的最佳實踐**：

**推薦方式：雙鉤環反向對扣**
```
    繩環
     │
 ┌───┴───┐
 │ 鉤環1 │◄── 開口朝左
 ├───────┤
 │ 鉤環2 │◄── 開口朝右
 └───┬───┘
     │
    繩子
```

**為什麼要反向對扣**：
- 即使一個鉤環意外打開，另一個仍有保護
- 繩索不會同時從兩個開口滑出
- 符合冗餘原則

**可接受的替代方案**：
- 兩個有鎖鉤環反向對扣（最安全）
- 一個有鎖鉤環（最低標準）

**不可接受**：
- 單一無鎖鉤環
- 快扣（繩門太容易打開）

**參考來源**：AMGA, Climbing Anchors, Freedom of the Hills

**標籤**：`主點`, `鉤環`, `反向對扣`

## Q8: 繩子的磨損問題

**難度**：⭐⭐

**情境**

你的頂繩固定點架設在岩壁邊緣。

**問題**

繩子最可能在哪裡受到磨損？

**選項**

- A. 在確保者手中
- B. 在攀登者吊帶處
- C. 在岩壁邊緣接觸處
- D. 在繩袋中

**正確答案**：C

**解釋**

**繩子磨損的高風險位置**：

1. **岩壁邊緣**（最危險）
   - 繩子在有重量時會摩擦岩邊
   - 尤其是降低和墜落時
   - 可能在短時間內嚴重損傷

2. **解決方案**：
   - 延長固定點，讓繩子垂直落下
   - 使用繩索保護器（Rope Protector）
   - 選擇繩子不會接觸岩邊的架設位置

**檢查時機**：
- 架設完成後檢查繩索路徑
- 每次使用前目視檢查
- 感覺繩子有卡頓時立即檢查

**參考來源**：AMGA, Petzl, REI Expert Advice

**標籤**：`繩索磨損`, `岩壁邊緣`, `安全檢查`

## Q9: 延長固定點的原因

**難度**：⭐⭐

**情境**

固定點位於岩壁頂端往內約 1 公尺的位置。

**問題**

為什麼需要延長固定點？

**選項**

- A. 讓固定點看起來更專業
- B. 讓繩子能夠垂直落下，減少岩邊摩擦
- C. 增加繩子的長度
- D. 讓攀登者爬得更高

**正確答案**：B

**解釋**

**延長固定點的目的**：

```
❌ 未延長              ✅ 延長後

岩邊──╮               岩邊──────╮
      │                         │
      │←繩子摩擦                │
      │                         │
      ▼                         ▼
   固定點              固定點──延長──主點
                                      │
                                     繩子（垂直）
```

**好處**：
1. 繩子垂直落下，不摩擦岩邊
2. 減少繩索阻力
3. 確保者操作更順暢
4. 攀登者墜落時繩子不會卡住

**延長方式**：
- 使用長繩環（120-240cm）
- 使用 Cordelette
- 注意延長後的角度仍要小於 60 度

**參考來源**：VDiff Climbing, Climbing Anchors

**標籤**：`延長固定點`, `繩索路徑`, `架設技巧`

## Q10: 獨自架設頂繩的安全

**難度**：⭐⭐⭐

**情境**

你需要獨自走到岩壁頂端架設頂繩。

**問題**

在岩壁頂端作業時，最重要的安全措施是？

**選項**

- A. 動作要快，減少在頂端的時間
- B. 全程自我確保，連接到可靠的固定點
- C. 大聲呼喊讓底下的人知道
- D. 穿著鮮豔的衣服

**正確答案**：B

**解釋**

**頂端作業的安全原則**：

1. **全程自我確保**
   - 到達頂端立即連接到固定點
   - 使用 PAS、Daisy Chain 或繩環
   - 確認承重後再開始作業

2. **安全順序**：
   - 接近岩邊前先找好固定點
   - 連接自我確保
   - 測試確認牢固
   - 再開始架設工作

3. **絕對禁止**：
   - ❌ 沒有自我確保就靠近岩邊
   - ❌ 依賴摩擦力或平衡
   - ❌ 趕時間而省略安全步驟

**參考來源**：AMGA, Freedom of the Hills

**標籤**：`自我確保`, `頂端作業`, `安全程序`

## Q11: 頂繩固定點的冗餘

**難度**：⭐⭐

**情境**

你在評估一個現有的頂繩固定點。

**問題**

「冗餘」（Redundancy）在固定點中的意義是？

**選項**

- A. 使用越多裝備越好
- B. 至少有兩個獨立的固定點，一個失效時另一個仍能承受
- C. 每個裝備都要是新的
- D. 固定點要比需要的更高

**正確答案**：B

**解釋**

**冗餘原則的意義**：

```
單點（無冗餘）          雙點（有冗餘）

    │                    ┌──────┐
    │                    │      │
    ▼                    ▼      ▼
  固定點              固定點1  固定點2
    │                    │      │
    │                    └──┬───┘
    ▼                       │
   繩子                    繩子

❌ 單點失效 = 完全失效   ✅ 一點失效仍有保護
```

**實現冗餘的方式**：
- 使用兩個 bolt
- 使用兩條獨立繩環
- 使用兩個鉤環（反向對扣）

**冗餘 ≠ 越多越好**：
- 過度複雜反而增加出錯機會
- 重點是「獨立」而非「數量」
- 兩個合格的固定點勝過五個可疑的

**參考來源**：AMGA, Climbing Anchors

**標籤**：`冗餘`, `Redundancy`, `固定點原則`

## Q12: 不延展原則

**難度**：⭐⭐⭐

**情境**

你正在學習固定點架設的進階概念。

**問題**

「不延展」（No Extension）原則是什麼意思？

**選項**

- A. 繩環不能有彈性
- B. 一個固定點失效時，系統不會突然延長造成衝擊
- C. 固定點不能延長
- D. 繩子不能有延展性

**正確答案**：B

**解釋**

**不延展原則**：

```
❌ 有延展風險              ✅ 無延展

   ╱╲                       │
  ╱  ╲                      │
 ╱    ╲                    ╱│╲
點1    點2                ╱ │ ╲
 ╲    ╱                  點1 │ 點2
  ╲  ╱                      │
   ╲╱                       │
   主點                     主點

如果點1失效：            如果點1失效：
主點會突然下墜            主點位置不變
造成衝擊負載              無額外衝擊
```

**如何實現**：
- 在繩環上打限制結（Limiting Knot）
- 使用 Cordelette 配合 Overhand Knot
- 使用預先均力化的系統

**為什麼重要**：
- 突然的延展會產生衝擊負載
- 可能導致剩餘固定點也失效
- 特別是在動態負載（墜落）時更危險

**參考來源**：Climbing Anchors, AMGA

**標籤**：`不延展`, `No Extension`, `固定點原則`

## Q13: 確保者的位置

**難度**：⭐⭐

**情境**

頂繩架設完成，你準備在底下確保。

**問題**

頂繩確保時，確保者應該站在哪裡？

**選項**

- A. 離牆壁越遠越好
- B. 在攀登者正下方或稍微後退的位置
- C. 在攀登者旁邊
- D. 在繩子會經過的位置

**正確答案**：B

**解釋**

**頂繩確保的站位原則**：

**最佳位置**：
- 攀登者**正下方**或**稍微後退**
- 可以清楚看到攀登者
- 繩子路徑直線且沒有障礙

**注意事項**：
- 不要站在繩子會擺盪的路徑上
- 注意頭頂是否有落石風險
- 確保可以聽到攀登者的聲音

**與先鋒確保的差異**：
- 頂繩不需要站太靠近牆壁
- 因為沒有地面墜落的風險
- 但也不要站太遠造成繩索角度

**參考來源**：REI Expert Advice, VDiff Climbing

**標籤**：`確保站位`, `頂繩確保`, `安全位置`

## Q14: 頂繩繩尾處理

**難度**：⭐

**情境**

頂繩架設完成，繩子一端穿過固定點垂到地面。

**問題**

繩子的尾端應該如何處理？

**選項**

- A. 不用特別處理
- B. 綁上一個結，防止繩尾意外滑過確保器
- C. 讓它自然垂地
- D. 把它收進繩袋裡

**正確答案**：B

**解釋**

**繩尾處理的重要性**：

**打結的原因**：
- 防止繩尾滑過確保器
- 避免攀登者降到一半繩子不夠
- 這是最後一道安全防線

**打結方式**：
- 在繩尾打一個大結（如雙漁人結、八字結）
- 結要夠大，不會穿過確保器
- 或將繩尾綁在確保者吊帶上

**什麼時候特別重要**：
- 路線長度接近繩子一半時
- 不確定繩子長度是否足夠時
- **永遠都應該打結**

**參考來源**：AMGA, AAC Accidents, Petzl

**標籤**：`繩尾處理`, `安全措施`, `確保技巧`

## Q15: 收回固定點的順序

**難度**：⭐⭐

**情境**

攀登結束，你需要收回架設在頂端的固定點。

**問題**

收回固定點時，正確的順序是？

**選項**

- A. 先拆繩子，再拆固定點
- B. 先拆固定點，再拆繩子
- C. 先確保自己安全連接，再依序拆除
- D. 全部一起拆最快

**正確答案**：C

**解釋**

**收回固定點的安全順序**：

1. **確保自我安全**
   - 到達頂端，先連接自我確保
   - 確認自我確保承重

2. **處理繩子**
   - 確認沒有人在繩子上
   - 解開繩子或準備抽繩

3. **拆除固定點**
   - 依序拆除（先拆繩環，再拆鉤環）
   - 裝備收好掛在吊帶上

4. **離開**
   - 確認所有裝備都已收回
   - 最後解除自我確保
   - 安全離開岩壁頂端

**關鍵原則**：
- 永遠保持自我確保直到最後
- 不要在沒有備份的情況下拆除最後一個固定點

**參考來源**：AMGA, VDiff Climbing

**標籤**：`收回固定點`, `安全程序`, `拆除順序`
