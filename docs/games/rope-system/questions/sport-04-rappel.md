## Q1: 垂降的基本裝備

**難度**：⭐⭐

**情境**

你準備進行第一次戶外垂降。

**問題**

垂降最基本需要哪些裝備？

**選項**

- A. 只要有繩子就可以
- B. 確保器（如 ATC）、繩子、吊帶、安全帽
- C. 只需要手套和繩子
- D. 確保器和吊帶就夠了

**正確答案**：B

**解釋**

**垂降基本裝備清單**：

| 裝備 | 功能 |
|------|------|
| **確保器** | 產生摩擦力控制下降速度（ATC、GriGri等） |
| **繩子** | 下降的媒介，需夠長到達底部 |
| **吊帶** | 連接身體與系統 |
| **安全帽** | 防止落石與碰撞 |
| **有鎖鉤環** | 連接確保器與吊帶 |

**建議額外裝備**：
- 普魯士繩（Autoblock 備份）
- 手套（長距離垂降）
- 繩套（整理繩子）

**參考來源**：AMGA, Petzl, Freedom of the Hills

**標籤**：`裝備`, `垂降基礎`, `安全裝備`

## Q2: 垂降前的檢查

**難度**：⭐⭐

**情境**

你已經架設好垂降系統，準備開始下降。

**問題**

開始垂降前，最重要的檢查項目是？

**選項**

- A. 確認天氣狀況
- B. 確認繩子有到達底部或打了繩尾結
- C. 確認有人在拍照
- D. 確認手機有電

**正確答案**：B

**解釋**

**垂降前必要檢查**：

**最重要**：
✅ **繩子是否到底**
- 目視確認繩子到達地面或下一個固定點
- 如果看不到，**必須打繩尾結**
- 繩尾結防止滑出確保器

**完整檢查清單**：
1. 繩子長度足夠（或有繩尾結）
2. 繩子是否卡住或纏繞
3. 確保器正確安裝
4. 鉤環已鎖緊
5. 備份裝置已設置（如 Autoblock）

**致命錯誤案例**：
- 繩子不夠長又沒打繩尾結
- 滑到繩尾直接墜落
- 這是常見的垂降致死原因之一

**參考來源**：AAC Accidents, AMGA, Petzl

**標籤**：`安全檢查`, `繩尾結`, `垂降準備`

## Q3: 繩子穿過確保器

**難度**：⭐⭐

**情境**

你正在設置 ATC 確保器進行垂降。

**問題**

雙繩垂降時，繩子應該如何穿過 ATC？

**選項**

- A. 只穿一股繩子
- B. 兩股繩子都要穿過確保器
- C. 一股穿過確保器，一股用手抓住
- D. 繩子不需要穿過確保器

**正確答案**：B

**解釋**

**雙繩垂降的正確設置**：

```
    固定點
      │
    ╱   ╲
   繩1   繩2
    │     │
    └──┬──┘
       │
   ┌───────┐
   │  ATC  │
   └───┬───┘
       │
      吊帶
```

**為什麼兩股都要穿**：
- 均勻分配摩擦力
- 確保受力平衡
- 下降更平穩可控

**常見錯誤**：
- ❌ 只穿一股 → 摩擦力不夠，速度失控
- ❌ 兩股交叉穿 → 可能卡繩
- ❌ 穿反方向 → 制動效果降低

**確認方式**：
- 兩股繩子在制動端應該平行
- 輕拉測試，確認滑順

**參考來源**：Petzl, AMGA, Black Diamond

**標籤**：`ATC`, `雙繩垂降`, `確保器設置`

## Q4: Autoblock 備份

**難度**：⭐⭐

**情境**

你正在學習更安全的垂降方式。

**問題**

Autoblock（自動鎖繩結）應該安裝在哪裡？

**選項**

- A. 確保器上方
- B. 確保器下方，連接到腿環
- C. 連接到固定點
- D. 綁在手上

**正確答案**：B

**解釋**

**Autoblock 的正確位置**：

```
    固定點
      │
     繩子
      │
  ┌───────┐
  │  ATC  │ ← 確保器
  └───┬───┘
      │
      │  ← 繩子
      │
  【Autoblock】← 在這裡！
      │
    腿環  ← 連接到腿環
```

**為什麼在下方**：
- 鬆開雙手時，Autoblock 會抓住繩子
- 自動停止下降
- 不會干擾確保器操作

**為什麼連接腿環**：
- 距離適當，不會卡到確保器
- 容易用手解開
- 不影響正常操作

**常見 Autoblock 繩結**：
- French Prusik（最常用）
- Prusik（傳統）
- Klemheist

**參考來源**：AMGA, Petzl, VDiff Climbing

**標籤**：`Autoblock`, `備份系統`, `垂降安全`

## Q5: 制動手的位置

**難度**：⭐⭐

**情境**

你正在垂降途中。

**問題**

垂降時，制動手應該放在哪裡？

**選項**

- A. 確保器上方抓住繩子
- B. 確保器下方抓住繩子
- C. 放在吊帶上
- D. 不需要用手抓繩子

**正確答案**：B

**解釋**

**制動手的正確位置**：

```
      │
  ┌───────┐
  │  ATC  │
  └───┬───┘
      │
      │← 制動手在這裡！
      │
      ▼
```

**原理**：
- 下方的繩子 = 制動端
- 握緊下方繩子可增加摩擦
- 鬆開會加速，握緊會減速或停止

**錯誤示範**：
- ❌ 手在上方 → 無法有效制動
- ❌ 完全不抓繩子 → 失去控制
- ❌ 抓太靠近確保器 → 可能卡到手

**操作技巧**：
- 制動手永遠不離開繩子
- 另一隻手可扶岩壁保持平衡
- 平穩放繩，避免忽快忽慢

**參考來源**：AMGA, Petzl, REI Expert Advice

**標籤**：`制動手`, `垂降技術`, `繩索控制`

## Q6: 下降速度控制

**難度**：⭐⭐

**情境**

你正在進行戶外垂降，發現下降速度很快。

**問題**

如何有效減慢垂降速度？

**選項**

- A. 把繩子繞過大腿增加摩擦
- B. 用力抓緊上方的繩子
- C. 大聲呼救
- D. 腳踢岩壁

**正確答案**：A

**解釋**

**減速技巧**：

**方法一：繩子繞腿**
```
     ATC
      │
      │
      ╲
       ╲──大腿
      ╱
     ╱
    │
```
- 增加摩擦面積
- 有效降低速度
- 長距離垂降常用

**方法二：加裝摩擦**
- 多繞一圈在確保器上
- 使用有輔助制動功能的確保器

**其他技巧**：
- 手握緊制動端（下方）
- 身體稍微後仰增加角度
- 保持平穩節奏

**避免**：
- ❌ 抓上方繩子（無效且危險）
- ❌ 腳踢岩壁（會彈開失控）
- ❌ 戴手套抓繩子（可能更滑）

**參考來源**：AMGA, Freedom of the Hills

**標籤**：`速度控制`, `摩擦力`, `垂降技巧`

## Q7: 通過繩結

**難度**：⭐⭐⭐

**情境**

垂降途中，你發現繩子中間有一個繩結（可能是接繩處）。

**問題**

遇到繩結時，正確的處理方式是？

**選項**

- A. 直接讓繩結通過確保器
- B. 停止，建立備份，重新架設確保器在繩結下方
- C. 用力拉扯讓繩結變小
- D. 把繩結解開

**正確答案**：B

**解釋**

**通過繩結的標準程序**：

1. **停在繩結上方**
   - 不要讓繩結進入確保器
   - 繩結會卡住確保器

2. **建立備份**
   - 用 Prusik 或繩環連接到上方繩子
   - 確認備份承重

3. **重新架設**
   - 在繩結下方架設確保器
   - 將繩子穿過

4. **轉移重量**
   - 確認新系統正確
   - 解除備份
   - 繼續下降

**為什麼不能直接通過**：
- 繩結會卡在確保器中
- 可能無法繼續下降
- 處於非常尷尬的位置

**參考來源**：AMGA, Freedom of the Hills, VDiff Climbing

**標籤**：`通過繩結`, `進階技術`, `垂降救援`

## Q8: 抽繩的技巧

**難度**：⭐⭐

**情境**

你垂降到底後，需要抽回繩子。

**問題**

雙繩垂降後，抽繩時應該注意什麼？

**選項**

- A. 隨便抽哪一端都可以
- B. 抽正確的那一端，並觀察繩子是否順暢下來
- C. 用力快速抽
- D. 兩端一起抽

**正確答案**：B

**解釋**

**抽繩的正確方式**：

**雙繩垂降的繩子配置**：
```
    固定點
    ╱    ╲
   │      │
  抽這端   卡住！
   │      │
   └──────┘
```

**正確步驟**：
1. **確認抽哪端**
   - 通常是沒有打結的那端
   - 或是穿過固定點掛片的那端

2. **慢慢抽**
   - 先抽幾公尺測試
   - 確認繩子有在動

3. **觀察繩子**
   - 繩子應該順暢落下
   - 如果卡住要立即停止

4. **站在安全位置**
   - 不要站在繩子正下方
   - 避免被落下的繩子打到

**抽繩卡住的原因**：
- 繩子纏繞或打結
- 卡在岩縫中
- 抽錯邊

**參考來源**：AMGA, VDiff Climbing

**標籤**：`抽繩`, `雙繩垂降`, `繩索管理`

## Q9: 垂降站位

**難度**：⭐⭐

**情境**

你正在岩壁上進行垂降。

**問題**

垂降時，身體相對於岩壁的正確姿勢是？

**選項**

- A. 身體貼緊岩壁
- B. 雙腿與岩壁約成直角，微微後仰
- C. 完全倒掛
- D. 側身下降

**正確答案**：B

**解釋**

**正確的垂降姿勢**：

```
    岩壁
      │
      │   ╱ 身體
      │  ╱
      │ ╱ ← 約 90 度
      │╱
    ──┴── 雙腳
```

**關鍵要點**：
- **雙腿打直**：與岩壁約成直角
- **雙腳與肩同寬**：穩定支撐
- **身體微後仰**：重心在腳上
- **膝蓋微彎**：吸收衝擊

**為什麼這個姿勢最好**：
- 腳可以穩定踩在岩壁上
- 不會旋轉或翻轉
- 可以清楚看到下方
- 容易控制速度

**常見錯誤**：
- ❌ 身體太直立 → 腳會滑開
- ❌ 身體太後仰 → 翻過去
- ❌ 雙腿彎曲 → 失去控制

**參考來源**：AMGA, Petzl, REI Expert Advice

**標籤**：`垂降姿勢`, `身體位置`, `垂降技術`

## Q10: 垂降時的溝通

**難度**：⭐⭐

**情境**

你的夥伴在固定點等待你垂降完成。

**問題**

垂降到達底部後，應該如何通知上方的夥伴？

**選項**

- A. 不用通知，他會自己知道
- B. 喊「Off Rappel」或「繩子解除」
- C. 拉動繩子
- D. 傳訊息

**正確答案**：B

**解釋**

**垂降溝通口令**：

| 口令 | 時機 | 意義 |
|------|------|------|
| **On Rappel / 準備垂降** | 開始前 | 我要開始垂降了 |
| **Rappelling / 垂降中** | 開始時 | 我開始下降了 |
| **Off Rappel / 繩子解除** | 到達後 | 我到了，繩子可以用了 |

**為什麼要喊「Off Rappel」**：
- 告訴上方的人繩子已經空了
- 上方的人可以開始垂降
- 確保不會同時使用繩子

**如果聽不到怎麼辦**：
- 事先約定拉繩子信號
- 例：拉三下 = 到了
- 或使用對講機

**危險情況**：
- 沒溝通，兩人同時垂降
- 繩子負載超過預期
- 上方的人以為繩子還在被使用

**參考來源**：AMGA, Freedom of the Hills

**標籤**：`溝通口令`, `團隊協作`, `垂降安全`

## Q11: 繩子長度確認

**難度**：⭐⭐

**情境**

你準備在一個 30 公尺的岩壁垂降。

**問題**

使用 60 公尺繩子進行雙繩垂降，最長可以垂降多遠？

**選項**

- A. 60 公尺
- B. 30 公尺
- C. 略少於 30 公尺（要扣除打結的長度）
- D. 120 公尺

**正確答案**：C

**解釋**

**雙繩垂降的長度計算**：

```
60m 繩子對折

    固定點
      │
     ╱ ╲
30m │   │ 30m
    │   │
    └───┘
```

**實際可用長度**：
- 理論：60m ÷ 2 = 30m
- 實際：**略少於 30m**

**減少的原因**：
- 繩子中點打結佔用長度
- 固定點連接佔用長度
- 繩尾結佔用長度
- 安全餘裕

**計算範例**：
- 60m 繩子
- 中點結：-1m
- 固定點連接：-0.5m
- 繩尾結：-0.5m
- **實際約 28m**

**安全建議**：
- 預留 2-3 公尺餘裕
- 不確定時，打繩尾結
- 較長的垂降分段進行

**參考來源**：AMGA, VDiff Climbing

**標籤**：`繩子長度`, `雙繩垂降`, `計算`

## Q12: 倒垂地形的處理

**難度**：⭐⭐⭐

**情境**

你在垂降時遇到一段倒垂（overhang）地形，腳無法接觸岩壁。

**問題**

通過倒垂地形時，最重要的是？

**選項**

- A. 加快速度衝過去
- B. 控制好速度，保持穩定，必要時用手扶繩
- C. 放開雙手抓岩壁
- D. 閉上眼睛

**正確答案**：B

**解釋**

**倒垂地形的垂降技巧**：

```
    ┐
    │
    │  ← 垂直段
    │
    └───┐
        │  ← 倒垂段（腳離開岩壁）
        │
        │
    ────┘
```

**關鍵要點**：

1. **預期會離開岩壁**
   - 知道什麼時候會進入倒垂
   - 心理準備

2. **控制速度**
   - 進入倒垂前減速
   - 不要讓自己擺盪

3. **保持平衡**
   - 上方手可以扶住繩子（不是制動手！）
   - 核心收緊，避免旋轉

4. **平穩下降**
   - 等速下降
   - 不要忽快忽慢

**避免**：
- ❌ 快速通過 → 會擺盪撞岩壁
- ❌ 放開制動手 → 失去控制
- ❌ 亂踢亂動 → 會旋轉

**參考來源**：AMGA, Petzl

**標籤**：`倒垂`, `進階技術`, `地形處理`

## Q13: 緊急停止

**難度**：⭐⭐⭐

**情境**

垂降途中，你發現繩子下方有障礙物需要清除。

**問題**

如何在垂降中途安全停止並空出雙手？

**選項**

- A. 叫別人幫忙抓住繩子
- B. 讓 Autoblock 鎖住，或打義大利半扣鎖繩
- C. 用腿夾住繩子
- D. 把繩子綁在吊帶上

**正確答案**：B

**解釋**

**垂降中途停止的方法**：

**方法一：使用 Autoblock**
- 如果有裝 Autoblock，鬆手即可停止
- 這是最簡單的方式
- 這也是為什麼建議永遠裝備份

**方法二：Mule Knot（騾結）**
```
    ATC
     │
     │
    ╭╮  ← Mule Knot
    ││
    ╰╯
     │
```
- 在確保器下方打結鎖住繩子
- 可以完全空出雙手
- 需要練習才能熟練

**方法三：義大利半扣 + 騾結**
- 適用於需要較長時間停止
- 更加安全可靠

**不建議的方式**：
- ❌ 用腿夾繩子（會滑、會燙傷）
- ❌ 依賴別人抓繩（不可靠）
- ❌ 單手操作其他事（危險）

**參考來源**：AMGA, Freedom of the Hills, VDiff Climbing

**標籤**：`緊急停止`, `Mule Knot`, `垂降技術`

## Q14: 單繩垂降

**難度**：⭐⭐⭐

**情境**

由於地形限制，你需要進行單繩垂降。

**問題**

單繩垂降與雙繩垂降相比，需要特別注意什麼？

**選項**

- A. 完全一樣，沒有差別
- B. 繩子需要固定在固定點，無法抽繩
- C. 速度會比較快
- D. 不需要打繩尾結

**正確答案**：B

**解釋**

**單繩 vs 雙繩垂降**：

| 項目 | 單繩 | 雙繩 |
|------|------|------|
| 繩子固定 | 綁死在固定點 | 繩中穿過固定點 |
| 可用長度 | 整條繩子 | 繩子的一半 |
| 抽繩 | **無法抽繩** | 可以抽繩 |
| 繩子回收 | 需要爬回去拿 | 從下方抽回 |

**單繩垂降的情況**：
- 只需要單次下降（不用抽繩）
- 有人會從上方帶繩子下來
- 繩子留在原地供其他人使用

**繩尾結仍然必要**：
- 不管單繩雙繩都要打繩尾結
- 防止滑出確保器

**參考來源**：AMGA, VDiff Climbing

**標籤**：`單繩垂降`, `繩子回收`, `技術比較`

## Q15: 多繩距垂降的固定點轉換

**難度**：⭐⭐⭐

**情境**

你正在進行多繩距垂降，到達中間的固定點。

**問題**

到達中途固定點後，正確的程序是？

**選項**

- A. 直接開始抽繩
- B. 先自我確保，再拆除垂降系統，重新架設下一段
- C. 把繩子丟下去，看會不會到底
- D. 等上面的人下來再說

**正確答案**：B

**解釋**

**多繩距垂降的轉換程序**：

1. **到達固定點**
   - 在固定點上方停止
   - 不要直接掛上去

2. **建立自我確保**
   - 用 PAS 或繩環連接到固定點
   - 確認承重

3. **喊「Off Rappel」**
   - 通知上方的人

4. **拆除垂降系統**
   - 解開 Autoblock
   - 把繩子從確保器取出

5. **架設下一段**
   - 抽繩（如果需要）
   - 穿過新的固定點
   - 重新架設確保器

6. **開始下一段垂降**
   - 確認系統正確
   - 解除自我確保
   - 繼續下降

**關鍵原則**：
- 永遠保持至少一個連接點
- 不要在沒有備份的情況下操作

**參考來源**：AMGA, Freedom of the Hills, VDiff Climbing

**標籤**：`多繩距`, `固定點轉換`, `垂降程序`
