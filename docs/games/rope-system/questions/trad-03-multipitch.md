## Q1: 先鋒到站的第一步

**難度**：⭐⭐

**情境**

你是先鋒，剛到達一個確保站。

**問題**

到達確保站後，第一個動作應該是什麼？

**選項**

- A. 立刻呼叫「固定」
- B. 先自我確保到固定點，確認承重後才呼叫「固定」
- C. 開始收繩
- D. 坐下來休息

**正確答案**：B

**解釋**

**先鋒到站的正確順序**：

```
1. 到達確保站
      │
      ▼
2. 檢查固定點狀態
      │
      ▼
3. 自我確保連接到固定點
      │
      ▼
4. 確認自我確保承重
      │
      ▼
5. 呼叫「固定」(Safe/Off Belay)
      │
      ▼
6. 開始架設收繩系統
```

**為什麼這個順序很重要**：

| 步驟 | 原因 |
|------|------|
| 先自我確保 | 確保你在解除確保前已經安全 |
| 確認承重 | 測試自我確保是否可靠 |
| 再喊固定 | 確保者才能解除確保 |

**常見錯誤**：
- ❌ 還沒自我確保就喊固定
- ❌ 自我確保沒測試就當作沒事
- ❌ 急著收繩忘了自我確保

**參考來源**：AMGA, Freedom of the Hills

**標籤**：`先鋒到站`, `自我確保`, `系統轉換`

## Q2: 確保站的溝通口令

**難度**：⭐⭐

**情境**

你在多繩距攀登中需要與夥伴溝通。

**問題**

當先鋒喊「固定」(Safe) 後，確保者的正確回應和動作是？

**選項**

- A. 不需要回應，直接解除確保
- B. 回應「收到」，解除確保，準備攀爬
- C. 繼續確保，不要理會
- D. 開始收拾裝備

**正確答案**：B

**解釋**

**多繩距標準溝通流程**：

| 口令 | 說的人 | 意義 | 回應 |
|------|--------|------|------|
| **固定 (Safe)** | 先鋒 | 我已自我確保，安全了 | 收到 |
| **解除確保 (Off Belay)** | 先鋒 | 你可以解除確保了 | 解除 |
| **收繩 (Take in Slack)** | 先鋒 | 我要收繩了 | 收到 |
| **確保完成 (Belay On)** | 先鋒 | 我準備好確保你了 | 收到 |
| **攀登 (Climbing)** | 跟攀者 | 我要開始爬了 | 請上 (Climb On) |

**為什麼要回應**：
- 確認聽到且理解
- 避免溝通錯誤
- 風大或距離遠時特別重要

**聽不清楚時**：
- 不要猜測，請對方重複
- 可以用拉繩子信號
- 考慮使用對講機

**參考來源**：AMGA, Freedom of the Hills, VDiff Climbing

**標籤**：`溝通口令`, `團隊協調`, `多繩距`

## Q3: 收繩方式

**難度**：⭐⭐

**情境**

你在確保站準備收繩確保跟攀者。

**問題**

收繩時，繩子應該如何整理？

**選項**

- A. 隨便堆在腳邊
- B. 整齊疊放或掛在裝備環上，避免糾纏
- C. 全部塞進背包
- D. 讓繩子垂到下一個確保站

**正確答案**：B

**解釋**

**繩索管理的方法**：

**方法一：腳踏法**
```
     你
   ╱    ╲
  腿    腿
   ╲    ╱
    繩子  ← 繩子放在腳上方
   ╱    ╲
  岩壁  岩壁
```
- 繩子搭在腿上或腳上
- 阻止繩子掉落或纏繞
- 常用於較小的確保站

**方法二：蝴蝶盤繩**
```
  ┌──────────┐
  │  ╱╲╱╲╱╲  │
  │ ╱      ╲ │
  └──────────┘
```
- 繩子盤成蝴蝶狀
- 掛在裝備環或固定點
- 確保下次放繩順暢

**方法三：繩袋/繩筐**
- 直接收入繩袋
- 適合有足夠空間的確保站

**為什麼管理很重要**：
- 防止繩子糾纏
- 確保下一段放繩順暢
- 避免繩子卡住或掉落

**參考來源**：AMGA, VDiff Climbing

**標籤**：`收繩`, `繩索管理`, `確保站`

## Q4: 交換先鋒

**難度**：⭐⭐⭐

**情境**

你們採用輪流先鋒（Swinging Leads）的方式攀登。

**問題**

跟攀者到達確保站後，交換先鋒的正確程序是？

**選項**

- A. 跟攀者直接繼續往上爬
- B. 跟攀者自我確保，原先鋒交出裝備，設定新的確保系統後開始下一段
- C. 兩人一起爬
- D. 每次都同一人先鋒

**正確答案**：B

**解釋**

**交換先鋒的步驟**：

```
跟攀者到達
     │
     ▼
自我確保到固定點
     │
     ▼
原先鋒將裝備轉給新先鋒
     │
     ▼
原先鋒設定確保（現在變成確保者）
     │
     ▼
新先鋒準備出發
     │
     ▼
開始下一段
```

**裝備轉移**：
- 新先鋒需要快扣、保護裝備
- 檢查有足夠裝備
- 確認沒有遺漏

**確保系統切換**：
- 原先鋒從「被確保」變成「確保別人」
- 確認確保器正確架設
- 繩子方向正確

**輪流先鋒的優點**：
- 兩人都能休息
- 公平分配風險
- 利用各自擅長的地形

**參考來源**：AMGA, Freedom of the Hills

**標籤**：`交換先鋒`, `Swinging Leads`, `系統轉換`

## Q5: 繩子的兩端管理

**難度**：⭐⭐

**情境**

在多繩距攀登中，你要管理繩子的兩端。

**問題**

繩尾（你這端）應該如何處理？

**選項**

- A. 不用管，讓它自然垂著
- B. 綁在固定點或吊帶上，並打上繩尾結
- C. 剪掉不需要的部分
- D. 交給跟攀者拿著

**正確答案**：B

**解釋**

**繩尾處理的重要性**：

```
固定點
  │
確保器
  │
繩子（往跟攀者）
  │
  │
繩尾 ← 這端要處理！
  │
  ▼
綁住或打結
```

**處理方式**：

1. **綁在固定點上**
   - 確保繩尾不會意外脫離
   - 特別是繩子長度剛好時

2. **綁在吊帶上**
   - 確保繩子不會滑出確保器
   - 即使意外也有備份

3. **打繩尾結**
   - 大結防止滑過確保器
   - 最後一道防線

**為什麼重要**：
- 防止繩子意外滑出系統
- 可能因收繩過多而忘記
- 致命事故的常見原因

**參考來源**：AAC Accidents, AMGA

**標籤**：`繩尾處理`, `安全措施`, `繩索管理`

## Q6: 同時攀登（Simul-climbing）

**難度**：⭐⭐⭐

**情境**

你和夥伴考慮在某些路段使用同時攀登。

**問題**

同時攀登是什麼？什麼情況適合使用？

**選項**

- A. 兩人沒有繩子連接，各爬各的
- B. 兩人同時移動，繩子連接但沒有確保站，適合簡單地形加快速度
- C. 兩人面對面攀爬
- D. 任何時候都適合使用

**正確答案**：B

**解釋**

**同時攀登（Simul-climbing）**：

```
     先鋒
       │
    保護點
       │
    保護點
       │
    保護點
       │
     跟攀者

兩人同時移動，中間有繩子和保護點連接
```

**特點**：
- 兩人同時移動
- 比傳統確保快很多
- 有保護點但沒有確保站

**適合情況**：
| 適合 | 不適合 |
|------|--------|
| 簡單地形 | 困難路段 |
| 兩人能力相當 | 能力差異大 |
| 需要趕時間 | 不熟悉的路線 |
| 經驗豐富 | 新手 |

**風險**：
- 一人墜落可能拉下另一人
- 需要非常好的默契
- 保護點必須可靠

**注意事項**：
- 保持適當間距
- 保護點要夠頻繁
- 雙方持續溝通

**參考來源**：AMGA, Freedom of the Hills

**標籤**：`同時攀登`, `Simul-climbing`, `進階技術`

## Q7: 確保站的選擇

**難度**：⭐⭐

**情境**

你在先鋒攀登，需要決定在哪裡建立確保站。

**問題**

選擇確保站位置時，最重要的考量是？

**選項**

- A. 選最舒適的地方
- B. 有足夠保護點、良好站位、繩長允許的地方
- C. 選最高的地方
- D. 選有陰影的地方

**正確答案**：B

**解釋**

**確保站位置的選擇標準**：

| 優先順序 | 考量因素 |
|----------|----------|
| 1 | 有足夠的保護點可架設 |
| 2 | 站位或坐位舒適 |
| 3 | 繩長範圍內（不超過繩子一半） |
| 4 | 避免落石風險區 |
| 5 | 便於溝通 |

**理想確保站特徵**：
```
     ┌─────────────────┐
     │ ✓ 有岩縫可放裝備 │
     │ ✓ 有平台可站立   │
     │ ✓ 避開落石線     │
     │ ✓ 視野良好       │
     └─────────────────┘
```

**不適合的位置**：
- ❌ 沒有保護點
- ❌ 懸吊確保（除非必要）
- ❌ 落石區正下方
- ❌ 繩子不夠長

**參考來源**：AMGA, Freedom of the Hills

**標籤**：`確保站選擇`, `位置`, `路線規劃`

## Q8: 跟攀者的職責

**難度**：⭐⭐

**情境**

你是跟攀者，準備開始攀爬。

**問題**

跟攀者在攀爬過程中的主要職責是？

**選項**

- A. 只要跟著繩子爬就好
- B. 清除保護點、管理裝備、與先鋒保持溝通
- C. 照相記錄
- D. 評論先鋒的路線選擇

**正確答案**：B

**解釋**

**跟攀者的職責清單**：

**1. 清除保護點**
- 每個快扣和保護裝備都要回收
- 正確使用 Nut Tool
- 不要遺漏任何裝備

**2. 管理裝備**
- 整理在裝備環上
- 按類型和大小排列
- 方便交給先鋒

**3. 保持溝通**
- 回應先鋒的口令
- 報告清除進度
- 提醒繩子狀況

**4. 路線觀察**
- 記住先鋒走的路徑
- 注意困難動作
- 學習保護點放置

**清除困難時**：
- 不要硬拔（可能損壞裝備）
- 多角度嘗試
- 真的拿不出來要告知先鋒

**參考來源**：AMGA, VDiff Climbing

**標籤**：`跟攀者`, `清除裝備`, `職責`

## Q9: 確保站的受力方向

**難度**：⭐⭐⭐

**情境**

你在架設確保站時考慮受力方向。

**問題**

確保站需要承受哪些方向的力量？

**選項**

- A. 只有向下（跟攀者的重量）
- B. 向下（跟攀者）和向上（先鋒墜落經過轉向）
- C. 只有向上
- D. 只有水平方向

**正確答案**：B

**解釋**

**確保站的受力分析**：

```
情況1：確保跟攀者        情況2：先鋒墜落

      確保站                    下一段路線
         │                          │
         ↓ 向下的力              保護點
         │                          │
      確保者                    確保站 ← 向上的力
         │                          │
      跟攀者                     確保者
```

**受力情況說明**：

| 情況 | 受力方向 | 來源 |
|------|----------|------|
| 確保跟攀者 | 向下 | 跟攀者的重量 |
| 確保跟攀者墜落 | 向下（較大） | 跟攀者墜落衝擊 |
| 先鋒墜落 | **向上** | 繩子經過保護點轉向 |

**設計原則**：
- 確保站要能承受多方向力
- 有些保護點專門對抗向上
- 有些專門對抗向下
- 或使用可多向承力的系統

**為什麼向上力很重要**：
- 經常被忽略
- 先鋒墜落時會產生
- 可能把向下式固定點拉出

**參考來源**：AMGA, Climbing Anchors

**標籤**：`受力方向`, `向上力`, `固定點設計`

## Q10: 繩距長度規劃

**難度**：⭐⭐

**情境**

你在規劃一條多繩距路線。

**問題**

每一段繩距的長度應該如何規劃？

**選項**

- A. 每段都要用完整條繩子
- B. 根據繩子長度、保護點位置和難度來決定，留有餘裕
- C. 越長越好
- D. 固定每段 30 公尺

**正確答案**：B

**解釋**

**繩距規劃的考量**：

| 因素 | 說明 |
|------|------|
| **繩子長度** | 不要超過一半（需要收繩） |
| **保護點** | 在好的位置建立確保站 |
| **難度** | 難段落可能需要更頻繁的確保站 |
| **溝通** | 太遠聽不到對方 |

**實際計算**：
```
60m 繩子
├── 先鋒攀爬：最多 ~28m（留餘裕）
└── 剩餘繩子：收繩用
```

**規劃建議**：

1. **研究路線資訊**
   - 查閱攀岩指南
   - 了解確保站位置

2. **保留彈性**
   - 可能需要臨時調整
   - 保護點不一定在預期位置

3. **溝通距離**
   - 超過 25-30m 可能聽不清楚
   - 考慮帶對講機

**參考來源**：Freedom of the Hills, AMGA

**標籤**：`繩距規劃`, `路線規劃`, `長度計算`

## Q11: 轉向（Redirect）的使用

**難度**：⭐⭐⭐

**情境**

你在確保站確保先鋒，繩子的方向很奇怪。

**問題**

什麼時候應該使用轉向（Redirect）？

**選項**

- A. 每次都要用
- B. 當直接確保會讓確保者被拉向不舒服的位置或有危險時
- C. 只有垂降時使用
- D. 轉向會降低安全性，不應該用

**正確答案**：B

**解釋**

**轉向（Redirect）的概念**：

```
無轉向：                有轉向：

   先鋒                    先鋒
     │                       │
     │                    轉向點
     │                       │
   確保器                   確保器
     │                       │
   確保者 ← 可能被拉起     確保者 ← 穩定
```

**使用時機**：

| 情況 | 是否需要轉向 |
|------|--------------|
| 確保者位置低，可能被拉起 | ✓ 需要 |
| 繩子會摩擦岩角 | ✓ 需要 |
| 確保者站位不穩 | ✓ 需要 |
| 正常情況 | ✗ 不需要 |

**轉向的優點**：
- 穩定確保者位置
- 減少被拉動的風險
- 調整繩子路徑

**轉向的缺點**：
- 增加系統摩擦力
- 增加固定點負載
- 架設較複雜

**參考來源**：AMGA, VDiff Climbing

**標籤**：`轉向`, `Redirect`, `確保技術`

## Q12: 處理繩子糾纏

**難度**：⭐⭐

**情境**

收繩時發現繩子嚴重糾纏。

**問題**

預防和處理繩子糾纏的最佳方式是？

**選項**

- A. 用力拉就會解開
- B. 預防：收繩時注意整理；處理：耐心從外圈開始解開
- C. 把糾纏的部分剪掉
- D. 忽略糾纏，繼續攀爬

**正確答案**：B

**解釋**

**預防糾纏**：

| 時機 | 方法 |
|------|------|
| 收繩時 | 整齊堆放或盤繩 |
| 放繩時 | 從上方開始放 |
| 攜帶時 | 使用繩袋 |
| 整理時 | 定期檢查繩子狀態 |

**處理糾纏**：

```
步驟：
1. 保持耐心（急躁會更糟）
2. 找到繩子的兩端
3. 從外圈開始慢慢解
4. 不要用力拉扯
5. 必要時把整個結構鬆開
```

**蝴蝶盤繩技巧**：
```
  ╱─────╲
 ╱   │   ╲
│    │    │
│    │    │
 ╲   │   ╱
  ╲──│──╱

中間是收繩處
兩側是鬆散的圈
```
- 下次放繩時不會糾纏
- 繩子會依序滑出

**參考來源**：VDiff Climbing, AMGA

**標籤**：`繩子糾纏`, `繩索整理`, `預防措施`

## Q13: 懸吊確保

**難度**：⭐⭐⭐

**情境**

確保站位置沒有可以站立的平台，必須懸吊確保。

**問題**

懸吊確保時的特別注意事項是？

**選項**

- A. 和普通確保完全一樣
- B. 確保繩子不會卡在身體下方，且固定點必須特別可靠
- C. 不需要自我確保
- D. 只能用義大利半扣確保

**正確答案**：B

**解釋**

**懸吊確保的挑戰**：

```
         固定點
            │
            │
        自我確保
            │
         你（懸吊）
            │
         確保器
            │
         跟攀者
```

**特別注意事項**：

| 項目 | 說明 |
|------|------|
| **固定點** | 必須特別可靠，承受全部重量 |
| **自我確保** | 要舒適且安全 |
| **繩子路徑** | 確保繩子不會卡在身體下方 |
| **操作空間** | 確保有足夠空間操作 |

**實際技巧**：
- 確保器掛在吊帶確保環
- 調整自我確保長度讓操作舒適
- 繩子走在身體側邊
- 考慮使用腳環增加穩定

**裝備建議**：
- 較長的 PAS 或 Daisy Chain
- 可鎖定的確保器（如 GriGri）更安全
- 考慮帶吊帶坐墊（Buttrest）

**參考來源**：AMGA, Freedom of the Hills

**標籤**：`懸吊確保`, `無平台`, `進階技術`

## Q14: 撤退決策

**難度**：⭐⭐⭐

**情境**

你們在多繩距路線的中途遇到困難。

**問題**

什麼情況應該考慮撤退？

**選項**

- A. 永遠不應該撤退
- B. 天氣惡化、體力不足、天黑前無法完成、隊員受傷時
- C. 只有下雨才撤退
- D. 只要還有裝備就繼續

**正確答案**：B

**解釋**

**撤退決策的考量因素**：

| 因素 | 撤退指標 |
|------|----------|
| **天氣** | 雷暴、大雨、強風即將到來 |
| **時間** | 無法在天黑前完成/撤離 |
| **體力** | 明顯疲勞、影響判斷 |
| **傷病** | 隊員受傷或身體不適 |
| **路線** | 難度超出能力 |
| **裝備** | 裝備不足或故障 |

**撤退不是失敗**：
- 山永遠在那裡
- 人的安全最重要
- 好的判斷是進步的一部分

**撤退的方式**：
1. **垂降**：如果有固定點
2. **原路攀降**：如果可行
3. **等待救援**：最壞情況

**提前規劃**：
- 出發前了解撤退路線
- 帶足夠的輔助繩（做垂降）
- 設定撤退時間點

**參考來源**：AAC Accidents, AMGA, Freedom of the Hills

**標籤**：`撤退`, `決策判斷`, `安全管理`

## Q15: 多繩距的效率

**難度**：⭐⭐

**情境**

你想提高多繩距攀登的效率。

**問題**

以下哪個做法能有效提高效率？

**選項**

- A. 省略安全檢查
- B. 流暢的系統轉換、良好的繩索管理、清晰的溝通
- C. 每個人都獨自攀爬
- D. 不休息一直爬

**正確答案**：B

**解釋**

**提高效率的關鍵**：

| 領域 | 方法 |
|------|------|
| **系統轉換** | 熟練操作、減少思考時間 |
| **繩索管理** | 預先整理、避免糾纏 |
| **溝通** | 清晰口令、減少誤解 |
| **裝備管理** | 有序排列、快速取用 |

**時間花在哪裡**：
```
典型多繩距時間分配：
├── 攀爬時間：40%
├── 系統轉換：30%  ← 這裡最有改進空間
├── 路線尋找：15%
└── 休息/其他：15%
```

**效率 ≠ 省略安全**：
- 熟練後自然快速
- 安全步驟要完整
- 永遠不要急躁

**練習建議**：
- 在地面練習系統轉換
- 模擬各種情境
- 與固定夥伴培養默契

**參考來源**：AMGA, Freedom of the Hills

**標籤**：`效率`, `時間管理`, `團隊協作`
