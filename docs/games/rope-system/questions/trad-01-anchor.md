# 固定點架設

> 學習傳統攀登中的多點固定點架設、均力系統與評估原則。

**類別代碼**：`trad-anchor`
**題目數量**：15 題
**難度範圍**：⭐⭐ - ⭐⭐⭐

## Q1: SERENE vs ERNEST

**難度**：⭐⭐

**情境**

你正在學習傳統攀登的固定點架設原則。

**問題**

傳統攀登固定點的評估原則中，以下哪個描述最完整？

**選項**

- A. 只要夠堅固就好
- B. SERENE（堅固、均力、冗餘、高效、無延展）
- C. 使用越多裝備越好
- D. 看起來專業即可

**正確答案**：B

**解釋**

**SERENE 原則詳解**：

| 字母 | 原文 | 中文 | 說明 |
|------|------|------|------|
| **S** | Solid | 堅固 | 每個固定點都能承受預期負載 |
| **E** | Equalized | 均力 | 負載平均分配到各固定點 |
| **R** | Redundant | 冗餘 | 至少兩個獨立固定點 |
| **E** | Efficient | 高效 | 簡單快速，減少出錯 |
| **NE** | No Extension | 無延展 | 一點失效不會產生衝擊負載 |

**另一個口訣：ERNEST**
- **E**qualized（均力）
- **R**edundant（冗餘）
- **N**o Extension（無延展）
- **E**fficient（高效）
- **S**olid（堅固）
- **T**imely（及時）

兩個口訣內容相似，都強調相同的關鍵原則。

**參考來源**：AMGA, Climbing Anchors (John Long), Freedom of the Hills

**標籤**：`SERENE`, `ERNEST`, `固定點原則`

## Q2: Cordelette 架設

**難度**：⭐⭐

**情境**

你有一條 6-7 公尺的 Cordelette（輔助繩），要架設三點固定點。

**問題**

使用 Cordelette 架設固定點的正確方式是？

**選項**

- A. 把 Cordelette 綁成一個大圈掛在三個點上
- B. Cordelette 分別穿過三個固定點，在預期受力方向收緊打結
- C. 只需要掛在最強的那個點
- D. 把三個點用獨立繩環分別連接

**正確答案**：B

**解釋**

**Cordelette 架設步驟**：

```
       點1    點2    點3
        │      │      │
        └──────┼──────┘
               │
          Cordelette
               │
          ┌────┴────┐
          │ Overhand │← 在受力方向打結
          │  Knot   │
          └────┬────┘
               │
             主點
```

**正確步驟**：
1. Cordelette 穿過（或掛入）所有固定點
2. 將所有股繩拉到預期受力方向
3. 打一個 Overhand Knot 或 Figure-8
4. 在結下方形成主點

**關鍵要點**：
- 結要在**預期受力方向**
- 所有股繩長度調整均勻
- 結打緊，不要有鬆弛

**Cordelette 的優點**：
- 簡單易學
- 一點失效有限的延展
- 適合固定點相對集中的情況

**參考來源**：AMGA, Climbing Anchors, VDiff Climbing

**標籤**：`Cordelette`, `架設技術`, `均力固定點`

## Q3: Sliding X 的風險

**難度**：⭐⭐⭐

**情境**

你的夥伴用繩環架設了一個 Sliding X（滑動X）固定點。

**問題**

Sliding X 最主要的缺點是什麼？

**選項**

- A. 架設太複雜
- B. 一點失效會造成大幅延展
- C. 不能均力
- D. 只能用在兩個點

**正確答案**：B

**解釋**

**Sliding X 的問題**：

```
正常狀態：              一點失效後：
   點1    點2              點1    ✖
    │      │                │
    ╲    ╱                 │
     ╲  ╱                   │
      ╳  ← Sliding X        │
     ╱ ╲                    │
    │                       │
   主點                   主點（下墜！）
```

**延展距離**：
- 如果繩環長度為 L
- 一點失效可能延展 L/2 或更多
- 延展會造成衝擊負載

**改良方式：Limiting Knot**
```
    點1    點2
     │      │
     ╲    ╱
    ╭─╳─╮  ← 在 X 處打限制結
     │
    主點
```
- 在 X 交叉處打一個結
- 限制最大延展距離
- 仍保持部分均力功能

**結論**：
- 純 Sliding X 不建議用於主要固定點
- 如要使用，必須加限制結

**參考來源**：Climbing Anchors, AMGA, VDiff Climbing

**標籤**：`Sliding X`, `延展風險`, `限制結`

## Q4: 固定點角度

**難度**：⭐⭐

**情境**

你要用繩環連接兩個相距較遠的保護點。

**問題**

繩環的角度超過多少度時，各點承受的力量會開始大幅增加？

**選項**

- A. 30 度
- B. 60 度
- C. 90 度
- D. 120 度

**正確答案**：B

**解釋**

**角度與負載的關係**：

| 角度 | 每點負載（相對於總負載） | 風險評估 |
|------|--------------------------|----------|
| 0° | 50% | 理想 |
| 30° | 52% | 優良 |
| **60°** | **58%** | **可接受臨界** |
| 90° | 71% | 風險增加 |
| 120° | 100% | **危險！** |
| 150° | 193% | 極危險 |

**關鍵數字**：
- **60 度以內**：安全範圍
- **60-90 度**：需要注意
- **超過 90 度**：應該調整

**為什麼角度很重要**：
```
窄角度（好）         寬角度（危險）
   ╱╲                ╱────────╲
  ╱  ╲              ╱          ╲
 點1  點2          點1          點2
   ╲╱                    ╲  ╱
   主點                   主點
```

**解決方案**：
- 使用更長的繩環
- 調整主點位置
- 考慮增加中間點

**參考來源**：AMGA, Climbing Anchors, Petzl

**標籤**：`角度`, `負載分配`, `力學原理`

## Q5: 岩楔放置評估

**難度**：⭐⭐⭐

**情境**

你在一條傳統路線上放置岩楔（Nut）。

**問題**

評估岩楔放置是否良好，最重要的因素是？

**選項**

- A. 岩楔的品牌和新舊程度
- B. 放置點的岩石品質與岩楔的卡合度
- C. 岩楔的顏色是否容易看見
- D. 放置的速度

**正確答案**：B

**解釋**

**岩楔放置的評估要點**：

**1. 岩石品質**：
- ✅ 堅硬、無裂縫的岩石
- ❌ 鬆動、破碎、風化的岩石
- 用手敲擊測試（實心聲 vs 空心聲）

**2. 卡合度**：
```
好的放置：            不好的放置：
┌─────┐               ┌─────┐
│     │               │     │
│ ╲   │               │  │  │
│  ╲══│← 楔形卡緊     │  ║  │← 只有點接觸
│   ╲ │               │  │  │
└─────┘               └─────┘
```

**3. 受力方向**：
- 岩楔應該在預期受力方向上卡緊
- 測試時往受力方向拉扯
- 不應該能夠搖動或旋轉

**4. 深度**：
- 不要太深（難以移除）
- 不要太淺（可能脫出）

**測試方法**：
- 往預期受力方向用力拉
- 觀察是否有任何移動
- 好的放置應該「越拉越緊」

**參考來源**：AMGA, Climbing Anchors, Traditional Lead Climbing (Luebben)

**標籤**：`岩楔`, `放置評估`, `保護裝備`

## Q6: Cam 的放置範圍

**難度**：⭐⭐

**情境**

你正在學習使用 Cam（凸輪裝置 / Friend）。

**問題**

Cam 的理想放置範圍是？

**選項**

- A. 完全收縮（最小狀態）
- B. 凸輪張開約 50-90% 的範圍
- C. 完全張開（最大狀態）
- D. 只要放得進去就好

**正確答案**：B

**解釋**

**Cam 的放置範圍**：

```
完全收縮          理想範圍          完全張開
   ╱╲            ╱    ╲            ╱      ╲
  ╱  ╲          ╱      ╲          ╱        ╲
 ╱    ╲        ╱        ╲        ╱          ╲
━━━━━━━━      ━━━━━━━━━━━━      ━━━━━━━━━━━━━━

❌ 太收縮       ✅ 50-90%        ❌ 太張開
```

**為什麼 50-90% 最好**：

| 狀態 | 問題 |
|------|------|
| 太收縮 | 容易走位、抓力不足 |
| 理想範圍 | 最大抓力、穩定 |
| 太張開 | 抓力降低、可能脫出 |

**實際判斷**：
- 凸輪葉片應該平均接觸岩壁
- 觸發桿還有收縮空間
- 不要「硬塞」或「勉強放入」

**其他注意事項**：
- 放置在平行或內縮的岩縫（不要外擴）
- 四片凸輪都要接觸
- 受力方向正確

**參考來源**：AMGA, Black Diamond, Wild Country

**標籤**：`Cam`, `放置範圍`, `凸輪裝置`

## Q7: 延長保護點

**難度**：⭐⭐

**情境**

你放置了一個保護點，但發現繩子路徑會造成很大的阻力。

**問題**

什麼時候應該延長保護點（加長繩環）？

**選項**

- A. 每個保護點都要延長
- B. 當繩子路徑會產生 Z 字形或大幅轉彎時
- C. 只有最後一個保護點需要
- D. 不需要延長，繩子會自己調整

**正確答案**：B

**解釋**

**繩索阻力問題**：

```
❌ 不延長（高阻力）       ✅ 適當延長（低阻力）
        │                        │
       ╱│                       │
      ╱ │                       │
 保護點  │                  ════│← 延長
      ╲ │                      ╱│
       ╲│                     ╱ │
        │                 保護點 │
```

**需要延長的情況**：
- 繩子會有大幅度轉彎
- 水平走位（Traverse）
- 保護點偏離直線很遠
- 岩角會磨損繩子

**延長的好處**：
1. 減少繩索阻力（更順暢）
2. 減少拉動保護點的側向力
3. 讓確保更靈敏
4. 減少保護點被拉出的風險

**延長長度選擇**：
- 輕微轉彎：60cm 繩環
- 明顯轉彎：120cm 繩環
- 嚴重走位：Alpine Quickdraw

**參考來源**：AMGA, Freedom of the Hills, VDiff Climbing

**標籤**：`延長`, `繩索阻力`, `保護點管理`

## Q8: 自然保護點

**難度**：⭐⭐

**情境**

路線上有一個穩固的岩柱（Horn）可以使用。

**問題**

使用自然保護點時，最重要的評估是？

**選項**

- A. 看起來是否美觀
- B. 岩石的穩固程度與形狀是否能固定繩環
- C. 有沒有其他人用過
- D. 位置是否方便

**正確答案**：B

**解釋**

**自然保護點的類型與評估**：

| 類型 | 評估重點 |
|------|----------|
| 岩柱（Horn） | 底部連接穩固、不會斷裂 |
| 岩穿（Thread） | 孔洞夠大、岩石堅固 |
| 大石塊 | 無法被移動、不會滾落 |
| 樹木 | 活樹、直徑>15cm、根系穩固 |

**測試方法**：

1. **視覺檢查**
   - 有無裂縫
   - 岩石顏色一致（無風化）
   - 連接處是否穩固

2. **物理測試**
   - 用手推拉（不應該移動）
   - 敲擊聽聲音（實心聲 = 好）

3. **繩環放置**
   - 繩環是否會滑脫
   - 受力方向是否正確

**常見錯誤**：
- ❌ 信任外觀漂亮但鬆動的岩石
- ❌ 繩環放置位置會滑動
- ❌ 沒有測試就信任

**參考來源**：AMGA, Climbing Anchors, Freedom of the Hills

**標籤**：`自然保護點`, `岩柱`, `評估方法`

## Q9: Quad 固定點系統

**難度**：⭐⭐⭐

**情境**

你正在學習使用 Quad（四股系統）架設固定點。

**問題**

Quad 系統的主要優點是什麼？

**選項**

- A. 架設最快速
- B. 同時具有均力與限制延展的功能
- C. 只需要一個保護點
- D. 不需要任何結

**正確答案**：B

**解釋**

**Quad 系統架設**：

```
      點1              點2
       │                │
       └──────┬─────────┘
              │
         ╭────┴────╮
         │ 限制結   │← Overhand Knots
         ╰────┬────╯
              │
           滑動段  ← 可以均力
              │
         ╭────┴────╮
         │ 限制結   │← Overhand Knots
         ╰────┬────╯
              │
            主點
```

**Quad 的特點**：

| 特點 | 說明 |
|------|------|
| **均力** | 中間滑動段可自動調整 |
| **限制延展** | 兩端的結限制最大延展 |
| **冗餘** | 四股繩提供備份 |
| **多功能** | 可掛確保、自我確保 |

**架設步驟**：
1. 繩環對折成四股
2. 在兩端各打一個限制結
3. 中間留 15-25cm 滑動空間
4. 兩端分別掛入固定點

**優於 Sliding X**：
- 有限制結，延展受控
- 仍保持自動均力
- 更加安全可靠

## Q10: 固定點的方向性

**難度**：⭐⭐⭐

**情境**

你在多繩距路線的確保站架設固定點。

**問題**

固定點的方向應該考慮什麼？

**選項**

- A. 只考慮向下的力量
- B. 考慮可能的受力方向：向下（確保跟攀）和向上（先鋒墜落）
- C. 只考慮向上的力量
- D. 不需要考慮方向

**正確答案**：B

**解釋**

**固定點的多向受力**：

```
向上（先鋒墜落）       向下（確保跟攀）
      ↑                     ↓
      │                     │
  ┌───────┐             ┌───────┐
  │ 固定點 │             │ 固定點 │
  └───────┘             └───────┘
      │                     │
      ↓                     ↓
    確保者                 確保者
```

**不同情境的受力方向**：

| 情境 | 主要受力方向 |
|------|--------------|
| 確保跟攀者 | 向下 |
| 先鋒墜落（有轉向） | 向上 |
| 垂降 | 向下 |
| 拖拉背包 | 向下 |

**設計原則**：
1. **多向性固定點**
   - 有些保護點對抗向上
   - 有些保護點對抗向下
   - 都連接到同一個主點

2. **或者分開架設**
   - 先鋒墜落用一組
   - 確保跟攀用一組

**為什麼向上力量很重要**：
- 先鋒墜落時，繩子會經過最後一個保護點
- 這會產生向上的拉力
- 如果固定點只對抗向下，可能失效

**參考來源**：AMGA, Freedom of the Hills, Climbing Anchors

**標籤**：`方向性`, `多向固定點`, `受力分析`

## Q11: 固定點的測試

**難度**：⭐⭐

**情境**

你完成了固定點架設，準備開始使用。

**問題**

使用固定點之前，應該如何測試？

**選項**

- A. 用眼睛看一下就好
- B. 整體系統施加體重測試，檢查每個元件
- C. 讓夥伴先試
- D. 直接開始使用，有問題再說

**正確答案**：B

**解釋**

**固定點測試流程**：

**1. 視覺檢查**：
- 每個保護點都正確放置
- 繩環/Cordelette 沒有扭轉
- 鉤環都已鎖緊
- 沒有遺漏的環節

**2. 施加體重測試**：
```
步驟：
1. 先連接自我確保（有備份）
2. 慢慢將體重轉移到固定點
3. 觀察每個保護點是否移動
4. 用力拉扯測試
5. 確認後才完全信任
```

**3. 檢查清單**：
- [ ] 所有保護點穩固
- [ ] 角度小於 60 度
- [ ] 有冗餘（多個保護點）
- [ ] 無過度延展風險
- [ ] 主點正確設置
- [ ] 所有鉤環鎖緊

**為什麼要測試**：
- 發現放置不良的保護點
- 確認系統整體強度
- 增加對系統的信心

**參考來源**：AMGA, Climbing Anchors

**標籤**：`測試`, `安全確認`, `使用前檢查`

## Q12: 保護點數量

**難度**：⭐⭐

**情境**

你正在架設一個傳統攀登的確保站。

**問題**

標準的確保站應該使用幾個保護點？

**選項**

- A. 1 個就夠了
- B. 通常 2-3 個，視保護點品質而定
- C. 一定要 5 個以上
- D. 數量不重要

**正確答案**：B

**解釋**

**保護點數量的原則**：

| 保護點品質 | 建議數量 |
|------------|----------|
| 優良（如：堅固的 bolt） | 2 個 |
| 良好（如：好的岩楔/Cam） | 2-3 個 |
| 一般（如：中等品質） | 3-4 個 |
| 較差（如：勉強可用） | 考慮增加或放棄 |

**判斷標準**：
- 每個保護點都應該能獨立承受全部負載
- 「兩個好的勝過五個差的」
- 品質優先，數量其次

**兩個好保護點的情況**：
```
     Cam        岩楔
    (好)       (好)
      │          │
      └────┬─────┘
           │
         主點
```
→ 這比四個不確定的保護點更安全

**過多保護點的問題**：
- 浪費時間和裝備
- 增加複雜度
- 可能增加出錯機會

**參考來源**：AMGA, Climbing Anchors, Freedom of the Hills

**標籤**：`保護點數量`, `品質評估`, `架設原則`

## Q13: 不同材質繩環的選擇

**難度**：⭐⭐⭐

**情境**

你在選擇架設固定點的繩環材質。

**問題**

以下關於繩環材質的描述，何者正確？

**選項**

- A. Dyneema 比 Nylon 更耐高溫
- B. Nylon 比 Dyneema 更能吸收衝擊
- C. 兩種材質完全一樣
- D. Dyneema 更適合打結使用

**正確答案**：B

**解釋**

**繩環材質比較**：

| 特性 | Nylon（尼龍） | Dyneema（高分子聚乙烯） |
|------|---------------|-------------------------|
| 強度 | 高 | 非常高 |
| 重量 | 較重 | 較輕 |
| 延展性 | 有（吸收衝擊） | 幾乎沒有 |
| 耐高溫 | 較好 | **較差**（~70°C 開始弱化） |
| 耐磨損 | 普通 | 較差 |
| 打結後強度 | 保留較多 | 損失較多 |

**實際應用**：

| 情況 | 建議選擇 |
|------|----------|
| 需要吸收衝擊 | Nylon |
| 可能摩擦生熱 | Nylon |
| 需要打結 | Nylon |
| 重量敏感 | Dyneema |
| 純靜態使用 | 都可以 |

**特別注意**：
- Dyneema 在快速滑動（垂降、降低）時可能因摩擦過熱
- 不建議用 Dyneema 繩環直接給繩子摩擦
- 打結會顯著降低 Dyneema 強度（30-50%）

**參考來源**：Petzl, Black Diamond, DMM

**標籤**：`繩環材質`, `Dyneema`, `Nylon`

## Q14: Pre-equalized vs Self-equalizing

**難度**：⭐⭐⭐

**情境**

你在學習不同的固定點均力系統。

**問題**

Pre-equalized（預先均力）和 Self-equalizing（自動均力）的主要差異是？

**選項**

- A. 完全沒有差異
- B. Pre-equalized 在架設時固定分配，Self-equalizing 可隨受力方向自動調整
- C. Self-equalizing 比較弱
- D. Pre-equalized 只能用兩個點

**正確答案**：B

**解釋**

**兩種均力系統比較**：

**Pre-equalized（預先均力）**：
```
    點1      點2
     │        │
     └────────┘
          │
       ═══╧═══  ← 打結固定
          │
        主點
```
- 架設時固定好分配
- 不會隨受力方向改變
- 無延展風險
- 範例：Cordelette

**Self-equalizing（自動均力）**：
```
    點1      點2
     │        │
     └───╳────┘
          │    ← 可以滑動
        主點
```
- 主點可以移動
- 自動調整到最佳分配
- 有延展風險（需加限制結）
- 範例：Sliding X, Quad

**如何選擇**：

| 情況 | 建議 |
|------|------|
| 受力方向明確 | Pre-equalized |
| 受力方向可能變化 | Self-equalizing（加限制結） |
| 快速架設 | Pre-equalized |
| 需要精確均力 | Self-equalizing |

**參考來源**：AMGA, Climbing Anchors, VDiff Climbing

**標籤**：`均力系統`, `Pre-equalized`, `Self-equalizing`

## Q15: 確保站的整體評估

**難度**：⭐⭐⭐

**情境**

你完成了一個多繩距確保站的架設。

**問題**

最後的整體評估應該確認什麼？

**選項**

- A. 只要自己覺得安全就好
- B. 確認符合 SERENE 原則且測試過每個環節
- C. 拍照給朋友看
- D. 確認裝備數量正確

**正確答案**：B

**解釋**

**整體評估檢查清單**：

**S - Solid（堅固）**：
- [ ] 每個保護點都經過測試
- [ ] 岩石品質良好
- [ ] 保護點不會移動

**E - Equalized（均力）**：
- [ ] 負載分配合理
- [ ] 角度小於 60 度
- [ ] 主點位置正確

**R - Redundant（冗餘）**：
- [ ] 至少兩個獨立保護點
- [ ] 使用雙鉤環或反向對扣
- [ ] 沒有單點失效風險

**E - Efficient（高效）**：
- [ ] 系統簡潔明瞭
- [ ] 沒有不必要的複雜
- [ ] 便於操作使用

**NE - No Extension（無延展）**：
- [ ] 有限制結或預先均力
- [ ] 一點失效延展有限
- [ ] 不會產生衝擊負載

**最終確認**：
- 全系統承重測試
- 確認所有鉤環鎖緊
- 確認自己了解整個系統

**心態**：
「如果我要在這個固定點上睡一晚，我放心嗎？」

**參考來源**：AMGA, Climbing Anchors, Freedom of the Hills

**標籤**：`整體評估`, `SERENE檢查`, `確保站`
