## Q1: 被動式 vs 主動式保護裝備

**難度**：⭐⭐

**情境**

你正在學習傳統攀登的保護裝備分類。

**問題**

以下關於被動式（Passive）與主動式（Active）保護裝備的描述，何者正確？

**選項**

- A. 被動式有彈簧機構，主動式沒有
- B. 被動式沒有活動部件，主動式有活動部件（如凸輪）
- C. 兩者完全一樣
- D. 主動式只能用在寬裂縫

**正確答案**：B

**解釋**

**保護裝備分類**：

| 類型 | 特徵 | 範例 |
|------|------|------|
| **被動式** | 沒有活動部件，靠形狀卡住 | 岩楔(Nut)、Hex、Tricam |
| **主動式** | 有活動部件（彈簧凸輪） | Cam (Friend)、Camalot |

**被動式優點**：
- 結構簡單，不易故障
- 重量輕
- 價格較低
- 可放置在淺裂縫

**被動式缺點**：
- 需要裂縫形狀配合
- 受力方向較單一
- 需要較多技術判斷

**主動式優點**：
- 適用範圍廣
- 放置快速
- 受力方向較寬容

**主動式缺點**：
- 較重、較貴
- 機械部件可能故障
- 不適合太淺的裂縫

**參考來源**：AMGA, Freedom of the Hills, Climbing Anchors

**標籤**：`被動式`, `主動式`, `保護裝備分類`

## Q2: 岩楔的放置原則

**難度**：⭐⭐

**情境**

你要在一條收縮的裂縫中放置岩楔（Nut）。

**問題**

岩楔應該放置在什麼樣的裂縫形狀？

**選項**

- A. 平行的裂縫
- B. 向內收縮的裂縫（內窄外寬）
- C. 向外擴張的裂縫（內寬外窄）
- D. 任何形狀都可以

**正確答案**：B

**解釋**

**裂縫形狀與岩楔放置**：

```
✅ 收縮裂縫（好）      ❌ 擴張裂縫（不好）

   ╲    ╱              ╱    ╲
    ╲  ╱              ╱      ╲
     ╲╱ ← 岩楔卡住    ╱        ╲ ← 岩楔會掉出
    ╱╲              ╱          ╲
   ╱  ╲            ╱            ╲
```

**放置要點**：

1. **找收縮處**
   - 裂縫越往裡越窄
   - 岩楔會「越拉越緊」

2. **接觸面積**
   - 岩楔表面要充分接觸岩石
   - 不是只有邊緣接觸

3. **受力方向**
   - 放置時考慮墜落方向
   - 岩楔應該往那個方向卡緊

**選擇正確尺寸**：
- 不要太大（硬塞）
- 不要太小（容易滑出）
- 應該剛好卡緊

**參考來源**：AMGA, Climbing Anchors, Traditional Lead (Luebben)

**標籤**：`岩楔`, `放置技術`, `裂縫形狀`

## Q3: Cam 的凸輪角度

**難度**：⭐⭐⭐

**情境**

你正在放置一個 Cam，觀察凸輪的角度。

**問題**

Cam 的凸輪角度約在什麼範圍時，抓力最強？

**選項**

- A. 完全收縮（0-25%）
- B. 中間範圍（50-90%）
- C. 接近全開（90-100%）
- D. 角度不影響抓力

**正確答案**：B

**解釋**

**凸輪角度與抓力的關係**：

```
收縮     25%      50%      75%      張開
  │       │        │        │        │
  ╱╲     ╱ ╲      ╱  ╲     ╱   ╲    ╱    ╲
 ╱  ╲   ╱   ╲    ╱    ╲   ╱     ╲  ╱      ╲
  ▓▓     ▓▓▓      ████      ▓▓▓      ▓▓

 差      可       最佳      可       差
```

**為什麼 50-90% 最好**：

| 範圍 | 抓力 | 問題 |
|------|------|------|
| 0-25% | 較弱 | 可能走位、凸輪打滑 |
| **50-90%** | **最強** | **理想放置** |
| 90-100% | 較弱 | 抓力降低、可能脫出 |

**凸輪工作原理**：
- 凸輪是對數螺旋形狀
- 受力時會試圖張開
- 適當的收縮空間讓這個機制有效運作

**實際判斷技巧**：
- 觸發桿還有明顯收縮空間
- 凸輪葉片平均接觸岩壁
- 不需要用力擠壓才能放入

**參考來源**：Black Diamond, Wild Country, AMGA

**標籤**：`Cam`, `凸輪角度`, `抓力原理`

## Q4: 裂縫中的「走位」問題

**難度**：⭐⭐

**情境**

你在先鋒攀登時放置了一個保護點。

**問題**

什麼是「走位」（Walking），如何預防？

**選項**

- A. 攀登者走動造成的問題，無法預防
- B. 保護裝備因繩子移動而逐漸位移，可用延長繩環減少
- C. 繩子打結的現象
- D. 只有岩楔會走位，Cam 不會

**正確答案**：B

**解釋**

**走位現象**：

```
原始位置              走位後
    │                    │
    │                    │
   ╱╲ ← Cam            ╱╲ ← Cam 移動到
  岩縫                 │   不穩定的位置
    │                 岩縫
    ╲                   │
     繩子擺動 →         ╲
```

**發生原因**：
- 繩子移動拉扯保護點
- 保護點在裂縫中逐漸滑動
- 最終可能移到不安全的位置

**預防方法**：

1. **延長繩環**
   - 減少繩子直接拉扯保護點
   - 特別是在橫移路段

2. **放置方向**
   - 考慮繩子會如何拉扯
   - 選擇不容易被拉動的位置

3. **使用較長的 Cam 桿**
   - 有些 Cam 有加長桿選項
   - 減少力矩作用

**特別注意**：
- Cam 比岩楔更容易走位
- 水平裂縫走位風險較高
- 定期回頭檢查

**參考來源**：AMGA, Climbing Anchors, VDiff Climbing

**標籤**：`走位`, `保護點移動`, `預防措施`

## Q5: Tricam 的使用

**難度**：⭐⭐⭐

**情境**

你在整理裝備，看到一個 Tricam。

**問題**

Tricam 有哪兩種使用方式？

**選項**

- A. 只能當岩楔用
- B. 可以當被動岩楔或凸輪模式使用
- C. 只能當 Cam 用
- D. 只能當快扣使用

**正確答案**：B

**解釋**

**Tricam 的雙重功能**：

```
岩楔模式（Chock）         凸輪模式（Cam）

     ▲                       ▲
    ╱ ╲                     ╱│╲
   ╱   ╲                   ╱ │ ╲
  ╱─────╲                 ╱  │  ╲
    │                       │
    繩環                    繩環

用平面卡住              用尖端抵住，旋轉卡緊
```

**岩楔模式**：
- 像普通岩楔一樣使用
- 用寬面卡在收縮裂縫
- 穩定性高

**凸輪模式**：
- 尖端抵住岩壁一側
- 受力時旋轉卡緊
- 適合水平裂縫或口袋

**Tricam 的優點**：
- 多功能（兩種模式）
- 重量輕、便宜
- 適合特殊地形（如口袋、水平縫）
- 不會像 Cam 一樣走位

**Tricam 的缺點**：
- 需要較多技術
- 放置較慢
- 受力方向較單一

**參考來源**：Camp USA, AMGA, Climbing Anchors

**標籤**：`Tricam`, `雙模式`, `特殊裝備`

## Q6: Hex 的放置

**難度**：⭐⭐

**情境**

你有幾個 Hex（六角形岩楔）需要放置。

**問題**

Hex 最適合放置在什麼類型的裂縫？

**選項**

- A. 非常窄的指縫
- B. 中寬到寬的裂縫，特別是收縮或扭曲的裂縫
- C. 只適合水平裂縫
- D. 只適合人工岩場

**正確答案**：B

**解釋**

**Hex 的特點**：

```
       ╱╲
      ╱  ╲
     │    │
     │    │
      ╲  ╱
       ╲╱

Hex 有多個面可以放置
```

**放置方式**：

| 方式 | 適用情況 |
|------|----------|
| 端對端放置 | 窄裂縫 |
| 面對面放置 | 較寬裂縫 |
| 凸輪模式 | 平行或收縮裂縫 |

**Hex 的優點**：
- 覆蓋範圍大（每個有多種放法）
- 價格便宜
- 重量適中
- 扭曲裂縫特別好用

**Hex 的缺點**：
- 不適合太窄的裂縫
- 放置較 Cam 慢
- 需要正確的裂縫形狀

**與 Cam 比較**：
- Cam 更快、更通用
- Hex 更輕、更便宜
- 長路線常帶幾個 Hex 節省重量

**參考來源**：Black Diamond, AMGA, Traditional Lead

**標籤**：`Hex`, `六角形岩楔`, `放置技術`

## Q7: 測試保護點

**難度**：⭐⭐

**情境**

你剛放置好一個岩楔。

**問題**

正確測試保護點的方式是？

**選項**

- A. 用眼睛看就好
- B. 往預期受力方向用力拉扯
- C. 往各個方向搖晃
- D. 讓它掛在那裡，繼續往上爬

**正確答案**：B

**解釋**

**測試保護點的正確方法**：

```
    ╲
     ╲  ← 裂縫
      ╲
    ╱╲ ← 岩楔
   ╱  ╲
      │
      │
      ↓ 往墜落方向拉
```

**測試步驟**：

1. **判斷受力方向**
   - 通常是向下偏向攀爬方向
   - 考慮繩子路徑

2. **往該方向拉**
   - 用快扣或繩環連接
   - 用力拉扯（不要用手直接拉裝備）
   - 觀察是否移動

3. **評估結果**
   - ✅ 不動或越拉越緊 = 好
   - ❌ 有移動或搖晃 = 重新放置

**為什麼不要各方向搖**：
- 保護點只需要在受力方向有效
- 搖晃可能把好的放置弄鬆
- 浪費時間和精力

**參考來源**：AMGA, Climbing Anchors, Traditional Lead

**標籤**：`測試`, `保護點評估`, `拉扯測試`

## Q8: 裂縫大小與裝備選擇

**難度**：⭐⭐

**情境**

路線上有各種大小的裂縫。

**問題**

以下裂縫大小與適用裝備的配對，何者正確？

**選項**

- A. 指縫（Finger Crack）→ 大號 Cam
- B. 手掌縫（Hand Crack）→ 小號岩楔
- C. 拳頭縫（Fist Crack）→ 大號 Cam 或 Hex
- D. 寬裂縫（Offwidth）→ 小號 Cam

**正確答案**：C

**解釋**

**裂縫大小與裝備對照**：

| 裂縫類型 | 寬度約 | 適用裝備 |
|----------|--------|----------|
| 細縫（Thin Crack） | < 1 cm | 小號岩楔、Offset Nut |
| 指縫（Finger Crack） | 1-2.5 cm | 小型 Cam (#0.3-0.75)、岩楔 |
| 手縫（Hand Crack） | 5-7 cm | 中型 Cam (#1-3) |
| **拳頭縫（Fist Crack）** | 7-12 cm | **大型 Cam (#3-4)、Hex** |
| 寬縫（Offwidth） | > 12 cm | 特大 Cam (#5-6)、Big Bros |

**選擇原則**：
- 裝備尺寸要配合裂縫寬度
- 太大放不進去
- 太小放進去也沒用

**攜帶裝備的建議**：
- 先研究路線資訊
- 帶預期會用到的尺寸
- 多帶幾個常用尺寸

**參考來源**：AMGA, Freedom of the Hills, SuperTopo

**標籤**：`裂縫大小`, `裝備選擇`, `尺寸配對`

## Q9: 放置間隔

**難度**：⭐⭐

**情境**

你正在先鋒一條傳統路線。

**問題**

保護點之間的理想間隔是？

**選項**

- A. 越密越好，每 50 公分放一個
- B. 越省越好，整條路線只放 2-3 個
- C. 根據墜落後果、難度和個人能力判斷，通常 2-4 公尺
- D. 固定每 5 公尺放一個

**正確答案**：C

**解釋**

**保護點間隔的考量因素**：

| 因素 | 放密一點 | 放疏一點 |
|------|----------|----------|
| 墜落後果 | 有平台、地面 | 乾淨墜落區 |
| 難度 | 接近極限 | 輕鬆路段 |
| 保護品質 | 放置點不佳 | 放置點很好 |
| 心理狀態 | 緊張 | 有信心 |

**一般原則**：
- 起攀處較密（減少地面墜落風險）
- 難關前後較密
- 簡單地形可疏一點
- 墜落區乾淨可疏一點

**計算墜落距離**：
```
墜落距離 ≈ 2 × (距離最後保護點的距離) + 繩索延展

例：距最後保護點 3m，墜落約 6m + 延展
```

**過密的問題**：
- 裝備不夠用
- 浪費時間
- 繩索阻力增加

**過疏的問題**：
- 墜落距離太長
- 心理壓力大
- 風險增加

**參考來源**：AMGA, Freedom of the Hills

**標籤**：`放置間隔`, `墜落距離`, `風險管理`

## Q10: 清除保護點

**難度**：⭐⭐

**情境**

你是跟攀者，需要清除先鋒放置的保護點。

**問題**

清除岩楔的正確技巧是？

**選項**

- A. 用力往下拉
- B. 往放置的反方向推或敲，讓岩楔鬆脫
- C. 用鉤環敲打岩壁
- D. 如果拿不出來就留在那裡

**正確答案**：B

**解釋**

**清除岩楔的技巧**：

```
放置時：              清除時：
    │                    │
    ↓ 卡入              ↑ 反方向推出
   ╱╲                   ╱╲
  ╱  ╲                 ╱  ╲
 岩縫                  岩縫
```

**步驟**：
1. 觀察岩楔是怎麼放的
2. 往**放入的反方向**推
3. 可以用 Nut Tool 幫助
4. 邊推邊輕輕搖動
5. 鬆脫後取出

**Nut Tool 的使用**：
- 金屬勾可以勾住岩楔
- 用來推或敲擊
- 省力且保護手指

**困難的情況**：
- 卡很緊：多嘗試不同角度
- 放太深：用 Nut Tool 勾出
- 真的拿不出：考慮留下（但要付錢給先鋒）

**清除 Cam**：
- 拉動觸發桿收縮凸輪
- 保持拉住同時取出
- 比岩楔容易清除

**參考來源**：AMGA, VDiff Climbing

**標籤**：`清除`, `岩楔回收`, `跟攀技術`

## Q11: 保護點的冗餘

**難度**：⭐⭐

**情境**

你在一個關鍵位置只找到一個可能的放置點。

**問題**

如果單一保護點的品質不確定，應該怎麼做？

**選項**

- A. 沒關係，繼續往上爬
- B. 在附近再放一個，或連結多個較弱的點
- C. 放棄這條路線
- D. 告訴自己不會墜落就好

**正確答案**：B

**解釋**

**處理不確定保護點的方法**：

**方法一：堆疊（Stacking）**
```
    岩楔1   岩楔2
      │       │
      └───┬───┘
          │
        快扣
```
- 在同一位置放多個
- 用繩環連接在一起
- 一個失效另一個接手

**方法二：附近增加**
- 在上方或下方找其他放置點
- 多個「可能好」勝過一個「很可能好」
- 但不要因此忽略品質

**方法三：評估風險**
- 這裡墜落的後果是什麼？
- 下一個保護點在哪裡？
- 是否應該降低攀爬？

**心態**：
- 「不確定」= 當它不存在
- 不要過度依賴單一可疑保護點
- 永遠有備案

**參考來源**：AMGA, Climbing Anchors

**標籤**：`冗餘`, `不確定保護點`, `風險管理`

## Q12: Z-clip 與 Back-clip

**難度**：⭐⭐

**情境**

你正在先鋒攀登，要把繩子掛入保護點。

**問題**

Z-clip 是什麼問題？

**選項**

- A. 繩子顏色是 Z 字型
- B. 繩子錯誤地從下方保護點穿過，造成繩子路徑呈 Z 字型
- C. 快扣掛成 Z 字型
- D. 一種特殊的掛繩技巧

**正確答案**：B

**解釋**

**Z-clip 圖解**：

```
正確路徑：              Z-clip：

    ◯ 新保護點           ◯ 新保護點
    │                    │╲
    │                    │ ╲ 錯誤穿入
    ◯ 舊保護點           ◯──╱ 舊保護點
    │                    │
    確保者               確保者

繩子是直線              繩子呈 Z 字型
```

**Z-clip 的問題**：
- 墜落時繩子會卡住
- 增加墜落距離
- 可能造成保護點承受怪異角度的力

**如何發生**：
- 掛新保護點時，誤拿了下方的繩子
- 通常發生在保護點距離近時

**如何預防**：
- 掛繩前確認拿的是正確段的繩子
- 繩子應該從腰部直接上去
- 不要從下方保護點下面拿繩

**如何修正**：
- 發現後立即修正
- 把繩子從新保護點取出重掛

**參考來源**：AMGA, VDiff Climbing, Freedom of the Hills

**標籤**：`Z-clip`, `掛繩錯誤`, `繩索管理`

## Q13: 保護點與岩石品質

**難度**：⭐⭐⭐

**情境**

你攀爬的路線岩石品質看起來參差不齊。

**問題**

如何評估岩石是否適合放置保護點？

**選項**

- A. 看顏色，越深越好
- B. 敲擊聽聲音、檢查裂縫和風化程度
- C. 看有沒有其他人用過的痕跡
- D. 岩石種類不影響保護品質

**正確答案**：B

**解釋**

**岩石品質評估方法**：

**1. 敲擊測試**：
- ✅ 實心聲（高音、清脆）= 堅固
- ❌ 空心聲（低沉、悶）= 可能有空洞或裂縫

**2. 視覺檢查**：
| 好的跡象 | 不好的跡象 |
|----------|------------|
| 顏色均勻 | 變色、發白 |
| 表面完整 | 剝落、碎裂 |
| 紋理細密 | 粗糙、鬆散 |

**3. 物理測試**：
- 用手推壓岩石
- 觀察是否有碎屑
- 裂縫邊緣是否穩固

**不同岩石類型**：
| 岩石 | 通常品質 |
|------|----------|
| 花崗岩 | 通常很好 |
| 砂岩 | 變化大，注意軟弱層 |
| 石灰�ite | 可能有溶蝕空洞 |
| 玄武岩 | 通常堅硬 |

**參考來源**：AMGA, Freedom of the Hills

**標籤**：`岩石品質`, `評估方法`, `地質`

## Q14: Offset Nut 的用途

**難度**：⭐⭐⭐

**情境**

你的裝備中有一些 Offset Nut（不對稱岩楔）。

**問題**

Offset Nut 最適合用在什麼情況？

**選項**

- A. 和普通岩楔完全相同
- B. 不平行或開口狹窄的裂縫
- C. 只適合寬裂縫
- D. 只能當配重使用

**正確答案**：B

**解釋**

**Offset Nut vs 一般 Nut**：

```
一般 Nut：              Offset Nut：
   ╱╲                      ╱╲
  ╱  ╲                    ╱  │
 ╱    ╲                  ╱   │
對稱形狀                 不對稱形狀
適合 V 形裂縫            適合不規則裂縫
```

**Offset Nut 的優勢**：

| 情況 | 效果 |
|------|------|
| 淺裂縫 | 可以放更淺 |
| Pin Scar | 適合舊釘孔 |
| 不平行裂縫 | 兩側不同厚度配合 |
| 水平裂縫 | 更好的卡合 |

**使用時機**：
- 普通岩楔放不好時
- 裂縫形狀不規則
- 需要在淺處放置

**攜帶建議**：
- 不需要帶完整一套 Offset
- 帶幾個常用尺寸
- 補充普通岩楔的不足

**參考來源**：DMM, Wild Country, AMGA

**標籤**：`Offset Nut`, `不對稱岩楔`, `特殊裂縫`

## Q15: 保護點失效的原因

**難度**：⭐⭐⭐

**情境**

你在分析一次保護點失效的事故報告。

**問題**

以下哪個是保護點失效最常見的原因？

**選項**

- A. 裝備本身製造缺陷
- B. 放置不當（位置、角度、尺寸選擇錯誤）
- C. 繩子太舊
- D. 天氣影響

**正確答案**：B

**解釋**

**保護點失效的原因分析**：

| 原因 | 比例 | 說明 |
|------|------|------|
| **放置不當** | **最高** | 人為錯誤 |
| 岩石破裂 | 次高 | 岩石品質問題 |
| 裝備故障 | 很低 | 現代裝備很可靠 |
| 其他 | 低 | 異常情況 |

**常見的放置錯誤**：

1. **尺寸不對**
   - 太大：硬塞，可能傷害裝備
   - 太小：容易脫出

2. **位置不佳**
   - 裂縫形狀不適合
   - 岩石品質差

3. **角度錯誤**
   - 受力方向與放置方向不符
   - 側向力導致脫出

4. **沒有測試**
   - 放完就走
   - 沒發現問題

**如何避免**：
- 正確訓練
- 每次放置都測試
- 不要急躁
- 持續學習

**參考來源**：AAC Accidents, AMGA, Climbing Anchors

**標籤**：`失效原因`, `人為錯誤`, `安全分析`
