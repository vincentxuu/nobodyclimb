**類別代碼**：`trad-rescue`
**題目數量**：15 題
**難度範圍**：⭐⭐⭐（全部為高難度）

## Q1: 普魯士結（Prusik Knot）

**難度**：⭐⭐⭐

**情境**

你需要在繩子上打一個可以抓住繩子的繩結。

**問題**

普魯士結的主要特性是什麼？

**選項**

- A. 用來連接兩條繩子
- B. 受力時會抓緊繩子，不受力時可以滑動
- C. 永久固定在繩子上
- D. 只能向上移動

**正確答案**：B

**解釋**

**普魯士結（Prusik）的特性**：

```
       主繩
         │
    ╭────┼────╮
    │    │    │
    │╭───┼───╮│  ← Prusik 繞圈（3-4圈）
    ││   │   ││
    │╰───┼───╯│
    │    │    │
    ╰────┼────╯
         │
       繩環
```

**工作原理**：
| 狀態 | 行為 |
|------|------|
| 受力 | 繩環收緊，抓住主繩 |
| 不受力 | 可以用手推動滑行 |

**常見應用**：
- 垂降備份（Autoblock）
- 繩索上升
- 救援系統
- 拖拉系統

**繩結要求**：
- 輔助繩直徑約為主繩的 60-80%
- 圈數：通常 3-4 圈
- 太粗或太細都會影響效果

**其他類似繩結**：
- **Klemheist**：更易單手操作
- **Bachmann**：加入鉤環更好操作
- **French Prusik**：較易鬆開

**參考來源**：AMGA, Freedom of the Hills, Petzl

**標籤**：`普魯士結`, `Prusik`, `抓繩結`

## Q2: 繩索上升

**難度**：⭐⭐⭐

**情境**

你需要沿著一條固定繩向上攀爬。

**問題**

使用兩個 Prusik 上升繩索的基本原理是？

**選項**

- A. 兩個 Prusik 同時往上推
- B. 輪流將重量轉移，一個承重時另一個往上推
- C. 只需要一個 Prusik
- D. 用手直接抓繩子往上

**正確答案**：B

**解釋**

**雙 Prusik 上升法**：

```
步驟1：          步驟2：          步驟3：

  │               │               │
  P1 ← 承重      P1 ← 往上推     P1 ← 承重
  │               │               │
  │               │               │
  P2              P2 ← 承重       P2 ← 往上推
  │               │               │
 腳              腳              腳

（P1 = 吊帶 Prusik，P2 = 腳環 Prusik）
```

**設置方式**：
| Prusik | 連接位置 | 功能 |
|--------|----------|------|
| 上方 Prusik | 吊帶確保環 | 支撐身體重量 |
| 下方 Prusik | 腳環（繩梯） | 提供踩踏點 |

**上升步驟**：
1. 站在腳環上（下方 Prusik 承重）
2. 把上方 Prusik 往上推
3. 坐在上方 Prusik 上（轉移重量）
4. 把下方 Prusik 往上推
5. 重複

**注意事項**：
- 永遠保持至少一個 Prusik 承重
- 上升時可打繩結備份
- 練習直到熟練

**參考來源**：AMGA, Freedom of the Hills

**標籤**：`繩索上升`, `Prusik`, `自我救援`

## Q3: 確保脫離（Escaping the Belay）

**難度**：⭐⭐⭐

**情境**

你正在確保時，攀登者受傷無法移動，你需要去救援。

**問題**

「確保脫離」的目的是什麼？

**選項**

- A. 放棄確保離開
- B. 將負載轉移到固定點，讓確保者可以騰出雙手去進行其他操作
- C. 快速降低攀登者
- D. 把繩子剪斷

**正確答案**：B

**解釋**

**確保脫離的目的**：

```
原始狀態：              脫離後：

固定點                  固定點
  │                       │
確保者 ← 被繩子困住     負載轉移 ← 繩子固定在此
  │                       │
攀登者                  攀登者
                          │
                       確保者 ← 可以自由移動
```

**使用時機**：
- 攀登者受傷需要救援
- 需要騰出雙手做其他操作
- 需要離開確保位置

**基本步驟概念**：
1. **鎖定確保器**（Mule Knot 或類似方式）
2. **將負載轉移到固定點**
3. **使用 Prusik 或繩環作為備份**
4. **解除確保器**
5. **確保者可以移動**

**關鍵原則**：
- 永遠保持攀登者有支撐
- 每個步驟都要有備份
- 緩慢、仔細地操作

**參考來源**：AMGA, Freedom of the Hills, Petzl

**標籤**：`確保脫離`, `救援技術`, `負載轉移`

## Q4: Mule Knot（騾結）

**難度**：⭐⭐⭐

**情境**

你需要暫時鎖定確保器。

**問題**

Mule Knot 的功能是什麼？

**選項**

- A. 永久綁死繩子
- B. 暫時鎖定繩子，但可以快速解開並控制放繩
- C. 連接兩條繩子
- D. 增加摩擦力

**正確答案**：B

**解釋**

**Mule Knot 的特性**：

```
    確保器
      │
      │
     ╭╮
     ││ ← Mule Knot
     ╰╯    （可快速解開）
      │
      │
    拉繩
      │
     ╱ ╲
    ╱   ╲ ← 可選擇加上
   Overhand   半扣備份
```

**優點**：
- 鎖定繩子，空出雙手
- 可以隨時解開
- 解開時可控制放繩

**打結方式**：
1. 在確保器下方的繩子上
2. 打一個滑結（Slip Knot）
3. 用繩尾穿過形成 Mule Knot
4. 可選擇再加一個 Overhand 備份

**使用場合**：
- 垂降中途需要停止
- 確保中需要空出雙手
- 確保脫離的第一步

**解開方式**：
- 拉住繩尾（制動端）
- 解開 Mule Knot
- 控制放繩

**參考來源**：AMGA, Petzl, VDiff Climbing

**標籤**：`Mule Knot`, `騾結`, `鎖定繩子`

## Q5: 從上方降低傷者

**難度**：⭐⭐⭐

**情境**

你的跟攀者在攀爬中受傷，你需要從上方將他降低。

**問題**

從上方降低傷者時，首先需要確認什麼？

**選項**

- A. 直接開始降低
- B. 確認繩子長度足夠到達安全地點，且系統正確設置
- C. 等待救援隊
- D. 讓傷者自己爬上來

**正確答案**：B

**解釋**

**降低傷者前的檢查**：

| 檢查項目 | 原因 |
|----------|------|
| **繩子長度** | 確保能到達安全位置 |
| **繩尾** | 打結或固定，避免滑出 |
| **固定點** | 承受得住負載 |
| **傷者狀態** | 是否能配合 |
| **降低路線** | 避開障礙物 |

**系統設置**：
```
        固定點
          │
      確保器/摩擦裝置
          │
        繩子
          │
       傷者（跟攀者）
```

**降低方式選擇**：
- **確保器降低**：最常見
- **義大利半扣**：備用方案
- **摩擦裝置**：長距離降低

**降低過程**：
1. 確認傷者準備好
2. 緩慢、穩定地放繩
3. 持續溝通
4. 注意繩子剩餘長度

**參考來源**：AMGA, Freedom of the Hills

**標籤**：`降低傷者`, `救援技術`, `上方降低`

## Q6: Z-Pulley（Z 字拖拉系統）

**難度**：⭐⭐⭐

**情境**

你需要協助拖拉一個無法自行上升的攀登者。

**問題**

Z-Pulley 系統的主要優點是什麼？

**選項**

- A. 看起來像 Z 字
- B. 提供 3:1 的機械優勢，減少拖拉所需的力量
- C. 只需要一個 Prusik
- D. 不需要固定點

**正確答案**：B

**解釋**

**Z-Pulley 系統原理**：

```
     固定點
        │
    Prusik 1 ← 抓繩結（進度捕捉）
        │
        ╰───╮
            │
         滑輪/鉤環
            │
        ╭───╯
        │
    Prusik 2 ← 抓繩結
        │
        │
      受害者

拉力方向 → 向下拉，受害者向上移動
機械優勢 = 3:1
```

**優點**：
- 3:1 機械優勢
- 用 100N 的力可以拉起 300N（理論值）
- 一個人可以拖拉較重的負載

**組件需求**：
- 2 個 Prusik 或抓繩結
- 2 個鉤環（或滑輪）
- 固定點

**操作步驟**：
1. 建立固定點的 Prusik（進度捕捉）
2. 繩子經過滑輪/鉤環折返
3. 在下方繩子上建立第二個 Prusik
4. 拉動繩子
5. 釋放後進度捕捉會保持

**參考來源**：AMGA, Freedom of the Hills, Petzl

**標籤**：`Z-Pulley`, `機械優勢`, `拖拉系統`

## Q7: 通過繩結（垂降）

**難度**：⭐⭐⭐

**情境**

垂降時繩子中間有一個繩結（例如接繩處）。

**問題**

通過繩結的正確程序中，第一步應該是？

**選項**

- A. 直接讓繩結通過確保器
- B. 在繩結上方停止，建立備份系統
- C. 解開繩結
- D. 加快速度衝過去

**正確答案**：B

**解釋**

**通過繩結的步驟**：

```
1. 停在繩結上方
      │
   確保器
      │
   【繩結】← 停在這裡上方
      │
      │

2. 建立備份
      │
   確保器
    Prusik ← 在繩結上方建立
      │
   【繩結】
      │

3. 轉移到備份
      │
   確保器（解開）
    Prusik ← 現在承重
      │
   【繩結】
      │

4. 重新架設
      │
    Prusik
      │
   【繩結】
      │
   確保器 ← 在繩結下方重新架設
      │

5. 繼續下降
```

**關鍵原則**：
- 永遠保持連接
- 每步驟都有備份
- 緩慢仔細操作

**為什麼繩結不能通過確保器**：
- 繩結會卡住
- 可能導致卡在半空中
- 無法繼續操作

**參考來源**：AMGA, VDiff Climbing, Petzl

**標籤**：`通過繩結`, `垂降技術`, `進階操作`

## Q8: 墜落因子與衝擊力

**難度**：⭐⭐⭐

**情境**

你在評估救援情況中可能的衝擊力。

**問題**

墜落因子（Fall Factor）的計算公式是？

**選項**

- A. 墜落距離 × 繩子長度
- B. 墜落距離 ÷ 繩索長度（吸收衝擊的繩段）
- C. 體重 × 墜落距離
- D. 高度 ÷ 時間

**正確答案**：B

**解釋**

**墜落因子公式**：

```
墜落因子 = 墜落距離 / 繩索長度（可吸收衝擊的部分）

範例：
墜落 4m，繩子出去 10m
Fall Factor = 4 / 10 = 0.4
```

**墜落因子與嚴重程度**：

| 墜落因子 | 嚴重程度 | 說明 |
|----------|----------|------|
| < 0.5 | 低 | 一般先鋒墜落 |
| 0.5 - 1.0 | 中 | 較大的墜落 |
| 1.0 - 1.5 | 高 | 固定點承受大力 |
| 1.5 - 2.0 | 極高 | 最大可能（例如因子2墜落） |

**因子 2 墜落**：
```
  固定點
     │
     │ 2m
     │
  攀登者 ← 在固定點上方 2m 墜落
                墜落距離 = 4m
                繩長 = 2m
                Factor = 4/2 = 2
```

**為什麼這很重要**：
- 高墜落因子 = 高衝擊力
- 影響固定點和裝備的選擇
- 救援時要避免高因子衝擊

**參考來源**：Petzl, UIAA, AMGA

**標籤**：`墜落因子`, `衝擊力`, `物理原理`

## Q9: 緊急垂降（無確保器）

**難度**：⭐⭐⭐

**情境**

你遺失了確保器，需要緊急垂降。

**問題**

沒有確保器時，可以使用什麼方式垂降？

**選項**

- A. 直接用手抓繩子滑下去
- B. 義大利半扣（Munter Hitch）配合有鎖鉤環
- C. 把繩子綁在腰上
- D. 不可能垂降

**正確答案**：B

**解釋**

**義大利半扣（Munter Hitch）**：

```
      有鎖鉤環
    ╭─────╮
    │     │
  ─┬─────┬─
    │╲   ╱│
    │ ╲ ╱ │  ← 義大利半扣
    │  ╳  │
    │ ╱ ╲ │
    │╱   ╲│
  ─┴─────┴─
        │
       繩子
```

**義大利半扣的優點**：
- 只需要一個有鎖鉤環
- 可以確保和垂降
- 緊急情況的救星

**使用注意事項**：
- 必須使用有鎖鉤環（HMS 梨形最佳）
- 繩子會扭轉（kinky）
- 較難控制速度
- 對繩子磨損較大

**操作要點**：
- 制動手永遠在下方
- 繩子兩端不要太靠近
- 速度控制靠握緊程度

**其他緊急選項**：
- **Dülfersitz**（身體纏繞法）：非常緊急時
- **鉤環制動**：用多個鉤環創造摩擦

**參考來源**：AMGA, Petzl, Freedom of the Hills

**標籤**：`義大利半扣`, `緊急垂降`, `Munter Hitch`

## Q10: Autoblock 失效

**難度**：⭐⭐⭐

**情境**

垂降時你的 Autoblock（普魯士備份）沒有正常工作。

**問題**

Autoblock 失效的最常見原因是？

**選項**

- A. 繩結本身有問題
- B. 輔助繩太粗或太細、圈數不對、繩子表面濕滑
- C. 天氣太熱
- D. 鉤環品質不好

**正確答案**：B

**解釋**

**Autoblock 失效原因分析**：

| 原因 | 說明 | 解決方案 |
|------|------|----------|
| **輔助繩太粗** | 無法收緊 | 使用較細的繩環 |
| **輔助繩太細** | 抓力不足 | 使用較粗的繩環 |
| **圈數太少** | 摩擦不夠 | 增加圈數（3-4圈） |
| **圈數太多** | 難以移動 | 減少圈數 |
| **繩子濕滑** | 摩擦降低 | 增加圈數、更注意操作 |
| **Dyneema 繩環** | 太滑 | 改用 Nylon |

**理想配置**：
- 輔助繩直徑約為主繩的 60-80%
- 7mm 輔助繩配 10-11mm 主繩
- 圈數：3-4 圈
- 材質：Nylon 優先

**測試方法**：
- 垂降前先測試
- 用體重拉扯看是否抓住
- 確認可以用手推動

**參考來源**：Petzl, AMGA, VDiff Climbing

**標籤**：`Autoblock`, `失效原因`, `備份系統`

## Q11: 負載轉移

**難度**：⭐⭐⭐

**情境**

你需要將繩子上的負載從一個系統轉移到另一個系統。

**問題**

負載轉移時最重要的原則是？

**選項**

- A. 動作要快
- B. 永遠保持負載有支撐，轉移過程中不能有空檔
- C. 只需要一個固定點
- D. 可以同時解開所有連接

**正確答案**：B

**解釋**

**負載轉移的原則**：

```
❌ 危險：有空檔
原系統 → 解開 → (沒支撐) → 新系統
                    ↑
               負載會墜落！

✅ 安全：無空檔
原系統 → 新系統接上 → 轉移負載 → 解開原系統
         ↑              ↑
    先連接新的     確認新的承重
```

**正確步驟**：
1. **建立新系統**（不要解開舊的）
2. **測試新系統**（確認可以承重）
3. **逐漸轉移負載**
4. **確認新系統承重**
5. **解除舊系統**

**常見應用場景**：
- 確保脫離
- 通過繩結
- 變換確保方式
- 救援操作

**心態**：
「在解開任何東西之前，先確認有其他東西在支撐」

**參考來源**：AMGA, Petzl, Freedom of the Hills

**標籤**：`負載轉移`, `救援原則`, `系統轉換`

## Q12: 求救訊號

**難度**：⭐⭐⭐

**情境**

你和夥伴被困在岩壁上，需要求救。

**問題**

國際山難求救訊號是？

**選項**

- A. 隨便大喊
- B. 每分鐘 6 次訊號（聲音、光線或動作），間隔一分鐘後重複
- C. 持續不斷發出訊號
- D. 每分鐘 3 次訊號

**正確答案**：B

**解釋**

**國際山難求救訊號**：

| 項目 | 規格 |
|------|------|
| 頻率 | 每分鐘 6 次 |
| 間隔 | 每次約 10 秒 |
| 休息 | 一分鐘後重複 |
| 方式 | 聲音、光線或動作皆可 |

**訊號方式**：
- **聲音**：哨子、大喊、敲擊
- **光線**：手電筒、反光鏡、手機螢幕
- **動作**：揮動布料、雙臂

**確認收到的回應**：
- 每分鐘 3 次訊號
- 表示「已收到，正在來」

**其他求救方式**：
- 手機撥打緊急電話
- 衛星通訊設備
- PLB（個人定位信標）

**被困時的原則**：
- 保持冷靜
- 評估情況
- 節省體力和資源
- 持續發出訊號

**參考來源**：Freedom of the Hills, 國際山岳搜救規範

**標籤**：`求救訊號`, `緊急應變`, `國際規範`

## Q13: 傷者評估

**難度**：⭐⭐⭐

**情境**

你的夥伴墜落後似乎受傷了。

**問題**

進行傷者評估時的第一優先是？

**選項**

- A. 立刻移動傷者到安全處
- B. 確認自己和傷者都處於安全位置，再進行初步評估（意識、呼吸、出血）
- C. 立刻開始 CPR
- D. 給傷者喝水

**正確答案**：B

**解釋**

**傷者評估的順序（ABCDE）**：

| 順序 | 項目 | 檢查內容 |
|------|------|----------|
| 0 | **安全** | 你和傷者是否安全 |
| A | **Airway** | 呼吸道是否暢通 |
| B | **Breathing** | 是否有呼吸 |
| C | **Circulation** | 脈搏、大量出血 |
| D | **Disability** | 意識狀態、脊椎損傷 |
| E | **Exposure** | 其他傷害、體溫 |

**確保安全**：
- 你們是否有墜落風險？
- 是否有落石風險？
- 傷者是否被確保住？

**不要移動傷者如果**：
- 懷疑脊椎損傷
- 除非有立即生命危險
- 需要專業固定再移動

**初步處置**：
1. 控制大量出血
2. 保持呼吸道暢通
3. 防止失溫
4. 安撫傷者
5. 求救

**參考來源**：Wilderness First Aid, AMGA

**標籤**：`傷者評估`, `急救`, `ABCDE`

## Q14: 拖拉傷者

**難度**：⭐⭐⭐

**情境**

你需要獨自將傷者拖拉一小段距離。

**問題**

簡易拖拉傷者時，什麼方式最有效？

**選項**

- A. 直接抓住手拉
- B. 使用傷者自己的吊帶或創造一個簡易背負系統
- C. 把傷者扛在肩上
- D. 讓傷者滾下去

**正確答案**：B

**解釋**

**簡易拖拉方法**：

**方法一：使用吊帶**
```
     你
     │
   繩環/繩子
     │
  傷者吊帶
     │
    傷者
```
- 連接到傷者的吊帶
- 用自己的吊帶或肩膀拖拉
- 適合短距離水平移動

**方法二：繩索拖拉**
- 把傷者綁在繩子上
- 使用滑輪系統增加機械優勢
- 適合垂直或陡峭地形

**方法三：背負**
- 用繩環做成背帶
- 傷者在背後
- 需要傷者能配合

**注意事項**：
- 避免造成二次傷害
- 脊椎損傷不要亂移動
- 保護傷者頭部
- 移動前先規劃路線

**單人限制**：
- 體力限制
- 無法同時照顧傷者
- 考慮是否等待救援

**參考來源**：Wilderness First Aid, AMGA

**標籤**：`拖拉傷者`, `簡易救援`, `移動技術`

## Q15: 預防勝於救援

**難度**：⭐⭐⭐

**情境**

你在反思如何減少需要救援的情況。

**問題**

以下哪個不是預防山難的有效方法？

**選項**

- A. 出發前研究路線和天氣
- B. 攜帶足夠的裝備和急救包
- C. 永遠推到極限才能進步
- D. 知道何時撤退、設定撤退時間

**正確答案**：C

**解釋**

**預防山難的原則**：

| 類別 | 具體做法 |
|------|----------|
| **事前準備** | 研究路線、了解難度、檢查天氣 |
| **裝備** | 攜帶適當裝備、急救包、備用食物和水 |
| **技能** | 持續學習、練習救援技術 |
| **判斷** | 知道何時撤退、不逞強 |
| **團隊** | 溝通良好、互相照應 |

**「推到極限」的問題**：
- 疲勞導致判斷失誤
- 體力透支無法應對緊急情況
- 小問題變成大災難

**安全攀登的心態**：
```
✓ 山永遠在那裡
✓ 退一步，下次再來
✓ 安全回家是最大的成功
✓ 逞強只會害己害人
```

**設定撤退點**：
- 時間：幾點前必須開始撤退
- 天氣：什麼情況必須撤退
- 體力：預留足夠的回程體力

**參考來源**：AAC Accidents, Freedom of the Hills, AMGA

**標籤**：`預防`, `安全心態`, `風險管理`
