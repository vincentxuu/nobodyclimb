# NobodyClimb 商業模式規劃

## 核心理念

從攀岩者的日常生活出發，在每個接觸點創造價值。NobodyClimb 不只是紀錄平台，而是攀岩者生活方式的入口。

**七大生活面向**：課程、活動、傷痛、食品、裝備、旅行、團練

## 一、課程（Learning）

### 市場機會

攀岩人口成長，但系統化學習資源不足。多數人靠自學或朋友教，缺乏正規訓練管道。

### 商業模式

#### B2C：線上學習平台

| 產品 | 內容 | 定價模式 |
|------|------|---------|
| **基礎課程** | 攀岩入門、安全觀念、基本技巧 | 免費（導流） |
| **進階課程** | 先鋒攀登、傳統攀登、抱石技巧 | 單課 $299-599 |
| **認證課程** | 繩索系統認證（結合現有遊戲） | 套裝 $1,999-3,999 |
| **訂閱制** | 全課程庫無限觀看 | $199/月 |

**內容形式**：
- 影片課程（專業拍攝）
- 互動測驗（整合 rope-system 遊戲）
- 電子講義
- 社群問答

#### B2B：教練/岩館課程管理系統

| 服務 | 功能 | 收費模式 |
|------|------|---------|
| **課程管理** | 排課、報名、簽到、學員管理 | $2,000/月 |
| **教練媒合** | 教練資料庫、評價系統、預約系統 | 抽成 10-15% |
| **認證授權** | 岩館可發放平台認證徽章 | 授權費 |
| **企業內訓** | 團體攀岩體驗課程 | 客製報價 |

#### OMO 整合

```
線上                              線下
┌─────────────┐                 ┌─────────────┐
│ 觀看課程    │                 │ 實體練習    │
│ 完成測驗    │ ◀──────────▶   │ 教練指導    │
│ 獲得認證    │                 │ 技術驗收    │
└─────────────┘                 └─────────────┘
       │                               │
       └───────────┬───────────────────┘
                   ▼
           ┌─────────────┐
           │ 人物誌徽章  │
           │ 技能認證    │
           └─────────────┘
```

**場景案例**：
1. 用戶在平台完成「基礎確保」線上課程
2. 到合作岩館進行實體技術驗收
3. 通過後獲得平台認證徽章，顯示在人物誌
4. 解鎖進階課程購買資格

## 二、活動（Events）

### 市場機會

攀岩活動分散，資訊難以取得。比賽報名流程不統一，活動品質參差。

### 商業模式

#### B2C：活動平台

| 活動類型 | 平台功能 | 收費模式 |
|---------|---------|---------|
| **比賽活動** | 報名、分組、計分、排名 | 報名費抽成 5-10% |
| **體驗活動** | 新手體驗、親子活動 | 報名費抽成 10-15% |
| **社群活動** | 攝影日、交流會、分享會 | 免費 / 小額抽成 |
| **挑戰活動** | 線上挑戰、積分賽 | 贊助 / 免費 |

**活動管理功能**：
- 活動建立與發布
- 線上報名與付款
- 參賽者管理
- 成績登錄與排名
- 自動產生完攀紀錄

#### B2B：活動主辦服務

| 服務 | 對象 | 收費模式 |
|------|------|---------|
| **活動 SaaS** | 岩館、協會、品牌 | $3,000-10,000/場 |
| **活動企劃** | 企業團建、品牌活動 | 客製報價 |
| **裁判系統** | 正式比賽計分系統 | 授權費 + 技術支援 |
| **直播服務** | 比賽直播、線上觀賽 | 服務費 + 廣告分潤 |

## 三、傷痛（Recovery）

### 市場機會

攀岩運動傷害普遍（手指、肩膀、手肘），但專業資源難找。多數人自己摸索復健，延誤治療。

### 商業模式

#### B2C：傷痛管理服務

| 產品 | 內容 | 定價模式 |
|------|------|---------|
| **傷痛知識庫** | 常見傷害、預防方法、自我評估 | 免費（內容行銷） |
| **復健課程** | 手指復健、肩膀穩定、核心訓練 | 單課 $199-399 |
| **專家諮詢** | 物理治療師線上諮詢 | $800-1,500/次 |
| **傷痛追蹤** | 記錄傷痛、追蹤恢復進度 | 免費 / 進階功能訂閱 |

**內容合作夥伴**：
- 物理治療師
- 運動醫學醫師
- 運動防護員
- 瑜伽/伸展教練

#### B2B：運動醫療媒合

| 服務 | 對象 | 收費模式 |
|------|------|---------|
| **專家入駐** | 物理治療所、診所 | 平台曝光費 / 轉介抽成 |
| **企業健康** | 岩館員工健康管理 | 年度合約 |
| **保險合作** | 運動傷害保險 | 保費分潤 |

#### OMO 整合

```
線上                              線下
┌─────────────┐                 ┌─────────────┐
│ 傷痛自評    │                 │ 專家診療    │
│ AI 初步建議 │ ◀──────────▶   │ 物理治療    │
│ 預約掛號    │                 │ 運動防護    │
└─────────────┘                 └─────────────┘
       │                               │
       └───────────┬───────────────────┘
                   ▼
           ┌─────────────┐
           │ 復健紀錄    │
           │ 進度追蹤    │
           │ 復出建議    │
           └─────────────┘
```

**場景案例**：
1. 用戶在平台記錄手指傷痛
2. 系統提供初步衛教資訊和自我評估
3. 推薦合作的物理治療師
4. 線上預約、線下治療
5. 治療紀錄回傳平台，追蹤恢復進度
6. 恢復後提供漸進式回歸訓練建議

| **營養諮詢** | 運動營養師一對一 | $1,000-2,000/次 |
| **飲食計畫** | 客製化飲食建議 | $2,999/月 |
| **營養品商城** | 蛋白粉、膠原蛋白、維生素 | 電商抽成 |

**內容方向**：
- 攀岩前/中/後的補給策略
- 體重管理與肌力維持
- 手指肌腱營養（膠原蛋白、維生素C）
- 長途攀岩行程的飲食規劃

#### B2B：品牌合作

## 五、裝備（Gear）

### 市場機會

裝備選擇困難，資訊分散。新手不知從何開始，進階者想找專業評測。二手市場需求大但交易不便。

### 商業模式

#### B2C：裝備服務

| 產品 | 功能 | 定價模式 |
|------|------|---------|
| **裝備知識庫** | 選購指南、品牌介紹、尺寸對照 | 免費 |
| **社群評測** | 用戶裝備心得、評分系統 | 免費（UGC） |
| **裝備商城** | 新品販售 | 電商抽成 15-25% |
| **二手交易** | 裝備買賣媒合 | 交易手續費 5-10% |
| **裝備租借** | 戶外裝備短租 | 租金抽成 |

**商城品類**：
- 岩鞋（入門、進階、競技）
- 粉袋與止滑粉
- 安全吊帶
- 確保器材
- 繩索與快扣
- 服飾配件

#### B2B：品牌合作

| 合作形式 | 對象 | 收費模式 |
|---------|------|---------|
| **品牌旗艦店** | 裝備品牌 | 年費 + 抽成 |
| **試穿活動** | 岩鞋品牌 | 活動贊助 |
| **團購揪團** | 社群團購 | 量大議價分潤 |
| **經銷授權** | 成為品牌經銷商 | 經銷利潤 |

#### OMO 整合

```
線上                              線下
┌─────────────┐                 ┌─────────────┐
│ 瀏覽評測    │                 │ 岩館試穿    │
│ 尺寸建議    │ ◀──────────▶   │ 門市取貨    │
│ 線上下單    │                 │ 二手面交    │
└─────────────┘                 └─────────────┘
       │                               │
       └───────────┬───────────────────┘
                   ▼
           ┌─────────────┐
           │ 裝備清單    │
           │ 使用紀錄    │
           │ 完攀連結    │
           └─────────────┘
```

**場景案例**：
1. 用戶在平台看岩鞋評測
2. 根據腳型和攀爬風格得到推薦
3. 預約到合作岩館試穿
4. 線上下單、岩館取貨
5. 裝備加入個人清單，完攀紀錄可標記使用裝備
6. 累積「這雙鞋完攀了 50 條路線」的數據

## 六、旅行（Travel）

### 市場機會

攀岩旅行需求成長，但行程規劃門檻高。資訊分散，找不到可靠的當地嚮導。

### 商業模式

#### B2C：旅行服務

| 產品 | 內容 | 定價模式 |
|------|------|---------|
| **岩場攻略** | 國內外岩場資訊、交通、住宿 | 免費 / 進階版付費 |
| **行程規劃** | AI 輔助行程建議 | 免費 / 進階功能 |
| **揪團出遊** | 社群自組旅行團 | 平台服務費 |
| **套裝行程** | 與旅行社合作的攀岩旅行團 | 分潤 15-25% |

**熱門目的地**：
- 國內：龍洞、關子嶺、壽山、東北角
- 亞洲：泰國喀比、日本小川山、韓國
- 歐美：西班牙、法國楓丹白露、優勝美地

#### B2B：旅遊合作

| 合作形式 | 對象 | 收費模式 |
|---------|------|---------|
| **旅行社合作** | 戶外旅行社 | 分潤 / 聯合行銷 |
| **嚮導媒合** | 當地攀岩嚮導 | 平台抽成 15-20% |
| **住宿合作** | 岩場周邊民宿 | 訂房抽成 |
| **保險合作** | 旅遊平安險 | 保費分潤 |

#### OMO 整合

```
旅行前                  旅行中                  旅行後
┌─────────────┐       ┌─────────────┐       ┌─────────────┐
│ 岩場研究    │       │ 即時岩況    │       │ 完攀紀錄    │
│ 行程規劃    │  ───▶ │ 導航指引    │  ───▶ │ 旅行遊記    │
│ 揪團報名    │       │ 社群互助    │       │ 照片分享    │
└─────────────┘       └─────────────┘       └─────────────┘
       │                     │                     │
       └─────────────────────┴─────────────────────┘
                             ▼
                     ┌─────────────┐
                     │ 攀岩足跡    │
                     │ 旅行徽章    │
                     └─────────────┘
```

**場景案例**：
1. 用戶想去泰國喀比攀岩
2. 在平台查看岩場攻略、社群評價
3. 加入已成團的揪團行程，或自己發起揪團
4. 平台媒合當地嚮導、推薦住宿
5. 旅途中記錄完攀、分享岩況
6. 回國後自動整理旅行回顧，獲得「喀比探險家」徽章

## 七、團練（Training）

### 市場機會

攀岩是社交運動，但約爬麻煩、找不到程度相近的夥伴。訓練缺乏系統性，不知如何進步。

### 商業模式

#### B2C：團練服務

| 產品 | 功能 | 定價模式 |
|------|------|---------|
| **約爬揪團** | 發起/加入約爬、時間地點配對 | 免費 |
| **訓練計畫** | AI 生成個人化訓練菜單 | $299/月 |
| **訓練營** | 週期性團體訓練課程 | 報名費 $3,000-8,000 |
| **私人教練** | 一對一教練媒合 | 抽成 15-20% |
| **訓練紀錄** | 訓練日誌、進步追蹤 | 免費 / 進階分析 |

**訓練內容**：
- 指力訓練
- 核心穩定
- 柔軟度
- 心理訓練
- 路線閱讀

#### B2B：岩館/教練服務

| 服務 | 對象 | 收費模式 |
|------|------|---------|
| **團練管理** | 岩館定期團練 | $1,500/月 |
| **教練工具** | 學員管理、訓練追蹤 | $500/月/教練 |
| **企業團建** | 公司攀岩活動 | 客製報價 |
| **校隊管理** | 學校攀岩社團/校隊 | 教育優惠 |

#### OMO 整合

```
線上                              線下
┌─────────────┐                 ┌─────────────┐
│ 發起約爬    │                 │ 岩館見面    │
│ 程度配對    │ ◀──────────▶   │ 實際攀爬    │
│ 訓練計畫    │                 │ 互相拍照    │
└─────────────┘                 └─────────────┘
       │                               │
       └───────────┬───────────────────┘
                   ▼
           ┌─────────────┐
           │ 訓練紀錄    │
           │ 完攀上傳    │
           │ 攀友連結    │
           └─────────────┘
```

**場景案例**：
1. 用戶想找人一起練習
2. 在平台發起「週六下午內湖運動中心抱石」
3. 系統推薦程度相近的用戶
4. 有人報名加入
5. 當天見面攀岩、互相拍照
6. 回家後互相標記、上傳完攀紀錄

## 八、延伸商業模式

除了傳統的 B2B、B2C、OMO，以下是更多創新商業模式的可能性：

### 8.1 C2C（用戶對用戶）

用戶之間直接交易或交換，平台作為媒合與信任中介。

| 模式 | 說明 | 平台角色 | 抽成 |
|------|------|---------|------|
| **二手交易** | 裝備買賣 | 媒合 + 金流 + 信評 | 5-10% |
| **技能交換** | 「我教你先鋒，你教我抱石」 | 配對服務 | 免費 / 進階配對 |
| **約爬配對** | 找程度相近的攀友 | 智慧配對 | 免費導流 |
| **接案市場** | 攝影師、確保手、私人教練 | 媒合 + 評價 | 10-15% |
| **場地共享** | 私人岩牆出租 | 媒合 + 保險 | 15% |

**C2C 信任機制**：
- 用戶評價與信譽分數
- 實名認證
- 交易保障（第三方託管）
- 爭議處理機制

```

│ 提供服務    │           │ 信任背書    │           │ 下單預約    │
│ 收款提領    │  ◀──────  │ 金流處理    │  ──────▶  │ 付款評價    │
└─────────────┘           └─────────────┘           └─────────────┘
```

### 8.13 延伸商業模式優先建議

### 核心策略

1. **先養後收**：用免費的核心功能（飛輪）累積用戶和內容

3. **生態閉環**：讓用戶的每個需求都能在平台內解決
4. **創作者賦能**：讓核心用戶也能賺錢，形成共生關係
5. **數據驅動**：累積數據資產，逐步開展數據變現
