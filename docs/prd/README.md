**版本**: 1.0
**最後更新**: 2025-12-30
**狀態**: Draft
**負責人**: Product Team
**審核人**: Development Team

## 文件概述

本 PRD 目錄包含 NobodyClimb 攀岩社群平台的完整產品需求文件，包括：

- **主 PRD** (`product-requirements.md`) - 產品願景、目標使用者、功能需求
- **功能規格** (`feature-specifications.md`) - 詳細功能描述和使用者故事
- **技術需求** (`technical-requirements.md`) - 技術架構和非功能需求
- **發布計畫** (`release-roadmap.md`) - 里程碑和優先級排序
- **成功指標** (`success-metrics.md`) - KPI 和資料追蹤

---

## 快速導覽

### 對於產品經理
1. 先閱讀 `product-requirements.md` 了解產品願景
2. 查看 `release-roadmap.md` 了解發布計畫
3. 參考 `success-metrics.md` 定義衡量標準

### 對於設計師
1. 查看 `feature-specifications.md` 了解使用者旅程
2. 參考 `product-requirements.md` 的目標使用者畫像
3. 檢視 `technical-requirements.md` 的設計限制

### 對於開發者
1. 閱讀 `technical-requirements.md` 了解技術棧
2. 查看 `feature-specifications.md` 的功能詳情
3. 參考 `release-roadmap.md` 確定實作優先級
