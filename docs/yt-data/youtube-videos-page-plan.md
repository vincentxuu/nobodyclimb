# YouTube 影片列表頁面規劃

## 概述
建立一個展示攀岩相關YouTube影片的列表頁面，讓使用者可以瀏覽並觀看來自YouTube的攀岩影片內容。

## 頁面路徑
`/videos` - YouTube影片列表頁面

## 功能需求

### 1. 核心功能
- **影片列表展示**：以網格或列表形式展示YouTube影片
- **影片預覽**：顯示影片縮圖、標題、頻道名稱、觀看次數和發布時間
- **影片播放**：支援內嵌播放器或彈出視窗播放
- **分頁載入**：支援分頁或無限滾動載入更多影片

### 2. 搜尋與篩選
- **關鍵字搜尋**：根據影片標題或描述搜尋
- **分類篩選**：
  - 影片類型（紀錄片、教學、比賽、戶外攀岩、室內攀岩）
  - 發布時間（最新、本週、本月、今年）
  - 時長（短片<5分鐘、中等5-20分鐘、長片>20分鐘）
- **排序功能**：
  - 發布時間（新到舊/舊到新）
  - 觀看次數
  - 相關性

### 3. 互動功能
- **收藏功能**：讓登入用戶可以收藏喜歡的影片
- **分享功能**：分享影片到社群媒體
- **評論功能**（選擇性）：允許用戶在站內留言討論

## 技術實作

### 1. 資料來源
- **YouTube Data API v3**：用於獲取影片資訊
- **本地資料庫**：儲存精選影片清單和使用者收藏

### 2. 檔案結構
```
src/app/videos/
├── page.tsx                 # 影片列表主頁面
├── [id]/
│   └── page.tsx             # 單一影片詳細頁面（選擇性）
└── layout.tsx               # 影片頁面布局

src/components/videos/
├── video-grid.tsx           # 影片網格組件
├── video-card.tsx           # 影片卡片組件
├── video-player.tsx         # 影片播放器組件
├── video-filters.tsx        # 篩選器組件
└── video-search.tsx         # 搜尋組件

src/lib/api/
└── youtube.ts               # YouTube API 整合

src/lib/types/
└── video.d.ts               # 影片相關型別定義
```

### 3. 資料結構
```typescript
interface YouTubeVideo {
  id: string
  title: string
  description: string
  thumbnailUrl: string
  channelName: string
  channelId: string
  publishedAt: string
  duration: string
  viewCount: number
  likeCount?: number
  embedUrl: string
  category?: string
  tags?: string[]
}
```

### 4. API 路由規劃
- `GET /api/videos` - 獲取影片列表
- `GET /api/videos/[id]` - 獲取單一影片詳情
- `POST /api/videos/favorite` - 收藏影片
- `DELETE /api/videos/favorite/[id]` - 取消收藏

## 實作方案選擇

### 選擇方案一：不使用YouTube API
考量到快速開發和簡單維護，初期選擇不使用YouTube API，採用手動維護精選影片清單的方式。

**優點：**
- 無需申請API金鑰
- 沒有API配額限制  
- 實作簡單快速
- 無額外成本
