# 人物誌內容調整建議

> 版本: 1.0
> 建立日期: 2026-01-11
> 基於: biography-redesign-spec.md, biography-questions-analysis.md

---

## 整體調整方向

```
現有結構                          調整後結構
─────────────────────────────────────────────────────────
Biography (單一表)                Biography (主表)
├─ 7 個純文字欄位                 ├─ 基本資訊
└─ bucket_list (純文字)           ├─ 核心故事欄位
                                  ├─ 進階故事欄位 (新增)
                                  │
                                  BucketListItem (獨立表)
                                  ├─ 結構化目標清單
                                  └─ 進度追蹤
                                  │
                                  BiographyStory (新增表)
                                  └─ 獨立故事卡片
```

### 2. BucketListItem 人生清單表（新增）

```typescript
interface BucketListItem {
  id: string
  biography_id: string

  // 基本內容
  title: string                         // 目標標題
  description: string | null            // 詳細描述
  category: BucketListCategory          // 分類

  // 目標細節
  target_grade: string | null           // 目標難度
  target_location: string | null        // 目標地點

  // 時間管理
  created_at: string
  target_date: string | null            // 預計完成
  completed_at: string | null           // 實際完成

  // 狀態
  status: 'planned' | 'in_progress' | 'completed' | 'on_hold'
  progress: number                      // 0-100

  // 完成故事
  completion_story: string | null       // 完成心得
  completion_media: string | null       // JSON: 完成照片/影片

  // 社群互動
  is_public: boolean
  likes_count: number
  inspired_count: number                // 被多少人加入清單

  sort_order: number
}

type BucketListCategory =
  | 'outdoor_route'      // 戶外路線
  | 'indoor_grade'       // 室內難度
  | 'competition'        // 比賽目標
  | 'training'           // 訓練目標
  | 'adventure'          // 冒險挑戰
  | 'skill'              // 技能學習
  | 'other'              // 其他
```

### 3. BiographyStory 故事卡片表（新增）

獨立的故事卡片，可以持續新增，形成動態內容。

```typescript
interface BiographyStory {
  id: string
  biography_id: string

  // 內容
  story_type: StoryType                 // 故事類型
  title: string | null                  // 標題（可選）
  content: string                       // 故事內容
  media_urls: string | null             // JSON: 配圖/影片

  // 關聯
  related_bucket_item_id: string | null // 關聯的人生清單項目
  related_location: string | null       // 關聯地點

  // 時間
  story_date: string | null             // 故事發生時間
  created_at: string
  updated_at: string

  // 社群
  is_public: boolean
  likes_count: number
  comments_count: number

  // 狀態
  is_featured: boolean                  // 是否被精選
}

type StoryType =
  | 'memorable_moment'    // 難忘時刻
  | 'breakthrough'        // 突破故事
  | 'challenge'           // 挑戰經歷
  | 'lesson_learned'      // 學到的事
  | 'gratitude'           // 感謝故事
  | 'milestone'           // 里程碑
  | 'daily'               // 日常分享
```

### 題目詳細設計

#### 第一階段：基本資訊

| 欄位 | 題目 | 填寫提示 | 類型 |
|-----|------|---------|-----|
| name | 你的暱稱 | 可以是真名或攀岩圈的稱號 | 文字 |
| avatar_url | 大頭照 | 上傳一張你攀岩時的照片 | 圖片 |
| climbing_start_year | 哪一年開始攀岩 | 選擇年份 | 下拉選單 |
| frequent_locations | 平常出沒的地方 | 可填多個，用逗號分隔 | 文字 |
| favorite_route_type | 喜歡的路線型態 | 抱石/先鋒/速度/傳攀/都喜歡 | 多選 |

#### 第二階段：核心故事

| 欄位 | 題目 | 填寫提示 | 字數建議 |
|-----|------|---------|---------|
| climbing_origin | 你與攀岩的相遇 | 描述第一次接觸攀岩的情景，是什麼讓你想繼續？ | 100-300字 |
| climbing_meaning | 攀岩對你來說是什麼 | 攀岩在你生活中扮演什麼角色？帶給你什麼？ | 100-300字 |
| advice_to_self | 給剛開始攀岩的自己 | 如果能回到起點，你會對自己說什麼？ | 50-200字 |

#### 第三階段：人生清單

見前文 `BucketListItem` 結構

#### 第四階段：進階故事

| 欄位 | 題目 | 引導問題 | 目的 |
|-----|------|---------|------|
| memorable_moment | 攀岩路上最難忘的一天 | 描述一個具體的場景，發生了什麼？你有什麼感受？ | 創造共鳴 |
| biggest_challenge | 曾經想放棄的時刻 | 是什麼讓你想放棄？又是什麼讓你繼續？ | 鼓勵低潮者 |
| breakthrough_story | 最大的突破經歷 | 你如何突破的？過程中學到什麼？ | 分享方法 |
| fear_management | 面對恐懼的方式 | 你如何處理墜落、高度的恐懼？有什麼具體方法？ | 實用建議 |
| climbing_lesson | 攀岩教會我的事 | 攀岩讓你對人生有什麼不同的理解？ | 人生智慧 |
| climbing_mentor | 攀岩路上的貴人 | 誰對你的攀岩旅程影響最大？想對他說什麼？ | 感謝傳承 |
| dream_climb | 夢想中的攀登 | 最想完成的攀登是什麼？為什麼？ | 展現夢想 |
| life_outside_climbing | 攀岩之外的我 | 你的工作、其他興趣是什麼？如何平衡？ | 完整的人 |

CREATE TABLE bucket_list_items (
  id TEXT PRIMARY KEY,
  biography_id TEXT NOT NULL,
  title TEXT NOT NULL,
  description TEXT,
  category TEXT DEFAULT 'other',
  target_grade TEXT,
  target_location TEXT,
  created_at TEXT DEFAULT (datetime('now')),
  updated_at TEXT DEFAULT (datetime('now')),
  target_date TEXT,
  completed_at TEXT,
  status TEXT DEFAULT 'planned',
  progress INTEGER DEFAULT 0,
  completion_story TEXT,
  completion_media TEXT,
  is_public INTEGER DEFAULT 1,
  likes_count INTEGER DEFAULT 0,
  inspired_count INTEGER DEFAULT 0,
  sort_order INTEGER DEFAULT 0,
  FOREIGN KEY (biography_id) REFERENCES biographies(id) ON DELETE CASCADE
);

CREATE INDEX idx_bucket_list_biography ON bucket_list_items(biography_id);
